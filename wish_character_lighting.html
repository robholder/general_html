WISH - Character Lighting: Main Characters

Rules

  • Limited detailed textures
  • Reduced specular highlights
  • No Peach Fuzz

Clothes

  • Painted details
  • Limited displacements
  • No XGen

Hair

  • Kuwahara Maps (painterly)
  • Limited noise and strays

Eyes

  • Used watercolor style maps
  • Specular treated non-raytraced
  • Specular contribution is being removed in comp (for all eye)

Ingredients

Skin Painted Mattes

  • lrigPainedLightingModifier (base):
    • matPalette:
      • paintedLighting_mains:
        • e.g.: asha_painted_lighting_skin_MOD
        • Inside - brows, face, nose base.

Import fresh one for mattes:

  • ShadowBox:
    • filter 'matte'
    • USD File:
      • pained_lighting_skin_colorMattes
      • Import button

Comes into shot_palette>cloned_materials

Character Modifiers

  • shot:
    • e.g. asha (main):
      • matPalette:
        • stylizationGlobalMattes_geo_costume

Comp Usage

Munge

  • No spec or reflection contribution on eyes (whole area)
  • Kuwuhara is controled by maps:
    • Red:
      • Size
      • Brighter means blurier
    • Blue:
      • Contrast
      • Brighter means sharper

Lineart

Two inputs:

  • Lookmaps
  • Geo Normals

Tips

Try to avoid cutouts in characters. Use overs if possible.
