Hacking With Swift - 17 - How to use a for loop to repeat work

For Loops

let platforms = ["iOS", "macOS", "tvOS", "watchOS"]

for os in platforms {
    print("Swift works great on \(os).")
}

Range

To set a range in a 'for' loop: 

for i in 1...12 {
    print("5 x \(i) is \(5 * i)")
}

5 x 2 is 10.
5 x 3 is 15.
5 x 4 is 20.

5 x 7 is 35.

5 x 10 is 50.
5 x 11 is 55.
5 x 12 is 60.

Nested Loops

for i in 1...12 {
    print("The \(i) times table:")

    for j in 1...12 {
        print("  \(j) x \(i) is \(j * i)")
    }

    print()
}
