Hacking With Swift - 17 - How to use a for loop to repeat work

For Loops

let platforms = ["iOS", "macOS", "tvOS", "watchOS"]

for os in platforms {
    print("Swift works great on \(os).")
}

Range

To set a range in a 'for' loop: 

for i in 1...12 {
    print("5 x \(i) is \(5 * i)")
}

5 x 2 is 10.
5 x 3 is 15.
5 x 4 is 20.
5 x 5 is 25.
5 x 6 is 30.
5 x 7 is 35.
5 x 8 is 40.
5 x 9 is 45.
5 x 10 is 50.
5 x 11 is 55.
5 x 12 is 60.

Swift has a similar-but-different type of range that counts up to but excluding the final number: ..<. This is best seen in code: 

for i in 1...5 {
    print("Counting from 1 through 5: \(i)")
}

print()

for i in 1..<5 {
    print("Counting 1 up to 5: \(i)")
}

Tip: ..< is really helpful for working with arrays, where we count from 0 and often want to count up to but excluding the number of items in the array.

Nested Loops

This is a nested loop example. Also, note the empty 'print()' command, that actually outputs a 'newline'. 

for i in 1...12 {
    print("The \(i) times table:")

    for j in 1...12 {
        print("  \(j) x \(i) is \(j * i)")
    }

    print()
}

Wildcard Pattern

If you are not going to reference the value of a count variable (like the i in the last few examples), you can use an 'anonymous' variable like _.

These are also called:

  • Wildcard
  • Discard
  • Anonymous variable
  • Placeholder
  • Ignore
