Gizmo to Group - In Nuke

Manual Method

In the Gizmo node's property panel > Node tab:

  • Click on "Copy to group" button.

Python Method

Select gizmo: 

n = nuke.selectedNode()
n.makeGroup()

If the Gizmo is inside an existing Group (creates the group OUTSIDE the group): 

# Edit for Group name:

group_node = nuke.toNode("Group")  # Replace with your Group name
selected_inside = [n for n in group_node.nodes() if n['selected'].value()]

if selected_inside:
    for node in selected_inside:
        if "gizmo_file" in node.knobs():
            node.makeGroup()
else:
    print("No node selected inside the Group.")
