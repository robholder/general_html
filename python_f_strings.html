F Strings

  • Introduced in Python 3.6

https://www.youtube.com/watch?v=Mfmr_Puhtew

Basic

number = 800

print(f"The number is {number}") # Regular number
>> The number is 800

print(f"The number is {number:x}") # Hexadecimal
>> The number is 320

print(f"The number is {number:o}") # Octal
>> The number is 1440

print(f"The number is {number:e}") # Exponent
>> The number is 8.000000e+02.

Padding

print(f"The number is {number:06}") # 6 position padding
>> The number is 000800

Decimal Places

print(f"The number is {number:.2f}") # 2 decimal places (rounding)
>> The number is 800.00

number = 100.23984
print(f"The number is {number:.2f}") # 2 decimal places (rounding)
>> The number is 100.24

Number Grouping

print(f"{44000000:,.2f}")
>> 44,000,000.00

print(f"{44000000:,.0f}")
>> 44,000,000

print(f"{440000000:,}")
>> 44,000,000

Percentage

print(f"{0.34567:%}")
>> 34.567%

print(f"{0.34567:.2%}")
>> 34.57%

print(f"{0.34567:.0%}")
>> 35%

Alignment

  • For numbers, it's like zero padding: 
    for number in range(5, 15):
    print(f"The number is {number:6}.")

>> The number is      5.
>> The number is      6.
>> The number is      7.
>> The number is      8.
>> The number is      9.
>> The number is     10.
>> The number is     11.
>> The number is     12.
>> The number is     13.
>> The number is     14.
>> The number is     15.
  • For regular characters/text: 
    greet = "Hi"

print(f"{greet:>10}") # Aligned 10 right
print(f"{greet:^10}") # Centered
print(f"{greet:<10}") # Left

>>        Hi
>>    Hi
>>Hi

# This becomes more clear when you add spacer charaters:

print(f"{greet:_>10}") # Aligned 10 right
print(f"{greet:_^10}") # Centered
print(f"{greet:_<10}") # Left

>>________Hi
>>____Hi____
>>Hi________

Datetime

import datetime

today = datetime.datetime.now()

print(f"simple date printing: {today}")
>> simple date printing: 2023-10-04 20:09:05.661436

print(f"{today:%H:%M}")
>> 20:10


print(f"{today:%Y-%m-%d %H:%M:%S.%f}")
>> 2023-10-04 20:10:39.991190 

print(f"{today:%y}")
>> 23

print(f"{today:%D}")
>> 10/04/23

print(f"{today:%T}")
>> 20:10:39

print(f"{today:%A}")
>> Wednesday

print(f"{today:%B}")
>> October

print(f"{{}}")
