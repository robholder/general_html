Deep-Dives - Edge Pooling

Edge pooling gives the watercolor edges as seen in watercolor painting.

  • Edges:
    • Boundaries of Light & Dark

Rim / Key Tagging

Use shuffles to create and add to the Rim and Key layers for use in creating the edgepooling.

Luminance Maps

This is where the heavy lifting happens (boundary between lights and darks).

  • Luminance key node:
    • Input is the Key layer (piped in with the Rim/Key Tagging)
    • Crunch to a black and white light map
  • Direct Diffuse might be a better light ingredient if there's issues
