Replacing or Removing Tags

Replace populated tags with empty tags: (Non-greedy, multiple lines) 

:%s~<id>\_.\{-}</id>~<id><\id>~g

Explanation: 

% - this file
s - substitute
~ - alternative to / (since this character is part of the search string)
<id> - part of search string
\_. - finds any character, including 'end of line'
\{-} - matches as few as possible (* will find to the last occurance)
<\id> - part of the search string
~ - alternative to / (since this character is part of the search string)
<id><\id> - replacement string
~ - alternative to / (since this character is part of the search string)
g - global
