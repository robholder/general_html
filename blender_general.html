Keyboard Shortcuts

In The 3d Viewport:

The default shortcut for 'frame selected' is <numpad> / and/or <numpad> ., but on a restricted Mac keyboard this isn't an option.

In Edit > Preferences > Keymap > 3D View > Frame Selected:

  • I changed from <numpad> / to F

Again on a Mac:

  • <fn> Left-Arrow (equivalent to 'Home' button) <-- Frame All
  • N - Properties
  • T - Tool-shelf (left)
  • <shift> C <-- Recenter the 3d Cursor

Shader Editor

  • <cntl><shift> LMB on a node <-- preview the shader node's output in the viewport's render.
