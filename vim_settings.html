Shell or Commandline - Vi Mode

When working in a shell/terminal with ordinary commandline, you can use vi-mode to be able to edit and navigate the line using the following:

  • Commandline:
    • Using vi-mode in a shell: 
set -o vi
  • Add to bashrc (or aliases) for permanent vi-mode and the option to use 'jk' instead of to enter 'normal' mode:

Bash

# terminal to use vi/vim moves
# https://unix.stackexchange.com/questions/303282/in-bash-vi-mode-map-jk-to-exit-insert-mode
set -o vi
bind '"jk":vi-movement-mode'

ZSH

## ZSH:
bindkey -v
#
# Map 'jk' to the escape sequence (exit insert mode)
bindkey -s "jk" "^["
