Camera

View tab

  • Set Clipping Range:
    • Near
    • Far
  • Focal Length:
    • set realistic focal length (this is often wildly insane)
    • see solution below
  • Horizontal Aperture:
    • set this to something reasonable (often starts at something impractical)
    • see solution below

Solution to crazy settings

  • MMB on Focal Length label for default (50):
    • Then adjust
  • MMB on Horizontal Aperture label for default (20.955):
    • Then adjust

Sampling tab

Camera settings

  • F-Stop is like a real lens' settings
    • F-Stop of zero means no Depth of Field calculation (all sharp)
  • Focus Distance is the plane of focus distance from the camera focal point.
    • NOTE: You can <shift> LMB in the viewport to set focus distance interactively.

Viewport feedback for Focus

(Camera node and Transform must be selected for the following to work.)

  • In Network editor (Stage) select camera
  • In viewport, select transform handle tool
    • <shift> f to activate visual indication of focus:
      • line will a=1
      • appear to indicate focus plane
    • <shift> f again to make regions more opaque
    • <shift> f again to switch off viewport guide

You can <shift> LMB in the viewport to set focus distance interactively.

Karma Render Settings Node

After the camera etc., you can add the Karma Render Settings node.

Render Engine

From the Rendering Engine drop-down, you can choose any renderer on the system (i.e., if available).

Viewport

By default, you'll only have one Viewport Render settings option:

  • Karma Persp dropdown (top-right of viewport):
    • Render Settings > Viewport Settings (check-circle)

Viewing with the Karma Render Settings Node active/viewable:

  • Karma Persp dropdown (top-right of viewport):
    • Render Settings > Viewport Settings
    • Render Settings > Default (/Render/rendersettings)
    • Render Settings > /render/rendersettings

The middle one will show with the Karma node settings.

Filter Settings

Denoiser is one setting to look at:

  • Image Output > Filters > Denoiser (drop-down)

AOVs

Karma Render Settings > Image Output > AOVs (Render Vars)

Pick and choose AOV options here, including whether to split each pass by LPT (each light).

LPE Tags

To set the LPE Tag for each light (to be used in the AOVs LPE output):

  • Light parameters:
    • Karma tab:
      • LPE Tag drop-down:
        • $OS (to use evaluate to the light name, or choose other options)

LPE in Viewport

Viewport panel (right):

  • three layer icon:
    • choose available LPEs to view

Cryptomatte

Karma Render Settings > Image Output > AOVs (Render Vars):

  • Extra Render Vars toggle-down:
    • LMB click on + to the right of Render Vars field to add pass.
    • Dropdown on right to choose pass:
      • Cryptomatte Object Name
      • Cryptomatte Material Name

You can see this as a preview in the viewport by selecting the Karma rendersettings & then choosing the pass in the three-layer icon on the side column of tools.

USD Render ROP - Render Node

End the network with a USD Render ROP node.

You can choose from a drop-down which 'render settings' to use (if there are multiple options set in your scene).
