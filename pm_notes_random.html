MIR Procedure

  • Go to shot:
    • s 2230 0510
  • Open Nuke:
    • nuke &
  • PAM Open:
    • precomp
    • open latest Mir precomp script
  • Fix Read paths:
    • Paramount Tools > Read Node File Path Fixer
    • or
    • <cntrl><shift> /:
      • Search for old file path
      • Enter replacement path
  • Import template:
    • ParamountTools > Custom Tools for avoa > Templates > Import_Template

For 3d Card shots

Characters:

  • Take the card projection elements and put them on a backdrop (Layer Transforms) at the base of each character (after DI Mattes etc).
  • Combine into a scene and scanlinerender in a 'shot transforms' backdrop. Add the usual reformat and crop nodes.

BG:

  • Sometimes these can be treated normally (after card set up).
