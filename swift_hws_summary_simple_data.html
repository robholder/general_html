Hacking With Swift - Summary: Simple Data

  • Constants - let
  • Variables - var
    • Use constants over variables if possible - it'll help prevent mistakes.
  • Strings:
    • " & "
    • or multiline with triple quotes (with last three on separate line:
      • """ and (newline) """
  • Whole numbers: 'Integers'
  • Decimal number: 'Double'
  • Boolean: true or false:
    • Can be flipped with ! operator or .toggle() method
  • String interpolation is sometimes more efficient and flexible than concatenation:
    • "A " + var + " string"
    • "A \(var) string"
  • These different constants and variables have 'methods' which can be called on them:
    • count
    • .uppercased()
    • .isMultiple(of: #)
    • .toggle()
