Flip Fluids

  • middle-click on a node to get info
  • make sure units are real-world for effects
  • Transform can be used to scale:
    • single transform for single geo
    • Master for groups of geo
  • Use Dorming Object for a collision object:
    • This creates an AutoDopNetwork
    • key node is top node
    • turn off 'deforming'
  • Flip Fluids use volumes, not particles
  • Switch off 'Display Geometry', switch on Collision Guide geometry
  • Global Animation Options:
    • Turn off 'Integer Frame Values' to be able to scrub interframes for sims
  • Don't forget to set UI to "Manual" (bottom right)
