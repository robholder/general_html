Vim Learning

Vim-Colortemplate Plugin

GitHub Repo:
https://github.com/lifepillar/vim-colortemplate

Install

This install process used vim-plug:

https://github.com/junegunn/vim-plug

Add this to the vimrc file and run :PlugInstall 

Plug 'lifepillar/vim-colortemplate'

Start

  • The help file can be opened like this:
    :help colortemplate.txt
  • Open the template in vim: 
    vim /Users/<username>/.vim/plugged/vim-colortemplate/templates/dark.colortemplate

    :edit templates/dark.colortemplate
:Colortemplate! ~/.vim
:colorscheme dark
  • Click on OutDir to set ~/.vim as the out dir: 
    Change to:
  • Click on Build! (at the top of the buffer; if the colortemplate has loaded successfully)
  • Set to a colortemplate file type if you need to (in a new buffer for example). This will gain access to Colortemplate's commands: 
    :set ft=colortemplate

Build!

  • Clicking build creates a colorscheme into the ~/.vim/colors directory, with the name put in the metadata at the start.
