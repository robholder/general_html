Hacking With Swift - 18 - How to use a while loop to repeat work

While Loop

While loops continue to loop until a condition is met. 

var countdown = 10

while countdown > 0 {
    print("\(countdown)…")
    countdown -= 1
}

print("Blast off!")

While loops are really useful when you just don’t know how many times the loop will go around.

We'll look at an example using dice rolls as an illustration (using a 20 sided die).

Random

First we need to know how to set random numbers (Int & Doubles):

This creates a new integer between 1 and 1000 

let id = Int.random(in: 1...1000)

And this creates a random decimal between 0 and 1: 

let amount = Double.random(in: 0...1)

We can use this functionality with a while loop to roll some virtual 20-sided dice again and again, ending the loop only when a 20 is rolled

Here’s the code to make that happen: 

// create an integer to store our roll
var roll = 0

// carry on looping until we reach 20
while roll != 20 {
    // roll a new dice and print what it was
    roll = Int.random(in: 1...20)
    print("I rolled a \(roll)")
}

// if we're here it means the loop ended – we got a 20!    
print("Critical hit!")
