InteractiveCacheUI Node (IC)

To make one:

  • Tab>Interactive>InteractiveCache
  • Drop into 'pass' stream (to pick up pass name and sets info for Quip)
  • Click on Quip Setup on properties to set up render nodes (for farm)

color coded

  • Green:
    • Live
  • Blue:
    • Precomp'ed

Quip

In the Quip panel, you should see each InteractiveCache node listed.

Choose and adjust to send to farm.

InteractiveCacheUI

To open the UI (User Interface)

  • Tab>InteractiveCacheUI
  • Switch to Live/Precomp (see color guide above)
  • Choose which IC nodes to include
  • Set frame range
  • When Quip is set up, you can submit to the farm.
