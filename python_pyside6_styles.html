PySide6 "Styles"

To set a style, use 'setStyle' on the application object: 

app = QApplication(sys.argv)
app.setStyle('Fusion')

Available styles

The available PySide6 styles may differ system to system, but because many people install the PySide6 binary the styles installed are often the same. You can get a list of available styles using the Python interpreter, like so: 

$ python3
Python 3.6.7 (default, Oct 22 2018, 11:32:17) 
[GCC 8.2.0] on linux
Type "help", "copyright", "credits" or "license" for more information.
>>> import PySide65.QtWidgets
>>> print(PySide65.QtWidgets.QStyleFactory.keys())
['Breeze', 'Oxygen', 'QtCurve', 'Windows', 'Fusion']

Dark Theme

You need to install the following for a dark theme:

https://pypi.org/project/pyqtdarktheme/

https://pyqtdarktheme.readthedocs.io/en/stable/

The package supports PyQt & PySide: 

pip3 install pyqtdarktheme

In the Python script, add this: 

import qdarktheme

## Program blah blah


app = QApplication(sys.argv)
# Apply dark theme.
qdarktheme.setup_theme()
