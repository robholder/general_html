Vimwiki - Clone to New Machine

Set up ssh keys with GitHub

If it's a newly set up machine (i.e. Git not set up yet):

Set git configs: 

git config --global user.email "myname@gmail.com"
git config --global user.name "MyName""

Setting Up SSH (to Communicate with GitHub)

  • Setup up SSH keys

Cloning Vimwiki From GitHub

cd
mkdir ~/vimwiki_dir
cd !$
git clone git@github.com:username/general_html.git
git clone git@github.com:username/general.git

Add the custom deployment script.

I created a 'bin' directory in my home directory: 

mkdir ~/.local/bin

Then in the .zshrc added this script to the PATH: 

# Add ~/.local/bin to the PATH variables:
# To view: echo -e ${PATH//:/\\n}
export PATH="$PATH:$HOME/.local/bin"

Make A Backup/Commit/Push Script

Now the mvwdeploy (or whatever for the OS) will run from the vimrc shortcuts.
