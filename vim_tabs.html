Vim - Tabs

The :tab command in Vim comes with several arguments and subcommands that expand its functionality. Here's a breakdown of some useful ones:

Managing Tabs

  • :tabnew [filename] --> Opens a new tab and optionally loads a file into it. For example:
    • :tabnew myfile.txt
  • :tabclose or :tabc Closes the current tab.
  • :tabonly or :tabo Closes all other tabs except the current one.
  • :tabnext or :tabn Moves to the next tab.
    • You can also specify a tab number like :tabn 3 to jump to the third tab.
  • :tabprevious or :tabp Moves to the previous tab.
  • :tabfirst --> Takes you to the first tab.
  • :tablast Takes you to the last tab.

Working with Tab Pages

  • :tabs Displays a list of all open tabs along with the buffers loaded in them.
  • :tabmove [N] --> Moves the current tab to position N (starting from 0). For instance:
    • :tabmove 2 --> moves the tab to the third position.
  • :tabe [filename] --> Shortcut for :tabedit. Opens a new tab with the specified file.

Converting Buffers to Tabs

You can use the :tab ball command to open all currently listed buffers in individual tabs. Here's how it works:

  • Make sure you have multiple buffers open:
    • You can check with :ls to list all your open buffers.

Run the command:

:tab ball --> This will create a new tab for each open buffer, effectively spreading them out across tabs.

It's a fantastic way to organize your buffers visually using tabs if you prefer working that way.
