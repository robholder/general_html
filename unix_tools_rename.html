Rename

“Rename” install on MacOS

This is the command prompt bin to rename files: 

brew install rename

MacOS Version

To rename like the olden days version, use the -s flag: 

rename -s <old-string> <new-string> <file-search-term>

# Example:
rename -s test temp test*

(Older?) Linux Versions

rename <old-string> <new-string> <file-search-term>

# Example:
rename test temp test*

NOTE: There are versions that work like sed with s/blah/such/ type syntax.
