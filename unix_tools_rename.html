Rename

A binary utility on some nix platforms, to rename files from the commandline.

There are more than a few binaries with this name out there. Each has it's own usage variations.

“Rename” install on MacOS

This is the Homebrew install command for MacOS: 

brew install rename

Current M3 MacOS Version

To batch rename, use the -s flag: 

rename -s <old-string> <new-string> <file-search-term>

# Example:
rename -s test temp test*

(Older?) MacOS & Linux Versions

The version found on my Intel Mac (installed by hand years ago): 

rename <old-string> <new-string> <file-search-term>

# Example:
rename test temp test*

NOTE: There are versions out there that work like sed with s/blah/such/ type syntax.
