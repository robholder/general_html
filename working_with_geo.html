Working With Geometry

  • You can drag from the shelf buttons to the Scene node graph to make primitives

Transform Tools

  • T:
    • Move
  • R:
    • Rotate
  • E:
    • Scale
  • S key:
    • allows you to re-select areas
    • then continue to transform
  • Select manipulator to move
  • MMB to freely move
  • Try not to scale at geo level:
    • instead, jump inside
  • Value ladder:
    • use MMB on geo
    • Select slider magnitude
  • Edit Nodes:
    • created when you move geo
  • Edit:
    • T/R/E
  • Slide On Surface:
    • L
  • Peak:
    • H
  • Sculpt:
    • B

Selection Options

  • 2:

  • 3:
    • edges
  • 4:
    • faces
  • Shift A:

Edit Options

Hit T key first, then:

  • Shift-C:
    • Make Circle
  • Shift-E:
    • Evenly Space Selection
  • Shift-R:
    • Relax Selection
  • Shift-S:
    • Straighten Selection
