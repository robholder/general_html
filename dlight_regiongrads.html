Region Grads

Setup

Light Region Grad

  • In Palette:
    • in Light - RMB on tiny sliver across top of expanded light panel
    • Add User Variable > regionGrad
    • Rename - purpose - RG:
      • eg. darkenRG
    • Open 'Region Grad' panel:
      • Region Grad Enable - check 'on'
      • RegionGrad table:
        • add darkenRG
        • Modes:
          • Add
          • Mix
          • Multiply (usual option):
            • 1 normal
            • > 1 brighter
            • < 1 darker
        • ~ (tilde) makes the grad NEGATIVE:
          • eg. ~darkenRG
        • Fallof interpolatoin is set in the regionGrad 'var' (top of light panel)
  • You can add any number of RegionGrads under one variable
  • Region grads can be copied from light to light:
    • RMB on variable (regionGrad) and 'copy'

ColorMatte Region Grad

  • Create 'colormatte' in Shot mods palette
  • Create 'colormatte in the dialog (not Char matte nor env matte)
  • Bind colormatte to items (in Bindings palette)
  • Add Region Grad like the light process above.

Visualize RG

  • Select locator (of light)
  • Shelf > Lighting > visualize RG and delete visualize RG
  • Select the region grad locator:
    • Shape attribute:
      • Extra attributes:
        • Hard Expand:
          • overall core size
        • Soft Falloff:
          • ramp soft falloff size
        • Size XYZ:
          • Changes core to lozenge, or more cubic
          • Size X
          • Size Y
          • Size Z
  • Region Grad appear in Maya's Outliner under a RegGradManip
  • Visualizer disappears when you save a scene

tip

  • Snap to vertex/locator:
    • Snap to points (v)
    • MMB drag to snap to locator

Nuke RegionGrad

  • Breakout utility > pos
  • Regiongrad node
  • move Sample position, and 'Set Position'
  • Values in Nuke are equivalent to DLight
