a.m.

Documented a great tip about tcsh aliases with arguments from Fonzie. 

Hey team, i don't know if anyone else hates typing things repetitively as much as I do, but I created two aliases for my ~./tcshrc file to help me go into the rendered EXR frames quicker, so i wanted to share them here if they're helpful. This was thinking on the MIR side of things, not sure if other studios are using different paths, but so far they seem to work for our MIR shots.
alias f 'cd /media/avoa/\!:1/\!:2/comp/frames/'
alias fv 'cd /media/avoa/\!:2/\!:3/comp/frames/\!:2_\!:3_comp_v\!:1/1/mono/default/'
Basically, if you want to go to a shot frames directory just do:
f 4650 0010
eg: f <seq> <shot>
And if you know the version of the EXRs you want to look at:
fv 03 4650 0010
eg: fv <version> <seq> <shot>
Don't forget to source your ~./tcshrc file after you save it, or open a new terminal to test these out.
