These use 'potrace' and ImageMagick. We have ImageMagick on our tool path (I got access via an email thread this week), but I've been trying to get 'potrace' on my local machine to assess.
I believe these are relatively accessible tools.
Potrace is an underlying ingredient of InkScape, which I believe used to be part of the Linux Rocky install but was recently removed from the default Linux install for size considerations.
The tools take a sequence (RGB) and can output to another RGB sequence (after doing a commandline round trip of pure black and white, svg (vectorized), then re-rasterized to RGB and output. All that happens on the commandline with one command.
I'd like to experiment with the best command args to get the best result, but it is encouraging.

A new Nuke utility tool is available - CellSelect
* Built especially for those wishing Nuke had a bucket tool for flat color selections (based on color/value boundaries).
* This tool creates mattes based on contiguous pixels - up to boundaries.
* Useful for selecting individual cells or regions of a color, rather than all pixels of a picked color value.

Available from Nuke's top menu:
Paramount Tools > Custom Scripts for avoa > Tools > CellSelect

Take a look at the CellSelect introduction page for usage:

https://paramount.atlassian.net/wiki/spaces/PFA/pages/580027139/CellSelect
