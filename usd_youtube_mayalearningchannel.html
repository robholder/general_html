  • USD is a file format developed by Pixar
  • USD is a file format that is used to distribute and share data across studios and software
  • Prims:
    • Primitives
    • Smallest divisible component
      • This is defined by the creator
      • In Maya, you CAN dive in and edit, then merge back to the USD prim
    • In Maya, you cannot break down a prim
    • Prims can have variants

Stage

This is the upper level, where all the USD elements are layered.

Opinions

  • Adjustments to layers (perhaps Layout) become higher opinions:
    • Override
  • Higher opinions do not overwrite a lower opion/layer:
    • Muting an opinion restores a lower opinion/layer
  • Maya has a USD Layer panel - where opinions appear as layers with mute options
