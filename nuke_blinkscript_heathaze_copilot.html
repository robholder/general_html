Copilot - BlinkScript - Heat Haze

Heat haze occurs due to light refraction caused by temperature differences, making objects appear wavy or distorted. This effect can be simulated by applying a noise-driven distortion to pixel positions.

BlinkScript Code: Heat Haze

kernel HeatHaze : ImageComputationKernel<ePixelWise>
{
    ImageParam src;
    param: float intensity;
    param: float time;  // Animation factor

    void process()
    {
        float2 uv = getNormalizedCoord();
        
        // Generate time-dependent distortion
        float noiseX = sin(uv.y * 10.0 + time * 2.0) * intensity * 0.01;
        float noiseY = cos(uv.x * 10.0 + time * 2.0) * intensity * 0.01;
        
        // Apply distortion
        float4 result = src(uv + float2(noiseX, noiseY));

        output() = result;
    }
};

How It Works

  • Uses sine and cosine functions to create wavy distortions that shift pixels based on heat fluctuations.
  • Animation control (time) lets the effect evolve dynamically (ideal for animated sequences).
  • Intensity parameter (intensity) adjusts the strength of the distortion.
