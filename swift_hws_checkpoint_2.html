Hacking With Swift - Checkpoint 2

Challenge

This time the challenge is to:

  • Create an array of strings
  • Prints the number of items in the array
  • Prints the number of unique items in the array.

My Attempt:

// Create String Array of words:
var myArray: [String] = ["Hello", "I", "am", "a", "sentence", "with", "the", "word", "Hello", "in", "it"]

// print the number of elements:
print(myArray.count)
// >> 11

// Create a Set made of the elements of the string array
let mySet: Set<String> = Set(myArray)

// Print the unique items (elements of the Set):
print(mySet.count)
// >> 10

Alternate:

var myArray = ["Hello", "I", "am", "a", "sentence", "with", "the", "word", "Hello", "in", "it"]
print(myArray.count)
// >> 11
print(Set(myArray).count)
// >> 10

Hints from the tutorial:

You should start by creating an array of strings, using something like let albums = ["Red", "Fearless"]

  • You can read the number of items in your array using albums.count.
  • Count also exists for sets.
  • Sets can be made from arrays using Set(someArray)
  • Sets never include duplicates.
