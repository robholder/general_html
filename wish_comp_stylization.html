Key & Rim Tags

  • Rim / Key Tags:
    • There are some copy & pastable nodes that 'tag' lights as Key or Rim.
    • These are shuffle nodes that duplicate the RGBA into channels for Key-R, Key-G, Key-B, Key-A (and likewise for the Rim).
    • Used for controling an emission/light's inclusion in munge, SSAO (screen spaced ambient occlusion):
      • Example: Rims are split away for separate custom 'munging'.

Kuwuhara

This blur gives a simplified brush-stroke style look (without ruining the indegrity of edges/geo).

Bilateral (Blur)

  • A Bilateral blur, but uses a brushy input image to control the look and contain the effect:
    • Watercolor splodge guide
    • Scene depth
    • Geo ID map

There is a No Munge map here too (from look)

Luminance Shuffle

This merges the RGBA into luminance.red, luminance.green etc. (like the Key/Rim shuffles in the Tag step).

CC Adjustments

Backdrop to contain all CC adjustments. (Although you can manipulate the emission inputs also.)

Layers of Interest

  • Manipulate depth map
  • Radials, region-grads etc.
  • Result is shuffled into 'LayersOfInterest' layer

Edge Pooling

  • Another watercolor effect:
    • SurfaceColor:
      • Adjust blurs of map channels
    • Luminance:
      • Adjust blurs of map channels
    • Keylight edge:
      • use luminance puller to create matte
  • Adds edges to changes on:
    • light edges
    • geometry edges
    • Doesn't on freckles hopefully!!
  • Two pooling stages:
    • Large
    • Small

SSAO (Screen Space Ambient Occlusion)

  • Creates drop shadow style AO
  • Adjustements:
    • bias
    • radius
  • Add a little surface colors
  • Use the Keylight tagged layer to punch out key AO

Luminance

Screen Space/Camera Oriented Textures

A few texture maps use the camera facing lattice trick to add textures:

  • Watercolor:
    • mostly on darker areas (not white)
  • Paper:
    • mostly only paper in bright areas.

The volumes are fed into these too to add more texture to the volumes.

Textures are reduced on the characters.
