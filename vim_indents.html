Vim Learning

Indents

  • >> --> indent current line
  • << - --> de-indent current line
  • >j --> indent including line below
  • >23j --> indent 23 lines
  • 5>> --> indent 5 lines
  • 5== --> un-indent lines

Indent Groups (blocks, tags, brackets)

  • gg=G -->re-indent entire buffer
  • >% --> indent a block with () or {}
  • >ib --> indent inner block with ()
  • >at --> indent a block with <> tags
  • <% --> de-indent a block with () or {}
  • >% --> increase indent of a braced or bracketed block (place cursor on brace first)
  • =% - --> re-indent of a braced or bracketed block (place cursor on brace first)
  • <% --> decrease indent of a braced or bracketed block (cursor on brace)
  • ]p --> paste text, aligning indentation with surroundings

Indenting Selections

  • > --> Indent selected lines
  • < --> De-indent selected lines

Indenting in Insert Mode

  • <cntl>t --> indent while in insert mode
  • <cntl>d --> de-indent while in insert mode
  • <cntl>o --> enter normal mode (from insert mode) to make one command
