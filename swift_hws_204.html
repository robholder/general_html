Hacking With Swift - 204 - Challenge 0 (My own)

This challenge is my own. The current app is really frustrating, in that short taps and long taps are very flaky. Plus, when you create a location, you can't delete it.

I want to show a dialog with:

  • Edit
  • Delete
  • Cancel

...instead of just the edit page.

Ai Insight

Can I have a dialog instead that offers to edit, delete or cancel?

You absolutely can! In SwiftUI, the perfect UI component for this is a Confirmation Dialog (historically called an Action Sheet). It slides up from the bottom of the screen with a clean menu of choices, which is excellent for a "Manage Pin" workflow.

To build this cleanly, we need to introduce a second piece of state to manage when the dialog appears, add a delete function to your ViewModel, and chain the modifiers together.

Step 1: Add a Delete Function to your ViewModel

First, open your ViewModel class and add a quick function so it knows how to remove a specific location: 

func delete(location: Location) {
    if let index = locations.firstIndex(of: location) {
        locations.remove(at: index)
        save() // Keep your local storage synchronized!
    }
}

Step 2: Add Dialog State to ContentView

Right now, your .sheet triggers automatically the second viewModel.selectedPlace is given a value. Because we want a dialog to show up before that edit sheet appears, we'll use a boolean @State variable to control the dialog's visibility, plus a separate storage property for the sheet.

Update your ContentView state area to look like this: 

@State private var viewModel = ViewModel()
@State private var showingOptions = false   // Controls the dialog menu
@State private var placeToEdit: Location?   // Controls the final edit sheet

Step 3: Wire up the Map and Modifiers

Now, rewrite your layout sequence. When the user long-presses, we toggle showingOptions = true. 

MapReader { proxy in
    Map(initialPosition: startPosition) {
        ForEach(viewModel.locations) { location in
            Annotation(location.name, coordinate: location.coordinates) {
                Image(systemName: "star.circle")
                    .resizable()
                    .foregroundStyle(.red)
                    .frame(width: 44, height: 44)
                    .background(.white)
                    .clipShape(.circle)
                    .simultaneousGesture(LongPressGesture(minimumDuration: 0.2)
                        .onEnded { _ in
                            // 1. Lock in the active location choice
                            viewModel.selectedPlace = location
                            // 2. Trigger the options dialog menu
                            showingOptions = true 
                        }
                    )
            }
        }
    }
    .onTapGesture { position in
        if let coordinate = proxy.convert(position, from: .local) {
            viewModel.addLocation(at: coordinate)
        }
    }
    // MARK: - 1. The Options Menu Dialog
    .confirmationDialog(
        "Manage Location", 
        isPresented: $showingOptions, 
        titleVisibility: .visible, 
        presenting: viewModel.selectedPlace
    ) { place in
        Button("Edit Details") {
            placeToEdit = place // Hands off to the sheet
        }
        
        Button("Delete Pin", role: .destructive) {
            viewModel.delete(location: place)
            viewModel.selectedPlace = nil // Reset state cleanup
        }
        
        Button("Cancel", role: .cancel) {
            viewModel.selectedPlace = nil // Reset state cleanup
        }
    } message: { place in
        Text("What would you like to do with \(place.name)?")
    }
    // MARK: - 2. The Final Edit Sheet
    .sheet(item: $placeToEdit) { place in
        EditView(location: place) {
            viewModel.update(location: $0)
            viewModel.selectedPlace = nil // Final pipeline cleanup
        }
    }
}

Why this works so beautifully

  • role: .destructive: Giving your delete button this role automatically renders the text in red and positions it natively according to iOS design guidelines.
  • role: .cancel: The system styles this cleanly as a bold, separated button at the bottom. Tapping it or anywhere outside the dialog automatically drops the menu without you writing extra dismissal code.
  • The Hand-Off Pipeline: The dialog uses viewModel.selectedPlace. If you tap "Edit", it copies that value over into placeToEdit. Because the .sheet modifier is listening closely to placeToEdit, the sheet slides up seamlessly the exact moment the options dialog slides away!

ContentView-ViewModel.swift

import CoreLocation
import Foundation
import LocalAuthentication
import MapKit

// Make extension to keep ViewModel easy to call.
// Just call it within the ContentView in this case.
extension ContentView {
    @Observable
    class ViewModel {
        // private(set) means other parts of the code can read it, but can't set it
        private(set) var locations: [Location]
        var selectedPlace: Location?
        var isUnlocked = false
        
        let savePath = URL.documentsDirectory.appending(path: "SavedPlaces")
        
        init() {
            do {
                let data = try Data(contentsOf: savePath)
                locations = try JSONDecoder().decode([Location].self, from: data)
            } catch {
                locations = []
            }
        }
        
        func save() {
            do {
                let data = try JSONEncoder().encode(locations)
                try data.write(to: savePath, options: [.atomic, .completeFileProtection])
            } catch {
                print("Unable to save data.")
            }
        }
        func addLocation(at point: CLLocationCoordinate2D) {
            let newLocation = Location(
                id: UUID(),
                name: "New location",
                description: "",
                latitude: point.latitude,
                longitude: point.longitude
            )
            locations.append(newLocation)
            save()
        }
        
        func update(location: Location) {
            guard let selectedPlace else { return }
            
            if let index = locations.firstIndex(of: selectedPlace) {
                locations[index] = location
                save()
            }
        }
        
        func authenticate() {
            let context = LAContext()
            var error: NSError?
            
            if context.canEvaluatePolicy(.deviceOwnerAuthenticationWithBiometrics, error: &error) {
                let reason = "Please authenticate yourself to unlock your places"
                
                context.evaluatePolicy(.deviceOwnerAuthenticationWithBiometrics, localizedReason: reason) { success, authenticationError in
                    if success {
                        // Force the UI update to hop back to the safe main thread
                        Task { @MainActor in
                            self.isUnlocked = true
                        }
                    } else {
                        // error handling
                    }
                }
            } else {
                // no biometrics
            }
        }
        
        // Challenge zero - option dialog
        func delete(location: Location) {
            if let index = locations.firstIndex(of: location) {
                locations.remove(at: index)
                save() // Keep your local storage synchronized!
            }
        }

    }
}

ContentView.swift

import MapKit
import SwiftUI

struct ContentView: View {
    let startPosition = MapCameraPosition.region(
        MKCoordinateRegion(
            center: CLLocationCoordinate2D(
                latitude: 56,
                longitude: -3),
            span: MKCoordinateSpan(
                latitudeDelta: 10,
                longitudeDelta: 10)
        )
    )
    
    @State private var viewModel = ViewModel()
    @State private var showingOptions = false // Controls the dialog menu
    @State private var placeToEdit: Location? // Controls the final edit sheet
    
    
    var body: some View {
        if viewModel.isUnlocked {
            MapReader { proxy in
                Map(initialPosition: startPosition) {
                    ForEach(viewModel.locations) { location in
                        Annotation(
                            location.name,
                            // Use computed property from Location struct
                            coordinate: location.coordinates) {
                                // Custom 'pin' icon:
                                Image(systemName: "star.circle")
                                    .resizable()
                                    .foregroundStyle(.red)
                                    .frame(width: 44, height: 44)
                                    .background(.white)
                                    .clipShape(.circle)
                                    .simultaneousGesture(LongPressGesture(minimumDuration: 0.2)
                                        .onEnded { _ in
                                            // 1. Lock in the active location choice
                                            viewModel.selectedPlace = location
                                            // 2. Trigger the options dialog menu
                                            showingOptions.toggle()
                                        }
                                    )
                            }
                    }
                }
                .onTapGesture { position in
                    if let coordinate = proxy.convert(position, from: .local) {
                        // Run the addLocation function in the viewModel
                        viewModel.addLocation(at: coordinate)
                    }
                }
                // MARK: - 1. The Options Menu Dialog
                .confirmationDialog(
                    "Manage Location",
                    isPresented: $showingOptions,
                    titleVisibility: .visible,
                    presenting: viewModel.selectedPlace
                ) { place in
                    Button("Edit Details") {
                        placeToEdit = place // Hands off to the sheet
                    }
                    
                    Button("Delete Pin", role: .destructive) {
                        viewModel.delete(location: place)
                        viewModel.selectedPlace = nil // Reset state cleanup
                    }
                    
                    Button("Cancel", role: .cancel) {
                        viewModel.selectedPlace = nil // Reset state cleanup
                    }
                } message: { place in
                    Text("What would you like to do with \(place.name)?")
                }
                // MARK: - 2. The Final Edit Sheet
                .sheet(item: $placeToEdit) { place in
                    EditView(location: place) {
                        viewModel.update(location: $0)
                        viewModel.selectedPlace = nil // Final pipeline cleanup
                    }
                }
            }
        } else {
            Button("Unlock Places", action: viewModel.authenticate)
                .padding()
                .background(.blue)
                .foregroundStyle(.white)
                .clipShape(.capsule)
        }
    }
}
