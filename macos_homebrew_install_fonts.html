Font Install with Homebrew

Installing Nerd Fonts on macOS is most efficient via Homebrew, but manual installation is a reliable backup if a specific version is needed from the Nerd Fonts website.

Option 1: Install via Homebrew (Recommended)

Homebrew has moved fonts into its main repository, so you no longer need to "tap" a separate font repository.

  • Search: To find the exact name for a font (e.g., "Hack"), use: 
    brew search --cask nerd-font | grep "hack"
  • Install: Use the font--nerd-font format: 
    brew install --cask font-hack-nerd-font

Example common names: font-jetbrains-mono-nerd-font, font-fira-mono-nerd-font, font-caskaydia-cove-nerd-font.

Option 2: Manual Installation (From .zip)

If you download a font directly from the Nerd Fonts Downloads page:

  • Unzip: Double-click the downloaded .zip file in your Downloads folder to extract the font files.
  • Locate Files: Open the folder and look for the .ttf (TrueType) or .otf (OpenType) files.
  • Install via Font Book:
    • Double-click the font file and click Install in the window that appears.
    • Alternative: Open the Font Book app (built-in to macOS) and drag the font files directly into the window.
  • Verification: Once installed, the font will be available system-wide for WezTerm to detect.

Pro-Tip for WezTerm Users

After installing a new font, run your "clean" command again to find the exact string for your my_fonts table:

{{sh wezterm ls-fonts --list-system | grep "wezterm.font" | sed -E 's/wezterm.font\("([^"]+)",.*/\1/' | sort -u }}}
