Turntable Setup

  • Circle:
    • switch to polygon
    • increase divisions
    • set horizontally
  • Sphere (temp object to view)
  • Camera:
    • Constrain shelf tool:
      • set camera
      • constrain to circle (target)
      • aim at sphere
      • no orient obj
    • Add transform in Constraint network:
      • To raise camera
  • Constraint Path expression (path node):
    • From default:
      • $\(FF/\)$RFFEND
      • (This is value from 0 to 1)
    • To:
      • fit($\(FF/\)$RFEND, 0, 0.5, 0, 1)
      • (This is relative to the framerange duration)
      • Camera will encircle subject for half the frames then stop
