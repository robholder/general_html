Git - Converting A Cloned Repo to a New Remote Repo

To convert a local repo clone into a separate new repo on GitHub, you need to update the local repository's remote URL to point to a new GitHub repository and then push its contents.

Steps to Convert and Push to a New GitHub Repo

Create a New GitHub Repository

  • Go to GitHub and create a new repository.
  • Crucially, do not initialize it with a README, license, or .gitignore file to avoid merge conflicts with your existing history.
  • Once created, copy the new repository's URL (HTTPS is recommended) from the "Quick Setup" page.

Navigate to Your Local Repository

  • Open your terminal or command prompt.
  • Change the current working directory to the root of your local cloned repository using the cd command.

Update the Remote Origin URL

Remove the old remote origin URL association: 

git remote remove origin

Populating a new GitHub repo (blank) with the local Repo:

HTTP method

Add the new GitHub repository's URL as the new origin: 

git remote add origin [NEW_REPO_URL]

Verify that the remote URL has been set correctly by running: 

git remote -v

SSH Method

Copy the ssh commands from the new repo setup guide 

git remote add origin git@github.com:<username>/<my_git_repo>.git
git push -u origin main

Push Your Local Repository to GitHub

  • Push your local commits and branches to the new remote repository: 
git push -u origin main
  • If your default branch is named something other than main (like master), replace main with your branch name in the command above.
  • If you encounter an error about unrelated histories (which can happen if you accidentally initialized the GitHub repo with a file), you may need to use the --allow-unrelated-histories flag during a pull/merge operation, or force push if you are sure you want to overwrite anything on the remote. A common and safe method is to simply force push the first time: 
    git push -u -f origin main

Your local repository's code and history will now be uploaded to the new, separate GitHub repository.
