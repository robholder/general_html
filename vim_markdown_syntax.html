Markdown Syntax

Markdown is a lightweight markup language that helps format text with simple syntax. Here’s a quick reference:

Basic Formatting

Bold

**bold text** or __bold text__

Italic

*italic text* or _italic text_

Strikethrough

~~strikethrough~~

Inline code

`inline code`

Headings

Use # for headings: 

# Heading 1
## Heading 2
### Heading 3

Lists

Unordered List:

- Item 1
- Item 2
- Item 3

Ordered List:

1. First item
2. Second item
3. Third item

Links & Images

  • Link: [Text](URL)
  • Image: ![Alt text](image_url)

Images

In GitHub-flavored Markdown, you can include images using the following syntax:

Basic Image Syntax

![Alt text](image_url)
  • Alt text is a description of the image (for accessibility).
  • image_url is the direct URL to the image.

Adding an Image from a GitHub Repository

If the image is stored in your repository, use a relative path: 

![Project Logo](images/logo.png)

This works best when the images/ folder is inside your repo.

Linking an Image to Another Page

If you want to make the image clickable: 

[![Alt text](image_url)](link_url)

Now, clicking the image will take you to link_url.

Resizing Images on GitHub (HTML Approach)

Markdown itself doesn’t support resizing images, but GitHub allows HTML inside Markdown: 

<img src="image_url" alt="Description" width="300">

This sets a custom width.

Blockquotes

In Markdown, blockquotes are created using the > symbol at the beginning of a line. Here's how: 

> This is a blockquote.
> It can span multiple lines.

This will render as: 

This is a blockquote.
It can span multiple lines.

If you want nested blockquotes (quotes within quotes), you just add another >: 

> This is the main quote.
>> This is a nested quote.

Which appears like this: 

This is the main quote.
    This is a nested quote.

Blockquotes are great for emphasizing text or quoting someone in documents.

Code Block

To create a code block in Markdown, you use triple backticks (```) before and after the code. Here's an example: 

```
def hello():
    print("Hello, Markdown!")
```

That will render like this: 

def hello():
    print("Hello, Markdown!")

If you're working with a specific programming language, you can specify it after the first triple backticks: 

```python
def hello():
    print("Hello, Python!")
```

This helps with syntax highlighting in many markdown editors.

Tables

| Column 1 | Column 2 | Column 3 |
|----------|----------|----------|

That’s the essentials! Markdown is great for notes, documentation, and formatting text files.
