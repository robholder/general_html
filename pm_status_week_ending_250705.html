SplineBlur Rebuild

Figured out an additional expression to void rather than error if there is no Custom Noise input into the group:

Generic:

[lsearch [channels this] rgba.alpha] > -1 + 0*frame

My case:

[lsearch [channels noise] rgba.alpha] > -1 + 0*frame

The 0*frame forces Nuke to re-assess network updates (like plugging or unplugging nodes). Without this addition, it only assesses once, and ignores network connection/disconnection actions).
