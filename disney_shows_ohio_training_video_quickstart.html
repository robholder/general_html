  • Switch 'Layout' to Lighting_MultiShot:

    • Inside will be a template.

Shot/Packet Lighting

  • In the 'Lights - Lightbox' area:
    • Drop a Lightbox node:
      • tab and type lightbox

Lightbox Node

The lightbox node has a parameter panel which is a template hierarchy.

Create lights in here, under the parent of appropriate headings (for example a new light called all_KEY_lgt: 

lgtAsset:
  | matPalette:
      | asset_materials
      | volume_materials
lrigLOCAL:
    | matPalette:
        | lr_sunSky
        | lr_stage:
            | all_KEY_lgt
        | lr_practical
        | shot_modifactions

Lights - Broken Out Palettes

This is a backdrop with a structure which allows for the laying down of lights in a similar hierarchy, but in a split out way.

Create dpipe lights here.

Hook up whichever structure you've chosen.

Switch lights on with:

  • enabled
  • emissive

Colors

When viewing color palettes, turn off the color management icon (a triangle curve icon).

Emission Categories

Give each light an emission category which equates to a per emission AOV.

Snapping Light To Geometry

  • Select light
  • In 3d view:
    • select the magnet icon on the left vertical panel
    • middle click on geo
    • light should snap to geo

Moving Lights

  • Move with manipulators:
    • Regular manipulators:
      • translate
      • rotate
    • Extra manipulators:
      • Select light (using the transform handle on the light listing after name):
        • <Shift> L
      • You'll get extra controls:
        • target & rotation pivots
      • With the magnet on (right vert panel):
        • middle click on geo to snap the target.

Or:

  • Shelf tabs:
    • Lighting tools:
      • Look through selected
      • Lock viewport to camera

When going back to the shot cam, hit unlock (lock icon on r right vertical panel) to revert to the correct camera position. (Leaving the lock on doesn't change the shot camera's position data.)

Viewport Renders

In viewport, look for the Persp dropdown, and choose:

  • Houdini VK
  • Chimera:
    • Proprietary Houdini VK Replacement
    • Turn on Scene lighting icon (right vert panel)
    • Turn off headlight icon (right vert panel)
  • ion:
    • It's a Render Preview (for lighting)
    • Estimate of full Renderer (it's a Hyperion proxy)
    • XGen not supported
    • materials are inaccurate
  • Karma XPU
  • Karma

Interactive Rendering

This is Helios.

There is an interactiveRender node that gets used for this:

  • Lighting Tool (shelf):
    • Interactive Render

Parameters

Usual controls for Hyperion here in the parameters panel.

These settings do not get passed to genuine renders.

If you make any changes (lighting) after the initiation of a render, as long as Helios is still active, you can click the Hyperedit button in the parameters to push the changes to Helios.

To keep active

Make sure the Edit > Preferences > Terminate on Completion to 'OFF'.

The progress area on the frame will be yellow if active.

Helios controls

  • Exposure slider
  • Click on geo to see item names
  • RIVit Catalog:
    • Hit i in the window to create a catalog and add the current image.
    • Each time you hit i, a new capture will be added to the catalog.

Collections

In order to make bindings to geometry/parts more manageable, you can create dpipe Collections.

There is a backdrop in the scene graph template, and collections can be organized here.

dpipCollection

  • Create with tab and dpipCollection
  • Drop into the pipe flow

This node has a parameters panel (that is lifted from DLight). 

* shot
    * CulledElements
    * CulledMeshes
    * ElemsChar
    * ElemsComp
    * ElemsCrowd
    * ElemsFx
    * ElemsProp
    * ElemsSet
    * IndirectElems
    * IndirectMeshes

These default collections should be quite accurate to their descriptions.

Create your own by:

  • Clicking on the orange collection+ icon
  • Rename in the panel:
    • ch_all_CL for example
    • en_all_CL as another example

Adding items

  • Select the collection from the list.
  • Select items from the Stage Graph Tree or viewport.
  • In the Collections Bindings panel (at the base of the Collections panel):
    • Click the blue cube with the +

Selecting Collection Items

  • Select the collection name
    • Hit the orange collection icon with the cursor motif.
