Shortcuts

Network Editor

Keystroke action
TAB Node menu (and type to narrow down search)
<shift> TAB Node menu, but append the new node to the selected node, and the render flag is set
R Set selected node's Display Flag and Render Flag
Q Toggle selected node's 'bypass'
E Toggle selected node's 'template' flag
<alt> LMB Drag to duplicate selected node
Y LMB Drag across wire/connections to delete
J LMB Drag across nodes to connect with one swipe
<shift> LMB Move node and all above with it
<cntl> LMB Move node and all below with it
L Vertically align selected node's branch
A LMB (drag vertically) Align selected nodes vertically
A LMB (drag horizontally) Align selected nodes horizontally
<cntl> B Maximize panel
P Show parameter panel as inset on Network Editor
   

| MMB| On a network node to show the 'i' pane.

MacOS  
<command> # Set Quickmark # (quick navigation), e.g. 'command + 2'
<cmd> MMB On a network node to show the 'i' pane, but keep open
Linux/Windows  
<cntl> # Set Quickmark # (quick navigation), e.g. 'control + 2'
<cntl> MMB On a network node to show the 'i' pane, but keep open
   
# (number keys) Jump to Quickmark location, e.g., '2'
   
F2 Rename selected node
select nodes <alt> LMB Drop pre-wired 'Merge'. Select nodes, alt click below in empty space - a merge will appear, all connected

Parameter Panel

Keystroke Action
In cell MMB drag Invoke increment ladder
On name MMB drag Invoke increment ladder for all parameter cells for attribute
In cell <scroll wheel> Increments of 0.1
In cell <cntl><scroll wheel> Increments of 1
In cell <shift> <scroll wheel> Increments of 0.01
In cell <alt> LMB Set keyframe
In cell <cntl> LMB Remove expression
In cell <cntl> E Open expression editor window
In cell <cmd> E Open expression editor window (MacOS)

Viewport

Keystroke Action
<enter> Handle tool
<esc> Selection tool
<space> *MB To invoke navigation when in Handle tool
W Toggle wireframe/solid
<shift> W Toggle wireframe visible
<space> 1 Persp view
<space> 2 Orthographic top
<space> 3 Orthographic front
<space> 4 Orthographic right
n Select All
<shift> n Deselect All
<shift> F1 Show/Hide the HUD help/hints

(Any) Node View

Keystroke Action
<shift> R switch inputs
