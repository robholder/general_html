Wezterm - Settings - Line Height

Setting the line height to anything but '1' can render the fonts horribly on some systems. 

-- Horizontal and Vertical Spacing
config.cell_width = 1
config.line_height = 1.1

Set to '1' for cleaner font rendering: 

-- Horizontal and Vertical Spacing
config.cell_width = 1
config.line_height = 1
