Push to GitHub

Create A New Repo - and Push

echo "# My README comment" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/rholder/myrepo.git
git push -u origin main

Push an existing Repo

git remote add origin https://github.com/rholder/myrepo.git
git push -u origin main

To Push All Branches to GitHub

git push --all

To Pull from a GitHub Repo

git fetch
git pull

or 

git pull
