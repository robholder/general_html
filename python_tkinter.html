I installed Tkinter on the system Python (3.9) and it works, but not for my pyenv 3.11 versions. I installed on 3.12 for example with no particular issues.

As described on my Mac Diary entry ('TkIner Hell' below), tkinter works with Python 3.9.6 (current Mac Sonoma system) and Python 3.12.4.
