MacOS - Mapping 'jk' to in Xcode with Karabiner-Elements

Install Karabiner-Elements, and allow all the premissions it asks for from the System.

Xcode - 'jk' to 'esc'

Xcode has a basic Vim emulation option. Editor > Vim Mode.

This mod makes a quick roll of j & k to trigger an escape key event.

  • Add this to the Karabiner-Elements Settings > 'Complex Modifications': 
{
    "description": "Change jk to escape (Only in Xcode)",
    "manipulators": [
        {
            "conditions": [
                {
                    "bundle_identifiers": [
                        "^com\\.apple\\.dt\\.Xcode$"
                    ],
                    "type": "frontmost_application_if"
                }
            ],
            "from": {
                "simultaneous": [
                    { "key_code": "j" },
                    { "key_code": "k" }
                ],
                "simultaneous_options": {
                    "detect_key_down_uninterruptedly": true,
                    "key_down_order": "insensitive",
                    "key_up_order": "insensitive"
                }
            },
            "parameters": { "basic.simultaneous_threshold_milliseconds": 100 },
            "to": [{ "key_code": "escape" }],
            "type": "basic"
        }
    ]
}
