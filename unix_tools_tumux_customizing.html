Tmux - Customizing Basics

The Config File

Tmux has a lot of "long-form vs. short-form" pairs to save you typing:

  • bind-key → bind
  • set-option → set
  • set-window-option → setw
  • new-session → new

Short-form keeps your .tmux.conf cleaner and easier to read. When you're looking at a file with 50 shortcuts, having bind at the start of every line makes the actual keys (like r, h, j) stand out much better than the longer version.

  • Pro Tip: In Tmux, you can almost always chop off the second half of a command name and it will still work!

To Set Keys

  • The default keys are:
    • <Ctrl> b

You can set your own keys, and any number of keys.

Suggestion:

  • In the following lines, we set j and f as keys.
  • -g - Global
  • Note: prefix combos need to be numbered - as in 'prefix2' 
set option -g prefix C-j
set option -g prefix2 C-f

Reload .tmux.conf File In Tmux

  • Easy config reload - r

bind r source-file ~/.tmux.conf \; display-message "Config reloaded!"

Mouse Mode

# Mouse Mode
set -g mouse on

This allows for:

  • Drag pane divisions to resize
  • Click on pane to select
  • Click on windows (status bar) to select

Switch Panes with 

# Set easier window split keys
bind v split-window -h
bind h split-window -v

Change Windows with /<Arrow-Right

# Use Shift arrow to switch windows
bind -n S-Left previous-window
bind -n S-Right next-window

Synchronize Panes (To run commands in sync)

This is an advanced concept, where you can open panes for each machine/server/directory and run commands simultaneously, with all panes updating with command input:

  • Synchronize panes

bind y set-window-option synchronize-panes\; display-message "synchronize mode toggled."
