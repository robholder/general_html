Terminal - Ascertaining Terminal Window Size

There are a few simple ways to check the dimensions (rows and columns) of your terminal window depending on what format you prefer:

1. The stty Command (Fastest & Universal)

To print the dimensions in rows columns format: 

stty size
  • Output: 48 120 (48 rows tall, 120 columns wide)

2. Using Shell Variables (\(LINES & \)COLUMNS)

Bash and Zsh automatically track window dimensions in built-in environment variables: 

echo "$LINES x $COLUMNS"
  • Output: 48 x 120

3. Using tput

If you need to query rows or columns individually (useful for shell scripts): 

# Get columns (width)
tput cols

# Get lines (height)
tput lines

4. For Pixel Dimensions (X11 / Desktop)

If you need the window's actual pixel size instead of character grid size (assuming you are running under an X11 session or standard desktop terminal): 

xwininfo -id $(xdotool getactivewindow) | grep -E 'Width|Height'
