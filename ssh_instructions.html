SSH

  • To be able to clone and push files to Github, you need an SSH key.
  • On a local machine, you need to create an SSH public and private key.
  • The public key needs to be entered into the Github account's settings:
    • Copy the contents of the .pub file and paste it onto Settings>SSH Security > add key area of Github.
  • (You might need to set the permissions of the ssh private key as readable by owner only.)
  • The ssh-agent might be needed to register the ssh key with the local machine.

Local Machine Terminal

Create the keys: 

ssh-keygen -t rsa =b 4096 -C "email@example.com"
#> Enter name of key: <your key name>
#> (Leave passphrase blank - and hit return twice)
# The file will be at ~/.ssh/
  • If you need to, you can register the key with ssh-agent 
    ssh-agent -s
ssh-add ~/.ssh/<your key>
