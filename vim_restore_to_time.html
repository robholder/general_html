Restore to Status @ Time

Command

  • :earlier <time value><units> --> revert to the state of the file/buffer at given time if present.
    • :ea --> abbreviated
  • :later <time value><units> --> revert to the state of the file/buffer at given time (forward if you've already reverted backwards).
    • :lat --> abbreviated

Units

The default is number of changes (i.e., with no unit):

  • <blank> --> Changes
  • s --> Seconds
  • m --> Minutes
  • h --> Hours
  • d --> Days
  • f --> Number of saves

Example: 

" Back 3 changes ago:
:earlier 3
"or
:ea 3

"Back two hours
:earlier 2h
"or
:ea 2h

"Forward one hour
:later 1h
"or
:lat 1h
