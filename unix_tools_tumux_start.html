Tmux

Tmux - Starting Tmux

What is it?

  • Terminal Multiplexer:
    • Terminals within terminals
  • Type tmux in a terminal to start.
  • You'll see a 'footer' at the base of the window.
  • Session is persistent, even if connections to remote terminal is broken.

Install

Check it isn't already installed:

Either:

  • command -v tmux:
    • path to command will be shown if it is installed.

Or:

  • which tmux

Installation Per System

Debian/Ubuntu type Linux:

  • sudo apt update
  • sudo apt install tmux

Fedora:

  • sudo dnf install tmux

Arch:

  • sudo pacman -S tmux

MacOS (modern):

  • brew update
  • brew install tmux

Launch

  • tmux

Detach From Session

  • <prefix_key then d:
    • Default <prefix_key> is 'b':
      • <Cntl> b then d --> detach

The terminal will be back to normal - the tmux session is in the background. The session will even survive a log-out of an SSH connection, and no longer connected to a server.

Look at previous session:

  • tmux list-sessions

Or:

  • tmux ls

Example output:

  • 0: 1 window (created Wed May 31 17:34:47 2023)

To reattach to the most recent session:

  • tmux attach

Or:

  • tmux a

Make a new session:

  • tmux

Creating Panes

  • Start temus with tmux
  • Vertical split:
    • <prefix_key> %
    • default is <Cntl> b then 5
  • Horizontal split:
    • <prefix_keys> <double_quote_key>
    • <Cntl> b then <Shift> "

Navigating Panes

  • <prefix_keys> & <arrow_direction>
    • <Cntl> b and ↑
    • <Cntl> b and ↓
    • <Cntl> b and →
    • <Cntl> b and ←

Deleting Panes

  • exit in each pane.

Or:

  • <prefix_keys> then x:
    • Confirm exit of pane.
    • e.g., <Cntl> b then x
  • Last exit (last pane) will close Tmux
    • This permanently closes the Tmux session. It can't be re-attached.
  • Run <prefix_keys> then d to close last pane but keep 'session' in list.
