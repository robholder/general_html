Maya Management

Quickly Remove Items

Unused Items

  • Use 'unload in maya':
    • 'render only' NOT RECOMMENDED

Switch from Constrain

If you aren't using a character to constrain lights:

  • Chars come in as 'constrain':
    • This loads the hierarchy
    • Switch to 'don't move'

Cull Visualizer

  • Tool in Lighting tab:
    • Use to 'cull' out unused items

DLight

Save Scene

  • Backup and Save:
    • Icon is a cloud with an 'S'
    • faster than:
      • Save a new version
      • Save As

Stockroom FX

  • All FX Stockroom elements come in as 'low res'.
  • Switch to required variant as needed.

Assets in the Shot

  • Think of using lower resolution assets to save on resources

Regex

For the USD paradigm...

  • Wild cards are expensive:
    • don't use "*:*"
    • try to be specific as possible
    • Use tokens:
      • Capture everything:
        • ${all}
      • Capture XGen:
        • ${xgen}

lrigGlobalModifier

This is WISH specific

lrigGlobalModifier contains modifiers for show.

  • Each character has it's own assignment of the global modifiers.
  • Try to make edits as close to the asset that needs as possible:
    • Don't change scene level, change char level
  • Clear Shot Assignments:
    • menu like a three-line hamburger menu

Nuke

  • Finding the slowest nodes:
    • Pipeline > Profiling > Enable Profiling:
      • slowest nodes will be labeled 'red'.
    • Disable 'Profiling' when done
  • Manifest (DeepMatte):
    • Switch to 'Manifest File': Combined Mode
    • This is default
  • BokehDisney:
    • Near override, not too small
  • Render Mode:
    • top-down
    • set in preferences
  • Limiting Read Nodes:
    • Use full resolution only when necessary.
    • Don't re-Read the same image files
  • Use caching nodes:
    • Live - nuke runs upstream
    • Cached - uses precomp
