DLight - copyParam

  • Copy a parameter to extract to an overriding modifier:
    • RMB-copyParam
    • Create a new 'empty' modifier
    • RMB-paste in the body of the mod
    • Bind the mod to the item (e.g., light locator)
    • Note: apparently, the mod 'wins' because it exists in World, so overrides the lrig palette.
  • This method allows the following:
    • Completely overrides original value (not a multiply or aggregate)
    • Allows for chiclet switching
    • Shared values
  • Example:
    • Character needs neutral light while other items have saturated light color:
      • RMB-copyParam 'Light color'
      • New 'empty' mod
      • RMB-paste
      • Switch to white
      • Bind to original light's locator
      • Conclusion and result:
        • Chiclet allows for color override on a pass
