TCL - Label Expressions

Just a label (On a merge, reading the 'mix' knob): 

[valule mix]

To make an expression: 

[expr {expression ? this : that}]

To make an expression: 

[expr {[value knob] ? this : that}]

Simple on off label: 

[expr {[value which] == 1 ? "Using Splineblur" : "No Splineblur"}]

At 100% show "Full Mix", at 0% show "No Mix", and anything between, the actual mix value: 

[expr {[value mix] == 1 ? "Full Mix" : ([value mix] == 0 ? "No Mix" : "Mix: [value mix]" ) }]

With HTML styling: 

<center>REDUCE CONTRAST
<b>[expr {[value mix] == 1 ? "Full Mix" : ([value mix] == 0 ? "No Mix" : "Mix: [value mix]" ) }]</b></center>

Alternative Syntax

[if {[value which] == 0} {return "First"} {return "Second"}]
