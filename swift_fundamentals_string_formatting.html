Swift - String Formatting

In Swift and SwiftUI, text formatting is divided into two primary toolsets: the modern SwiftUI .formatted() syntax (best for dates, numbers, and currencies) and the classic String(format:) specifier syntax (best for layout alignment, padding, and precision control).

Here is a curated cheat sheet of the most practical string formatting recipes to keep in your development notes.

1. Zero-Padding & Width Control (String(format:))

This is the classic C-style syntax you just used. It is indispensable for asset naming, timers, and aligned terminal outputs. 

// Zero-Padding to a specific width
String(format: "%02d", 6)    // "06"  (Perfect for dice, calendar days)
String(format: "%03d", 42)   // "042" (Perfect for game levels, ID numbers)

// Space-Padding (Useful for aligning columns in monospaced text)
String(format: "%5d", 7)     // "    7" (Takes up 5 total character slots)

2. Floating-Point Precision (String(format:))

When dealing with Double or Float values, Swift will normally print as many decimal places as it can. You can use %f combined with a dot (.) to lock down the exact decimal length. 

let pi = 3.14159265

String(format: "%.2f", pi)   // "3.14"  (Rounds to 2 decimal places)
String(format: "%.0f", pi)   // "3"     (Drops decimals entirely)
String(format: "%06.2f", pi) // "003.14" (Total width of 6 characters, zero-padded)

3. The Modern SwiftUI .formatted() System

For user-facing UI text, iOS 15 introduced the .formatted() modifier. It automatically handles localization (e.g., swapping dots and commas for European users) out of the box.

Numbers & Percentages

// Percentages
0.75.formatted(.percent)                           // "75%"
0.1234.formatted(.percent.precision(.fractionLength(1))) // "12.3%"

// Large Number Grouping
1000000.formatted()                                // "1,000,000" (or "1.000.000" in Europe)

Raw Integer Injection

Here is a quick look at the difference between inserting a raw integer versus formatting it: 

let itemsCount = 5
let hugeScore = 1500000

// 1. RAW INTEGER INJECTION (%d)
// It grabs the number exactly as it lives in memory and drops it in.
String(format: "You have %d items", itemsCount)  // Output: "You have 5 items"
String(format: "Score: %d", hugeScore)           // Output: "Score: 1500000"

// 2. FORMATTED INJECTION
// It intercepts the raw integer and rearranges it before displaying it.
String(format: "You have %03d items", itemsCount) // Output: "You have 005 items"
hugeScore.formatted()                             // Output: "1,000,000" (Adds regional commas)

Why use %d for raw insertion?

In standard SwiftUI, you usually write raw numbers using standard string interpolation like this: "You have \(itemsCount) items".

However, if you are building complex strings using the older String(format:) engine, %d is simply the standard token symbol you use to tell the computer: "Hey, stick a raw whole decimal integer right here!"

Currencies

You don't need to hardcode symbols like ＄. Tell iOS what currency code you want, and it will draw the symbol and format the placement automatically based on the user's phone region settings. 

24.99.formatted(.currency(code: "USD"))            // "$24.99"
1500.formatted(.currency(code: "EUR"))             // "€1,500.00"

Dates and Times

Formatting dates manually used to take 10 lines of setup code. Now you can describe the "style" you want in one line. 

let now = Date()

now.formatted(date: .abbreviated, time: .omitted)  // "Jun 30, 2026"
now.formatted(date: .numeric, time: .shortened)    // "6/30/2026, 11:21 PM"

// Individual Component Extraction
now.formatted(.dateTime.weekday(.wide).month(.wide)) // "Tuesday, June"

🛠️ Quick Reference Cheat Sheet

Specifier Data Type Expected Output Example Common Use Case
%d or %i Integer 42 Raw integer injection
%02d Integer 05 Timers (01:05), asset names
%.2f Double/Float 19.99 Price displays, sensor readings
%@ String / Object "Hello" Injecting text variables inside a format

How to use them natively in SwiftUI views:

You can wrap String(format:) directly inside a SwiftUI Text element, or pass a .formatted() configuration directly to the view: 

VStack(spacing: 10) {
    // Using String Format
    Text(String(format: "Score: %04d", currentScore))
    
    // Using SwiftUI Native Formatter
    Text(price, format: .currency(code: "USD"))
}
