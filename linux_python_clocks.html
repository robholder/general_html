Linux - Clocks (Python)

Standalone Portable Desktop Clock (Best GUI replacement)

Run this single command in terminal to test creating a minimal GUI floating clock window: 

python3 -c '
import tkinter as tk, time
root = tk.Tk()
root.title("Clock")
root.geometry("200x80")
root.attributes("-topmost", True)
#lbl = tk.Label(root, font=("Helvetica", 28, "bold"), bg="black", fg="#00FF00")
# Alt color options
lbl = tk.Label(root, font=("Helvetica", 28, "bold"), bg="black", fg="#dba135")
lbl.pack(expand=True, fill="both")
def update():
    #lbl.config(text=time.strftime("%H:%M:%S"))
    # Alt number format:
    lbl.config(text=time.strftime("%I:%M %p").lower())
    root.after(1000, update)
update()
root.mainloop()
'

How to make it easy to use:

Put that code into a script named ~/.local/bin/myclock:

  • mkdir -p ~/.local/bin
  • Create the file ~/.local/bin/myclock with the python script above.
  • chmod +x ~/.local/bin/myclock
  • Now just type myclock & anytime to launch a tiny floating green clock window on your desktop.

Analog

python3 -c '
import tkinter as tk, time, math

root = tk.Tk()
root.title("Analog Clock")
root.geometry("240x240")
root.attributes("-topmost", True)

canvas = tk.Canvas(root, width=240, height=240, bg="black", highlightthickness=0)
canvas.pack(fill="both", expand=True)

def update():
    canvas.delete("all")
    # Draw Clock Face Outer Ring
    canvas.create_oval(10, 10, 230, 230, outline="#00FF00", width=3)
    
    # Current Time & Angles
    now = time.localtime()
    h, m, s = now.tm_hour % 12, now.tm_min, now.tm_sec
    
    sec_angle = math.radians(90 - s * 6)
    min_angle = math.radians(90 - (m + s/60) * 6)
    hr_angle  = math.radians(90 - (h + m/60) * 30)

    # Center Point (120, 120)
    cx, cy = 120, 120
    
    # Draw Hour Hand (Thick green)
    canvas.create_line(cx, cy, cx + 50 * math.cos(hr_angle), cy - 50 * math.sin(hr_angle), fill="#00FF00", width=4)
    # Draw Minute Hand (Medium green)
    canvas.create_line(cx, cy, cx + 75 * math.cos(min_angle), cy - 75 * math.sin(min_angle), fill="#00FF00", width=2)
    # Draw Second Hand (Thin red)
    canvas.create_line(cx, cy, cx + 85 * math.cos(sec_angle), cy - 85 * math.sin(sec_angle), fill="red", width=1)

    root.after(1000, update)

update()
root.mainloop()
'
