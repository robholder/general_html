Practice and Examples for Shell Tools

Copy the content following examples into a file to practice with shell tools like awk and cut. I'll also suggest tasks to try out on the sample files. Let's create a couple of examples:

Example File 1: employees.txt

This file contains information about employees in a hypothetical company: 

ID,Name,Department,Salary,Start_Date
101,John Doe,Engineering,75000,2018-03-15
102,Jane Smith,Marketing,65000,2020-06-01
103,Emily Davis,Finance,72000,2019-11-23
104,Michael Brown,Engineering,78000,2017-05-30
105,Sarah Wilson,HR,60000,2021-01-15

Tasks:

  • Extract the names and salaries of all employees: 
cut -d',' -f2,4 employees.txt
  • Display only employees in the "Engineering" department: 
awk -F',' '$3 == "Engineering" {print $0}' employees.txt
  • Find all employees who started after 2020: 
awk -F',' '$5 > "2020-01-01" {print $0}' employees.txt

Example File 2: products.csv

This file contains data about products in a store inventory: 

ID,Product_Name,Category,Price,Stock
1,Smartphone,Electronics,699.99,25
2,Laptop,Electronics,1199.99,15
3,Desk Chair,Furniture,89.99,50
4,Notebook,Stationery,5.99,100
5,Table,Furniture,129.99,10

Tasks:

  • Extract only the product names: 
cut -d',' -f2 products.csv
  • List all products that belong to the "Furniture" category: 
awk -F',' '$3 == "Furniture" {print $0}' products.csv
  • Calculate the total stock of all products: 
awk -F',' '{sum += $5} END {print "Total Stock:", sum}' products.csv
  • Print products priced above $100: 
awk -F',' '$4 > 100 {print $0}' products.csv

Example File 3: logfile.log

This file simulates a typical log file for a server: 

[2025-03-21 10:00:12] INFO: Server started
[2025-03-21 10:05:45] ERROR: Connection timeout
[2025-03-21 10:10:03] WARNING: High memory usage detected
[2025-03-21 10:15:22] INFO: Connection established
[2025-03-21 10:20:10] ERROR: Disk space low
[2025-03-21 10:30:14] INFO: Server shutdown

Tasks:

  • Extract all lines containing the word "ERROR": 
grep "ERROR" logfile.log
  • Count the number of errors in the log file: 
grep -c "ERROR" logfile.log
  • Extract only the timestamps from the log file: 
cut -d']' -f1 logfile.log | tr -d '
