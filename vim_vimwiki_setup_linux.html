Git Push Script - Linux

This was a script used successfully on PMount's tcsh Linux terminals:

Place in ~/bin/ dir, and make it executable: 

chmod +x vwdeploy

In my Vimrc, I have a shortcut: 

cnoreabbrev lwww !vwdeploy

The script - vwdeploy:

#!/bin/tcsh

# Find git and store its path in GIT variable
set GIT = `which git`

# Set the commit timestamp
set COMMIT_TIMESTAMP = `date +'%y%m%d_%h%m %z'`

# Convert wiki files to HTML files
#vim +VimwikiTabIndex +VimwikiAll2HTML +qall
# This version forces a regular Vim, not an aliased one like GVim in my case.
vim -E -s +VimwikiTabIndex +VimwikiAll2HTML +qall

# Change directory to the general wiki folder,
# add all changes, commit, and push to the "master" branch.
cd ~/vimwiki_dir/general
$GIT add --all .
$GIT commit -am "Automated update on $COMMIT_TIMESTAMP"
$GIT push origin master

# Now do the same for the generated HTML folder.
cd ~/vimwiki_dir/general_html
$GIT add --all .
$GIT commit -am "Automated update on $COMMIT_TIMESTAMP"
$GIT push
