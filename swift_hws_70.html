AI Instruction on Extracting SleepTime

QUESTION

How can we extract the result of Calculate and display it live on the screen at all times.

ANSWER

This is a great step in moving from a "beginner" app structure to a more "pro" reactive one. In SwiftUI, the best way to do this is to convert your calculation logic into a computed property.

Instead of waiting for a button click to trigger a function, a computed property will re-run automatically whenever any of the @State variables it relies on (wakeUp, sleepAmount, or coffeeAmount) change.

Step 1: Create the Computed Property

Replace your calculateBedtime() function with a variable that returns a string. We will wrap the same logic in a do-catch block inside the property. 

var sleepResults: String {
    do {
        let config = MLModelConfiguration()
        let model = try SleepCalculator(configuration: config)
        
        let components = Calendar.current.dateComponents([.hour, .minute], from: wakeUp)
        let hour = (components.hour ?? 0) * 60 * 60
        let minute = (components.minute ?? 0) * 60
        
        let prediction = try model.prediction(wake: Double(hour + minute), estimatedSleep: sleepAmount, coffee: Double(coffeeAmount))
        
        let sleepTime = wakeUp - prediction.actualSleep
        return sleepTime.formatted(date: .omitted, time: .shortened)
    } catch {
        return "Error"
    }
}

Step 2: Update the UI

Now you can remove the "Calculate" button and the alert logic entirely. Just place a Section or a Text view inside your Form to display the result. 

Form {
    // ... your existing sections for Wake Up, Sleep Amount, and Coffee ...

    Section("Your ideal bedtime is") {
        Text(sleepResults)
            .font(.largeTitle)
            .foregroundStyle(.blue)
    }
}

How this works (The SwiftUI Way)

In SwiftUI, the body property is refreshed whenever a @State variable changes. Because your new sleepResults property uses those @State variables, SwiftUI knows it needs to re-calculate that string and update the Text on the screen instantly.

Refined Code Structure

Here is how your ContentView looks with these changes integrated: 

struct ContentView: View {
    @State private var wakeUp = defaultWaketime
    @State private var sleepAmount = 8.0
    @State private var coffeeAmount = 1

    static var defaultWaketime: Date {
        var components = DateComponents()
        components.hour = 7
        components.minute = 0
        return Calendar.current.date(from: components) ?? .now
    }

    // This is the "Live" calculation engine
    var sleepResults: String {
        do {
            let config = MLModelConfiguration()
            let model = try SleepCalculator(configuration: config)
            
            let components = Calendar.current.dateComponents([.hour, .minute], from: wakeUp)
            let hour = (components.hour ?? 0) * 60 * 60
            let minute = (components.minute ?? 0) * 60
            
            let prediction = try model.prediction(wake: Double(hour + minute), estimatedSleep: sleepAmount, coffee: Double(coffeeAmount))
            
            let sleepTime = wakeUp - prediction.actualSleep
            return sleepTime.formatted(date: .omitted, time: .shortened)
        } catch {
            return "Sorry, error calculating."
        }
    }

    var body: some View {
        NavigationStack {
            Form {
                Section("When do you want to wake up?") {
                    DatePicker("Please enter a time", selection: $wakeUp, displayedComponents: .hourAndMinute)
                        .labelsHidden()
                }

                Section("Desired amount of sleep") {
                    Stepper("\(sleepAmount.formatted()) hours", value: $sleepAmount, in: 4...12, step: 0.25)
                }
                
                Section("Daily coffee intake") {
                    Stepper("^[\(coffeeAmount) cup](inflect: true)", value: $coffeeAmount, in: 1...20)
                }

                // THE LIVE DISPLAY
                Section("Your ideal bedtime is") {
                    Text(sleepResults)
                        .font(.largeTitle)
                        .fontWeight(.bold)
                        .frame(maxWidth: .infinity, alignment: .center)
                }
            }
            .navigationTitle("BetterRest")
        }
    }
}

Note: You can now delete the alertTitle, alertMessage, and showingAlert variables, as well as the .alert modifier at the bottom of your NavigationStack. It’s much cleaner now!
