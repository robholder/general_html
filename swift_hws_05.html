Hacking With Swift - 05 - How to store decimal numbers

Decimal numbers are stored and manipulated with a floating point.

Regular floating points are not very accurate: 

import Cocoa

let number = 0.1 + 0.3
print(number) # -> 0.3000000000000000004 Oops!

Swift uses 'doubles'. This is a data storage using twice the data format as floating.

NOTE: You can't combine 'int' variables and floating points unless declares.

For example: 

let a = 1
let b = 2.0
let c = a + b # -> Error, because a is an integer, b is a float

To correct, convert the float (2.0) to an integer (2): 

let c = a + Int(b)

Swift decides on the number type based on whether it has a decimal point, even if the digit after the point is zero: 

let double1 = 3.1
let double2 = 3131.3131
let double3 = 3.0
let int1 = 3

Swift decides the variable 'kind', and you can't change it after the fact directly: 

var name = "Nicolas Cage"
name = 57 # -> Cannot assign value of type 'Int' to type 'String'

Floats and doubles have the same math methods as integers: 

var rating = 5.0
rating *= 2 # -> 10.0

You can mix/combine operations with floats and doubles.
