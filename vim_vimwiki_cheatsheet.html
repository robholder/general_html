Tables

Create a table with VimwikiTable <col> <row> command:

:VimwikiTable 2 12

I created the abbreviation table above by the command above, then <ctrl>v to visually select the existing column data from a website. So the text was pasted into the correct column.
Next, I used gww (normal mode) to reformat the table (cursor in the table somewhere).

Col A Col B
hello a
goodbye b

Inserting a <cr> after the last column line will create another row.
Tab while in INSERT within a column will pop to the next column.

To slide a whole column back or ahead in its order:
<option><left> - (sometimes written as <A-Left>)
<option><right> - (sometimes written as <A-Right>)

ci\ - Change contents in the cell.
cic - Change contents of a column. (Erases all content from the column first!!)

Do the next two work?
a <right arrow> will navigate to the next column.
a <left arrow> will navigate back to the previous column.

Vimwiki Search

:VimwikiSearch /pattern/ <-- Search through entire vimwiki pages for a pattern
:lnext <-- next found result
:llast <-- goto last result
:lprevious <-- previous result
:lfirst <-- goto first result
:lope <-- List all search results from above command in a split window. (Clicking on a match will hope to the page)
