Git Aliases

You can set aliases within git itself (directly in the config settings rather than a shell or system alias).

The following sets 'loga' as an alias for 'git log --online --graph --all'. 

# To see an alias for a detailed log output.
# (Notice there's no 'git' included in the expanded command):
git config --global alias.loga "log --online --graph --all"

Git aliases allow for the addition of further arguments and flags at use time. So you can add git as aliases.

Example: 

# To add 'commit' as an alias:
git config --global alias.c "commit"

# Now you can commit by using this shortcut...
git c -m "my commit message"

# ...which expands to:
git commit -m "my commit message"

# To add 'add' as an alias:
git config --global alias.a "add"

# Now you can add using this shortcut...
git a .

# Expands to:
git add .

Show Details of an Alias

# To expand the alias:
git help <alias>

# e.g., 
$ git help c
>'c' is aliased to 'commit'

Use git config --list --show-origin | grep alias to look at them as an echoed list.

Suggestions

Here's a list to copy and paste: 

git config --global alias.loga "log --oneline --graph --all"
git config --global alias.c "commit"
git config --global alias.a "add"
git config --global alias.s "status"
git config --global alias.b "branch"
git config --global alias.sw "switch"
git config --global alias.ch "checkout"
