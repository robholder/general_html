Hacking With Swift - 07 - How to join strings together

Two ways to join strings:

  • Concatenation:
    • +
  • String interpolation:
    • \(variable_or_value)

Concatenation

The + is 'overloaded': which means '+' will do different tasks in different contexts. 

let people = "Haters"
let action = "hate"
let lyric  = people + " gonna " + actions

print(lyric)
# >> Haters gonna hate

Even thought numbers get treated as strings with this technique, you can explicitly define as a string like so: 

let number = 11
let missionMessage = "Apollo " + String(number) + " landed on the moon."

print(missionMessage)

# >> Apollo 11 landed on the moon.

String Interpolation

Using this technique, you can join (concatenate) both strings and integers: 

let name = "Taylor"
let age = 26
let message = "Hello, may name is \(name) and I'm \(age) years old."

print(message)

# >> Hello, my name is Taylor and I'm 26 years old.

let number = 11
let missionMessage = "Apollo " \(number) " landed on the moon."

print(missionMessage)

# >> Apollo 11 landed on the moon.

You can even perform math evaluation as part of the interpolation: 

print("5 x 5 is \(5 * 5)")

# >> 5 x 5 is 25
