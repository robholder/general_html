Git in Vim

You can run Git commands directly from Vim's Command mode using the :! (bang) operator, which allows you to execute shell commands.

Here’s how you can do it:

Running Git Commands in Vim

To run shell commands from Vim, prepend them with ! 

:!git add .
" Add message with -m; otherwise you might end up with a swap file issue:
:!git commit -m "Your commit message"
:!git push origin main

Each command will be executed in your shell, and Vim will briefly show the output.

Using Vim’s Built-in Terminal

You can also open a terminal inside Vim and run Git commands interactively: 

:terminal

Then, run Git commands as usual within the terminal window.

Running Git from Within a Vim Buffer

If you're editing a Git commit message and need to check status, you can use: 

:read !git status

This will insert the output of git status into your current buffer.

Mappings

You can enhance your Git workflow in Vim using key mappings and plugins to speed up operations like staging, committing, and pushing changes.

Key Mappings for Git Commands

You can create custom mappings to run Git commands without leaving Vim. To do this, add these lines to your ~/.vimrc or ~/.vim/init.vim: 

" Add all changes
nnoremap <leader>ga :!git add .<CR>

" Commit with a message prompt
nnoremap <leader>gc :!git commit -m "<C-R>=input('Commit message: ')<CR>"<CR>

" Push to origin/main
nnoremap <leader>gp :!git push origin main<CR>

Now, you can press:

  • <leader>ga to add all changes
  • <leader>gc to commit with a message prompt
  • <leader>gp to push to the remote repo

Useful Vim Plugins for Git

To integrate Git directly into Vim, here are two powerful plugins:

Fugitive.vim (The Git Wrapper for Vim)

  • Lets you run Git commands directly inside Vim
  • Use :Git status, :Git add, :Git commit, and :Git push
  • Explore Git logs with :Git log

Install via Vim-Plug:

Plug 'tpope/vim-fugitive'

Then reload Vim and run: 

:PlugInstall

Gina.vim (Async Git Integration)

  • Faster than Fugitive, uses asynchronous execution
  • Runs Git commands like :Gina status, :Gina commit, :Gina push
  • Efficiently manage branches and diffs inside Vim

Install using: 

Plug 'lambdalisue/gina.vim'
