Hacking With Swift - 08 - How to store ordered data in arrays

I addition to String, Int and Double, there is the Array.

Example with String Array

var beatles = ["John", "Paul", "George", "Ringo"]
let numbers = [4, 8, 15, 16, 23, 42]

The items in an array can be referenced by the 'index'. Indexes start at '0' (zero).

Appending to an Array

var beatles = ["John", "Paul", "George", "Ringo"]
beatles.append("Adrian")
beatles.append("Jeff")
# or
beatles[3] = "Benoit"

let numbers = [4, 8, 15, 16, 23, 42]
numbers.append(100)
# or to assign to an index:
numbers[3] = 19

Mixing Types in Arrays

You cannot mix data types in an array. Therefore this won't work: 


var beatles = ["John", "Paul", "George", "Ringo"]
beatles.append(8)
# nor:
beatles[0] = 8

Specifying an Array Type

var scores = Array<Int>()

scores.append(100)
scores.append(89)

print(scores)
# >> 89

var albums = Array<String>()

albums.append("Folklore")
albums.append("Fearless")

# But not:
albums.append(9)

Alternative Declaration of Array Types

You can declare array types with the following shorthand:

  • String array:
    • [String]()
  • Integer array:
    • [Int]()

Also, Swift can figure out the array type by the first assignment: 

var words = ["hello"]

Removing Items

Count and Remove methods: 

var characters = ["Lana", "Pam", "Ray", "Sterline"]

print(characters.count)
# >> 4

characters.remove(at: 2)
print(characters.count)
# >> 3

characters.removeAll()
print(characters.count)
# >> 0

Contains: 

let bondMovies = ["Casino Royal", "Spectre", "No Time To Die"]
print(bondMovies.contains("Frozen"))
# >> false

Sorted: 

let cities = ["London", "Tokyo", "Rome", "Budapest"]

# or to specify String explicitly:
var cities: [String] = ["London", "Paris", "New York"]

print(cities.sorted())
# >> ["Budapest", "London", "Rome", "Tokyo"]

Reversed:

(Result is an object that retains the input order, but remembers the mechanism to reverse at call time.) 

let presidents = ["Bush", "Obama", "Trump", "Biden"]
let reversedPresidents = presidents.reverse()
print(reversedPresidents)
# >> ReversedCollection<Array<String>>(_base: ["Bush", "Obama", "Trump", "Biden"])

# Stores as an (original) object and rembers the reverse order a an internal process.
