Tmux - Starting Tmux

To pull in a different configuration file for Starship when you are inside a tmux session, you need to set the STARSHIP_CONFIG environment variable.

When Starship initializes, it checks this variable for a file path. If it's not set, it defaults to ~/.config/starship.toml.

Method 1: Automatic Detection in Shell Config (Recommended)

You can add a check to your shell configuration file (.zshrc or .bashrc) that detects if you are in a tmux session and exports the alternative path automatically.

Add this to your ~/.zshrc or ~/.bashrc: 

if [ -n "$TMUX" ]; then
  # Path to your tmux-specific config
  export STARSHIP_CONFIG=~/.config/starship-tmux.toml
fi
  • Why it works: The $TMUX variable is only set when you are inside a tmux session.
  • Note: Make sure this export happens before the eval "$(starship init ...)" line in your config.
