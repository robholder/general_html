Python - Square Image Edit - Quick Action

While you cannot automate the native Preview app to do this (Preview is designed purely as a manual viewer and lack robust canvas-padding tools), you can absolutely do this using a Quick Action / Workflow script in Automator!

Here is how the automated workflow functions behind the scenes to pad your image:

How to Create the "Make Square with Black Padding" Quick Action

  • Open Automator and create a new Quick Action.
  • Set the top dropdowns to:
    • Workflow receives current: image files
    • in: Finder.app
  • Drag the Run Shell Script action from the library into the workflow area.
  • Configure the top of the Run Shell Script block exactly like this:
    • Shell: /usr/bin/python3 or /usr/local/bin/python3 (Whichever is present.)
    • Pass input: as arguments
  • Replace all the default text in that box with the following script:
NOTE: There are two variables to add a color string (for filename) and color hex code. 
import os
import shutil
import subprocess
import sys

bg_col_suffix = "black"
bg_col_hex = '000000'

def make_image_square(file_path):
    # 1. SPLIT FILE PATH AND GENERATE UNIQUE SUFFIX
    base_path, extension = os.path.splitext(file_path)
    
    # Try the default destination first
    new_square_path = f"{base_path}_square{extension}"
    
    # If the file already exists, loop until a unique number suffix is found
    counter = 1
    while os.path.exists(new_square_path):
        new_square_path = f"{base_path}_{bg_col_suffix}_square_{counter}{extension}"
        counter += 1
    
    # 2. GENERATE THE SAFE DUPLICATE
    try:
        shutil.copy2(file_path, new_square_path)
    except Exception:
        return # Skip if file copying fails
        
    # 3. READ ORIGINAL DIMENSIONS VIA SIPS
    try:
        cmd = ['sips', '-g', 'pixelWidth', '-g', 'pixelHeight', new_square_path]
        result = subprocess.run(cmd, capture_output=True, text=True, check=True)
        
        width = 0
        height = 0
        
        for line in result.stdout.split('\n'):
            if 'pixelWidth:' in line:
                width = int(line.split(':')[1].strip())
            elif 'pixelHeight:' in line:
                height = int(line.split(':')[1].strip())
        
        # 4. CALCULATE CANVAS & APPLY BLACK PADDING
        max_dimension = max(width, height)
        
        if max_dimension > 0:
            subprocess.run([
                'sips',
                '--padToHeightWidth', str(max_dimension), str(max_dimension),
                '--padColor', bg_col_hex,
                new_square_path
            ], check=True)
            
    except Exception:
        # Cleanup broken copy on failure
        if os.path.exists(new_square_path):
            os.remove(new_square_path)

if __name__ == "__main__":
    for path in sys.argv[1:]:
        if os.path.exists(path):
            make_image_square(path)

Press Cmd + S to save it, and name it something like ⏹️ Make Image Square BlackBG

Right-clicking on an image and selecting the Quick Actions context menu will expose the command/script, and the image will be squared to the longest dimension and the filler pixels will be black.
