Hacking With Swift - 19 - How to skip loop items with break and continue

There are two ways to skip one or more items in a loop.

  • Continue
  • Break

Continue

continue will skip a step, then continue. 

let filenames = ["me.jpg", "work.txt", "sophie.jpg", "logo.psd"]

for filename in filenames {
    if filename.hasSuffix(".jpg") == false {
        continue
    }

    print("Found picture: \(filename)")
}

Break

break will not only skip a loop, it will abort all following loops too. 

let number1 = 4
let number2 = 14
var multiples = [Int]()

for i in 1...100_000 {
    if i.isMultiple(of: number1) && i.isMultiple(of: number2) {
        multiples.append(i)

        if multiples.count == 10 {
            break
        }
    }
}

print(multiples)

// >> [28, 56, 84, 112, 140, 168, 196, 224, 252, 280]

A new method to note:

  • .isMultiple(of: <number>)
