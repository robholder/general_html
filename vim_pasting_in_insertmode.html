Pasting in Insert mode

You can paste the yanked text while you're in insert mode using a combination of Vim commands. Here’s how to do it:

Use <C-r> (Control + r):

  • In insert mode, press <ctrl> r followed by the register name to paste the yanked text.
  • The 'unnamed' register is ":
    • So you can use <ctrl>r " to paste the yanked text (although this get's replaced by deleted text in addition to 'yank').
  • The dedicated 'latest yank' register is 0
    • So you can use <ctrl>0 to guarantee the latest 'yanked' text.

Here’s a step-by-step example:

Yank & Paste

Yank the text in normal mode (e.g., yy to yank a line).

  • Enter insert mode (e.g., press i).
  • Press <ctrl> r followed by " to paste the yanked text.

Regular register

So, the sequence would look something like this:

  • yy --> Yank the current line
  • i --> Enter insert mode
  • <ctrl>r" --> Paste the yanked text while in insert mode

Other register

If you yanked the text into a different register (e.g., register a), you can use Ctrl + r + a to paste it:

  • "ayw --> Yank a word into register 'a'
  • i --> Enter insert mode
  • <ctrl>-ra --> Paste the yanked text from register 'a' while in insert mode

This method allows you to insert the yanked text seamlessly as you type in insert mode.
