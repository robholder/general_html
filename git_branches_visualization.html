Git - Visualization

To see all your branches and their current status relative to their remote counterparts, you need to use verbose flags.

1. View Local Branches & Sync Status

The most direct command for your needs is: 

git branch -vv

What it shows:

  • List of all local branches.
  • The upstream (remote) branch it is tracking (e.g., [origin/main]).
  • Whether you are ahead (unpushed commits), behind (commits available to pull), or up-to-date.
  • The latest commit hash and message.

2. View ALL Branches (Local + Remote)

If you want to see every branch that exists on the server as well: 

git branch -a -vv

3. Essential: Update the "Status" First

Git commands only report the status based on the last time you connected to the server. If someone else pushed a change 5 minutes ago, your computer won't know until you "fetch."

To see the real current status without merging any code, run: 

git fetch --all
git branch -vv

Summary Table of Status Indicators

Indicator Meaning
[origin/branch] Up to date with the last known state of the remote.
[origin/branch: ahead 2] You have 2 commits locally that aren't on the server yet.
[origin/branch: behind 5] The server has 5 new commits you haven't pulled yet.
[origin/branch: ahead 1, behind 1] You and the server have diverged; you'll likely need to merge.
(nothing in brackets) This branch has no tracking/upstream information set up.

Visual Tree

To see a visual tree of your branches and their divergence, you can use the terminal or a graphical tool.

1. Terminal Visualization (The Quick Way)

Run this command to see a text-based "graph" of your history: 

git log --oneline --graph --all --decorate
  • --graph: Draws the ASCII lines showing branch splits and merges.
  • --oneline: Condenses each commit to a single line.
  • --all: Shows all branches (local and remote), not just your current one.
  • --decorate: Labels the commits with branch names like origin/main or HEAD.

2. Built-in "gitk" (Simple GUI)

Most Git installations come with a basic visualizer. In your terminal, type: 

gitk --all

This opens a window showing a clear tree of all commits, branches, and tags.

3. Recommended 2026 Tools (Modern GUIs)

If you prefer a more intuitive, interactive interface, these are the top-rated tools as of 2026:

  • VS Code (GitLens/GitGraph): If you use VS Code, install the GitGraph extension for a searchable, clickable tree directly in your editor.
  • GitKraken: Widely considered one of the best visual clients in 2026 for its drag-and-drop merging and beautiful branch graphs.
  • Lazygit: A popular terminal-based UI (TUI) that gives you a visual dashboard without leaving your command line.
  • Sourcetree: A free, comprehensive GUI that provides a clean map of how branches diverge.

4. Pro Tip: The "What's Missing" Check

To see exactly which commits are on the remote branch but not yet on yours (after running git fetch), use: 

git log HEAD..origin/main --oneline

Replace origin/main with your specific remote branch name.
