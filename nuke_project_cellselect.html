CellSelect Tool

I made this tool to take advantage of two factors:

  • The Flood Tool in Nuke
  • The fact that avoa is using ink lines and flat colors

Selects areas of continuous color (like a bucket-tool or magic-wand in Photoshop): 

set cut_paste_input [stack 0]
version 15.0 v7
push $cut_paste_input
Group {
 name CellSelect4
 selected true
 xpos 1509
 ypos 14166
 addUserKnob {20 tab_user l CellSelect}
 addUserKnob {26 title l "CELL SELECT"}
 addUserKnob {26 info l "" +STARTLINE T "* Select are surrounded by ink lines to create an alpha channel output.\n* Move the fllpt control to an enclosed cell or region.\n* Alpha channel will contain the matte result."}
 addUserKnob {26 ""}
 addUserKnob {4 view l View M {Mix Input Matte "" ""}}
 addUserKnob {41 fillpt l "Fill Point" T FloodFill3.fillpt}
 addUserKnob {20 tab_tech l Tech}
 addUserKnob {26 author l "Created by:" T "Robert Holder"}
 addUserKnob {26 version l Version: T 1.0.0}
}
