Here's a summary of the available flags in Nuke's Python API and options for locking knobs:

Nuke Python API flags

Nuke knobs have flags that control their appearance and behavior. You can manipulate these flags using the setFlag() and clearFlag() methods on a knob object.

Commonly Used Flags:

Use with with setFlag() and clearFlag():

  • nuke.DISABLED: Disables a knob, greying it out and preventing user interaction.
  • nuke.NO_ANIMATION: Prevents the knob from being animated or having expressions added via right-click.
  • nuke.DO_NOT_WRITE: Prevents writing the knob's data to the script file when saving or copying/pasting.
  • nuke.INVISIBLE: Hides the knob's panel and Viewer widgets.

Important Notes about Flags:

  • Some flags are exposed directly through the nuke module (e.g., nuke.DISABLED), while others require finding their hexadecimal values in the Knob.h file located in Nuke's installation directory.
  • You can also find flags listed under "all variables" in the Nuke Python API.
  • There are also type-specific flags (e.g., nuke.SLIDER for numeric knobs), which also have corresponding hexadecimal values.

Example:

To disable a knob named "mix" on a node named "Grade1", you would use the following code: 

nuke.toNode("Grade1").knob("mix").setFlag(nuke.DISABLED)
# or
nuke.toNode("Grade1").knob("mix").setEnabled(False)

To clear these flags: 

nuke.toNode("Grade1").knob("mix").clearFlag(nuke.DISABLED)
# or
nuke.toNode("Grade1").knob("mix").setEnabled(True)

You can still set values with this flag in place. Here we can still set the mix to a value of 1, even though the knob may be grayed out by setFlag(nuke.DISABLED). 

nuke.toNode("Grade1").knob("mix").setValue(1)

To clear a flag using its hexadecimal value, you'd do this (example for the SLIDER flag): 

nuke.toNode("Grade1").knob("mix").clearFlag(0x0000000000000002)
Use code with caution.

Options for locking knobs

While there isn't a direct "lock" method in Nuke's Python API, there are several ways to achieve similar behavior:

  • Disabling Knobs using setEnabled(False) or setFlag(nuke.DISABLED): This is the most common approach. It greys out the knob and prevents user interaction. 
selectedNode().knob('mix').setEnabled(False)  # or
selectedNode().knob('mix').setFlag(nuke.DISABLED)
  • Hiding Knobs using setVisible(False): This completely hides the knob from the user interface. 
selectedNode().knob('mix').setVisible(False)
  • Setting Expressions: You can set an expression on a knob that always returns a specific value, making it effectively locked to that value unless the expression is removed. 
nuke.toNode("Grade1")['disable'].setExpression("False")
