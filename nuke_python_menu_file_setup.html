Menu.py

Below is a concise, note-form breakdown for creating a basic menu.py in Nuke that's ready to import and expose custom tools via custom menus:

File Location & Creation

Create menu.py:

  • Place the file in your Nuke home directory (typically ~/.nuke/ on macOS/Linux or the corresponding directory on Windows).

Purpose:

This file is automatically sourced on Nuke’s startup, allowing your custom menus to be built immediately.

Set Up Imports

  • Import Nuke Module: 
    import nuke
  • Import Custom Tool Modules (if any):
    • If your custom tools are defined in separate script files (e.g., customTools.py), import them: 
      import customTools  # Replace with your module name(s)

Create a Custom Menu

Access the Main Menu:

  • Use nuke.menu("Nuke") to get the main Nuke menu.

Add a New Submenu:

customMenu = nuke.menu("Nuke").addMenu("Custom Tools")

Tip:

  • You can name your menu anything that makes sense for your workflow.

Add Menu Commands

Define Menu Commands:

  • Use the addCommand method to add a menu item that will execute a tool function. For example: 
    customMenu.addCommand("Do Something", "customTools.doSomething()", "ctrl+shift+d")

Parameters Explained:

  • "Do Something": The label for your menu item.
  • "customTools.doSomething()": The Python command (as a string) to run when selected.
  • "ctrl+shift+d": An optional keyboard shortcut.

Wrap in a Function (Optional)

Organize for Clarity:

  • Encapsulate the menu-building code in a function, then call it immediately: 
    def createCustomMenu():
    customMenu = nuke.menu("Nuke").addMenu("Custom Tools")
    customMenu.addCommand("Do Something", "customTools.doSomething()", "ctrl+shift+d")

createCustomMenu()

Benefit:

This makes it easier to maintain and extend your setup.

Test Your Setup

Reload Nuke:

Restart Nuke to ensure menu.py is loaded.

Verify:

  • Check Nuke’s menu bar for your “Custom Tools” menu.
  • Click the command and verify that it properly calls your custom tool.

By following these steps, you’ll have a basic menu.py set up to import and expose custom tools via a custom Nuke menu. This setup not only organizes your tools but also integrates them directly into the Nuke interface for quick access.
