BASH - Scripting

Command History

  • history --> show list of commands (as used historically)
  • up --> back through history of commands
  • down --> forward through history.

Line clearing

  • <cntl> l --> clear screen (move current line to the top)
  • <cntl> u --> clear current line contents
  • <cntl> w --> clear word
  • <cntl> c --> cancel
  • <cntl> k --> clear after cursor

Navigation

  • cd .. --> up one (directory) level
  • cd - --> back to last directory

rm

  • rm -i --> rm interactive (get asked to confirm each match)
  • Alias with
    • alias rm='rm -i'

cat

  • cat <path> --> print file contents to the shell

grep

  • grep '<pattern>' <file_path> --> find/list files with search pattern
    • grep 'dave' ~/Documents/myfile.txt --> find lines which contain 'dave'
  • -i --> option to ignore case (upper/lower case)
  • grep '<pattern>$' <file_path> --> find pattern at the END of lines (dollar)
  • grep '^<pattern>' <file_path> --> find pattern at the START of lines (carat)
