Break down the keystroke sequence

gdcgn<edit><esc> step-by-step:

  • gd
    • --> This command jumps to the definition of the symbol under the cursor. It's useful when you want to go directly to the function or variable definition in your code.
    • --> It puts the word under the cursor into the search register (similar to *).
  • cgn --> This is a powerful combination of commands:
    • c The change command, which deletes the selected text and places you in Insert mode.
    • gn This selects the next instance of the current search pattern, selecting its bounds.

The text that matches the search pattern will be selected and changed.

  • <edit> --> This placeholder represents the changes you want to make to the selected text. You can type in the new text you want to replace the selected pattern.
  • <esc> --> This command exits Insert mode and returns you to Normal mode.

Here's how the entire sequence works together:

  • You start by jumping to the definition of the symbol under the cursor with gd.
  • Then, you use cgn to change the next occurrence of the search pattern, placing you in Insert mode with the matched text selected.
  • You make your desired changes ().
  • Finally, you exit Insert mode with <esc>.

This keystroke sequence is a quick and efficient way to refactor your code by jumping to definitions and making changes to matching patterns.
