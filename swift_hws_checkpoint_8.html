Hacking With Swift - Checkpoint 8

Now that you understand how protocols and extensions work, it’s time to pause our learning and take on a challenge so you can put it all into practice.

Challenge

Protocol

Your challenge is this: make a protocol that

  • Describes a building:
    • add various properties and methods
  • Then create two structs:
    • House and Office, that conform to it.

Elements

Your protocol should require the following:

  • A property storing how many rooms it has.
  • A property storing the cost as an integer (e.g. 500,000 for a building costing $500,000.)
  • A property storing the name of the estate agent responsible for selling the building.
  • A method for printing the sales summary of the building, describing what it is along with its other properties.

I’ll provide some hints in a moment, but first I recommend you go ahead and try it yourself.

My Attempt

protocol Building {
    var roomNumber: Int { get }
    // or: let roomNumber: Int
    var cost: Int { get set }
    var agent: String { get set }
    func summary()
}

extension Building {
    func summary() {
        print("The building has \(roomNumber) rooms, costs $\(cost), and is sold by \(agent).")
    }
}

struct House: Building {
    var roomNumber: Int
    var cost: Int
    var agent: String
}

struct Office: Building {
    var roomNumber: Int
    var cost: Int
    var agent: String
}

let home = House(roomNumber: 4, cost: 350_000, agent: "Alice")
home.summary()

// >> The building has 4 rooms, costs $350000, and is sold by Alice.

Further Enhanced

This uses a method from the Cocoa library, which has to be imported: 

import Cocoa


protocol Building {
    var roomNumber: Int { get }
    // or: let roomNumber: Int
    var cost: Int { get set }
    var agent: String { get set }
    func summary()
}

extension Building {
    var formattedCost: String {
        let formatter = NumberFormatter()
        formatter.numberStyle = .decimal
        return formatter.string(from: NSNumber(value: cost)) ?? "\(cost)"
    }
}

struct House: Building {
    let roomNumber: Int
    var cost: Int
    var agent: String

    func summary() {
        print("""
        🏡 House for Sale
        -----------------
        Rooms: \(roomNumber)
        Price: $\(formattedCost)
        Agent: \(agent)
        
        """)
    }
}

Tutor's Hints

  • Start by designing the protocol fully before you go near any structs.
  • Remember you can’t provide implementations for methods in your protocol.
  • You get to decide whether your properties should be get-only or have getters and setters. For example, maybe the number of rooms is constant, but perhaps you also want to take into account someone adding an extension.
  • You can control the read-write status of a property using { get } or { get set }.
