Delete 'To' and 'inCluding'

dt[character] - delete UP to character. df[character] - delete UP to and INCLUDING character. dT[character] - delete BACK to character. dF[character] - delete BACK to and INCLUDING character.

also vt[character] - select UP to character. vf[character] - select UP to and INCLUDING character. vT[character] - select BACK to character. vF[character] - select BACK to and INCLUDING character.

bonus 0vg_ - Select to end of line (without newline); from beginning of line. ^vg_ Select to end of line (without newline); from first character of line.

Explanation: g_ is last 'non-blank' character of a line.

Repeat

. <-- Repeat last edit in normal mode. @: <-- Repeat last command-line command.
