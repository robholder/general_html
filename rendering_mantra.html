Houdini

Mantra

  • Render View:
    • Render Button for IPR
    • Choose ROP or cameras
    • Shift-select to drag area of IPR
    • Snapshots at base
  • Render Region:
    • Choose area by dragging
    • Render to MPlay - like playblast render
    • Render view (see above)
  • Render Output Nodes:
    • Render > create Render Node > Mantra - PBR
    • You can chain ROPs to create dependencies
  • Render Layers:
    • Use 'image planes' to create render layers:
      • Direct Lighting
      • Indirect Lighting
      • Shadows
      • Depth
      • etc.
  • Supports other 3rd Party Renderers:
    • RenderMan
    • Arnold
    • OctaneRender
    • Redshift
    • Maxwell

MPLAY

  • A file sequencer
  • Menu to load images
  • Option to see render layers
  • Timeline
  • Render time feature
  • RGB Channels
  • Gamma settings
