Reload - Importlib

When working with something like Nuke, and it's Python API, if you update a method or function, unless the code is 'reloaded', the original definition will not be revised or updated to the new definition. To remedy this, the script or module must be 'reloaded' to refresh the definitions.

In Python 3, the reload command has to be called from the importlib module. (In Python 2, you could just use reload.)

Basic reload (Python 3)

import importlib
import some_module

importlib.reload(some_module)

Reload with aliasing

import importlib
import some_module

importlib.reload(some_module)

# Re-import the alias to get the updated version
from some_module import sub_lib as sl
