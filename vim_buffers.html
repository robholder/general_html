:bn <-- Show next buffer (if it has been saved, or has no changes)
:b[number] <-- jump to buffer number (as labelled in the :ls results)
:bd <-- delete current buffer (window)
:buffo bd <-- delete all buffers (leaving empty vim session)

:%bd|e#|bd# <-- Delete all buffers, and return to last edited (leave current open only)
