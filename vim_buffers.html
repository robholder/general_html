Vim - Buffers

NOTE: You can see a list of buffers with :ls. 

:bp
:bn
:b2
:bd

Jump To Buffers

:bn <-- Show next buffer (if it has been saved, or has no changes)
:bp <-- Show previous buffer
:b[number] <-- jump to buffer number (as labelled in the :ls results)
:b# <-- Show alternative buffer (marked with # in the 'ls' list of buffers)

Control-^ to Jump Buffers

Switch between last buffer and back:

<ctrl> ^ --> Toggles back and forth from last to current to last.

Switch between last buffer and back:

  • #<ctrl> ^ --> Jump to buffer #

e.g.:

  • 1<ctrl>^ --> Jump to buffer 1
  • 8<ctrl>^ --> Jump to buffer 8

Open Buffers in Windows

:ba <-- horizontal windows, one for each buffer
:sba <-- same as :ba
:vert ba <-- vertical windows, one for each buffer
:on <-- return to single window
:only <-- same as :on

Open Buffers in Tabs

:tab ba <-- tab for each buffer
:tabo <-- return to single tab
:tabonly <-- same as :tabo

Delete Buffers

:bd <-- delete current buffer (window)
:%bd <-- delete all buffers (leaving empty vim session)
:buffo bd <-- same as above

https://stackoverflow.com/questions/4545275/vim-close-all-buffers-but-this-one

:%bd|e#|bd# <-- Delete all buffers, and return to last edited (leave current open only)

  • :%bd <-- Delete all buffers
  • e# <-- Edit last buffer open
  • bd# <-- close previous buffer (which would be [no name])

map <leader>o :%bd\|e#<cr>\|bd#<cr> <-- you could map this to leader-o to make a shorcut
