Docstring Italics

If italics are activated for vim, and you are using a 'python' syntax file, you can add the following to have docstrings behave like comments (which are italicized in most Python syntax implementations).

" CUSTOM ADDITION FOR DOCSTRINGS - START
" ADDED FROM https://superuser.com/questions/697347/vim-how-to-treat-three-quotations-in-a-row-as-a-comment-rather-than-a-string
" Highlight docstrings as comments, not string.
syn region pythonDocstring  start=+^\s*[uU]\?[rR]\?"""+ end=+"""+ keepend excludenl contains=pythonEscape,@Spell,pythonDoctest,pythonDocTest2,pythonSpaceError
syn region pythonDocstring  start=+^\s*[uU]\?[rR]\?'''+ end=+'''+ keepend excludenl contains=pythonEscape,@Spell,pythonDoctest,pythonDocTest2,pythonSpaceError

hi def link pythonDocstring pythonComment"
" CUSTOM ADDITION FOR DOCSTRINGS - END
