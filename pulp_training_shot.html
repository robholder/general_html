PULP Training Shot

  • Exit any existing 'gig':
    • exit
    • gig PULPTR
    • set shot number: 
      exit
gig PULPTR
export myshot=26
  • Duplicate master shot: 
    dupeShot PULP/900.0/009.00 testLighting/900.$myshot -s
  • Navigate to shot: 
    n testLighting 900.$myshot lighting
  • Run lighting_setup: 
    lighting_setup
  • Propagate from the parent shot:
  • gig to PULP 
    exit
gig PULP
export myshot=26
  • Propagate: 
    propagateTrainingData /disney/shows/PULP/work/Sequences/seq/900.0/009.00/lighting/maya/scenes/lighting_v0040_forPropagation /disney/shows/PULPTR/work/Sequences/seq/testLighting/900.$myshot/lighting/maya/scenes/lighting_v0002/
    • You'll be asked to confirm.
    • ..takes a while
  • You can open the scene as normal: 
    exit
gig PULPTR
export myshot=26
n testLighting 900.$myshot lighting
doMaya maya/scenes/lighting_v0002/

PULP Publicity Test Shot

  • Exit any existing 'gig': 
    exit
gig PULPTR
  • set shot number: 
    export myshot=26
  • Create a test shot: 
    dupeShot PULP/posesPhase1/group.001 posesPhase1/group.0$myshot --deepCopy --shotgun
  • Navigate to the directory/shot: 
    n posesPhase1 group.0$myshot lighting
  • Run publicity_setup 
    publicity_setup
  • git to PULP: 
    exit
gig PULP
export myshot=26
  • Propagate the training data: 
    propagateTrainingData /disney/shows/PULP/work/Sequences/seq/posesPhase1/group.001/publicity/maya/scenes/publicity_v0032/ /disney/shows/PULPTR/work/Sequences/seq/posesPhase1/group.0$myshot/publicity/maya/scenes/publicity_v0002/
    • confirm
    • ..takes a while.
  • Open scene to work on: 
    exit
gig PULPTR
export myshot=26
n posesPhase1 group.0$myshot lighting
doMaya maya/scenes/publicity_v0002/
