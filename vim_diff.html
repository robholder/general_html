Vim Diff

  • diffput # Push difference to other buffer 
    dp
  • diffget # Pull difference from other buffer 
    do
  • diffupdate # Update diff to show current differences 
    diffu
  • To diff on all open windows: 
    :windo diffthis
:windo difft

(Else, run 'diffthis' in each buffer.)

  • To revert to regular vim: 
    :diffoff!
:diffo!

(Note: exclamation mark indicates 'all windows/buffers'.)

  • To open two files in Diff Mode: 
    vimdiff [file 1] [file 2]
# or
vim -d [file 1] [file 2]
  • Launch another file in vertical split diff (starting from regular vim and a single file: 
    vert diffsplit <file>
# or
vert diffs <file>
  • Switch between windows: 
    Cntl-W Cntl-W
  • Jump to next change: 
    ]c
  • Jump to previous change: 
    [c
