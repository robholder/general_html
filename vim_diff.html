Vim Diff

Prepare buffers for diff

<ctrl>w v <-- vertical split
or:

# Vertical split:
:vs

<ctrl>w s <-- horizontal split
or

:sp

Launch another file in vertical split diff (starting from regular vim and a single file: 

vert diffsplit <file>
# or
vert diffs <file>

To open two files in Diff Mode: 

vimdiff [file 1] [file 2]
# or
vim -d [file 1] [file 2]

Create an empty buffer into which you can paste text: 

:enew
# and even change filetype to get the same syntax:
:set ft=python

Example

I keep two files for my vimrc (one with an OSX title, one without). It's a quirk of mine - only the inline title comment (ascii art) is different in the body-text of the files. But it gives me the excuse to practice vimdiff for muscle memory.

  • Start with VIMRC_OSX open.
  • :vs
  • :enew
  • Paste in VIMRC (old non-OSX version probably from GitHub)
  • :windo difft
  • gg
  • ]c
  • Changes:
    • do <-- to get (obtain) changes
    • dp <-- to put changes
  • <ctrl>ww to change windows
  • :diffo

Start to Diff

To diff on all open windows: 

:windo diffthis
:windo difft

(Else, run 'diffthis' in each buffer.) To revert to regular vim: 

:diffoff!
:diffo!

(Note: exclamation mark indicates 'all windows/buffers'.)

Edits

diffput # Push or 'Put' difference to other buffer 

dp

diffget # Pull or 'Obtain' difference from other buffer 

do

diffupdate # Update diff to show current differences 

diffu

Jump to next change: 

]c

Jump to previous change: 

[c

Window Management

Switch between windows: 

Cntl-W Cntl-W

Switch panes (exchange windows): 

Cntl-w x

Change Vertical split to a Horizontal split: 

<ctrl>w H

Change Horizontal split to a Vertical split: 

<ctrl>w J
