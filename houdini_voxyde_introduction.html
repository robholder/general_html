Incidential nodes covered

attribute create - adding arbitrary attribute (not really needed as subsequent nodes can create).
attribute adjust float - editing the arbitrary attribute (in example, randomized). Select component (point, prim etc.)
add - create points, polys or particles
attribute adjust vector - editing the 'P' attribute (noise, and noise type animationg)
attribute noise vector - like above, but with blend option (to blend with input state)
attribute vop - This is the super-duper attribute node. Can reproduce all the others.
Vector to float - Takes vector (3d) and offers three outputs
Fit range - remap values (like -3 to 3 --> 0 to 1: source min is -3, source max is 3, dest min is 0, dest max is 1)
attribute wrangle - This is a VEX version of the attribute vop node (which uses nodes to apply edits)


Inside attribute vop

Color outputs on vop nodes:

  • green - vector
  • teal (brighter than green) - float
  • blue - integer
