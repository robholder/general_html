HScript Variables

Here are some of the most commonly used variables in Houdini expressions:

Frame and Time Variables:

  • $F --> The current frame number.
  • $FF --> The current frame as a floating-point number.
  • $T --> The current time in seconds.

Geometry Variables:

  • $PT --> The point number (index of the current point).
  • $NPT --> The total number of points in the geometry.
  • $CR --> The color components of a point (red, green, blue).
  • $CG
  • $CB
  • $TX --> The position coordinates of a point (x, y, z).
  • $TY
  • $TZ

Attribute Variables:

  • $VX --> The velocity components of a point (x, y, z).
  • $VY
  • $VZ
  • $NX --> The normal vector components of a point (x, y, z).
  • $NY
  • $NZ
  • $AGE --> The age of a particle.
  • $LIFE --> The lifespan of a particle.

Bounding Box Variables:

  • $MINX --> The minimum bounds of the geometry (x, y, z).
  • $MINY
  • $MINZ
  • $MAXX --> The maximum bounds of the geometry (x, y, z).
  • $MAXY
  • $MAXZ
  • $CEX --> The center of the bounding box of the geometry (x, y, z).
  • $CEY
  • $CEZ
  • $SIZEX --> The size of the bounding box of the geometry (x, y, z).
  • $SIZEY
  • $SIZEZ

Primitive Variables:

  • $PR --> The primitive number (index of the current primitive).
  • $NPR --> The total number of primitives in the geometry.
  • $PTPR --> The number of points in the current primitive.

Miscellaneous Variables:

  • $ID --> The unique identifier for particles or points.
  • $BBX --> The bounding box coordinates normalized to the range 0 to 1 (x, y, z).
  • $BBY
  • $BBZ
  • $LIFE --> The lifespan of particles.

These variables are frequently used in expressions to control and manipulate various attributes and behaviors in Houdini. They provide a powerful way to create procedural effects and automate tasks.
