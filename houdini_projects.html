Houdini

Projects

Files and elements in Houdini can be saved virtually anywhere, but it's better to create and manage a 'project' for jobs/projects

Set up a Project directory

  • File > New Project...
  • Choose from the list of directories.
  • Set project directory name.
  • Set location; $HOME/my_project_path/
  • Accept

Save scenes into the root of the project directory.

Set an existing Project

  • File > Set Project...
  • Click $JOB to start navigation:
    • This is to set the machine agnostic initial root

Saving elements and assets in file links

  • Always start import paths with the $JOB variable.
  • This safely references elements from with the Project (making it portable)

Checking reference paths in a scene

  • Render > Pre-Flight Scene...
  • Check if elements are green
  • Red referenced files need to be fixed:
    • Click on referenced files
    • Click on Expression:
      • Erase any path parts that lead to the project dir.
      • Replace up to and including project name with $JOB
