Vim Learning

Whitespace

Remove Linestart Whitespace

:left <-- moves current line to the left edge.
:le <-- short form of above

Select multiple lines before command to move them all:

vap:le <-- move paragraph to left edge

:%le <-- Move all lines in the buffer to the left edge

Shift Codeblocks Right & Left

Unselected Lines

<< <-- moves current unselected lines left
>> <-- moves current unselected lines right

Selected blocks/lines

< <-- moves selected blocks left
> <-- moves selected blocks right
