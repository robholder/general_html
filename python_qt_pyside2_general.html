Changes (from PySide2)

  • From PySide6.QtGui:
    • QRegExpValidator is now QRegularExpressionValidator
  • From PySide6.QtCore:
    • QRegExp is now QRegularExpression

When using escape chars you need the raw flag, e.g.: r"\d" instead of "\d" (...or "\\d")
