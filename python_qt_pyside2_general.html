PySide6 - General

Installing PySide 6

pip3 install PySide6


Webpage with great tutorials:

https://www.pythonguis.com/pyside6/

Changes (from PySide2)

  • From PySide6.QtGui:
    • QRegExpValidator is now QRegularExpressionValidator
  • From PySide6.QtCore:
    • QRegExp is now QRegularExpression

When using escape chars you need the raw flag, e.g.: r"\d" instead of "\d" (...or "\\d")

PySide6 Designer

Qt Designer is launched from terminal/commandline (not from the MacOS Applications directory).

To launch PySide6 Designer run: 

pyside6-designer

Once launched via commandline (terminal), middle-click the Designer app on the Dock and make it a permanent shortcut on the Dock.

RMB click on the Dock item, Show in Folder, and create an alias in the Applications folder.

PySide6 (Designer) UIC Converter

pyside6-uic 

# Using:
pyside6-uic <ui_file_in> -o <py_file_out>

# Example:
pyside6-uic my_spiffy_qtlayout.ui -o my_spiffy_qtlayout.py
