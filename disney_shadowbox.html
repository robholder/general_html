ShadowBox

  • Lighting Repository:
    • Similar to FX Toolshed
  • Icons:
    • Plus:
      • Add snippets and nodes
      • Name
      • File:
        • Either current selection of navigate to file to import
      • Message
    • Person:
      • Click to select:
        • Local:
          • personal area only
        • Show:
          • visible by everyone
    • Location string:
      • you can type in a show
    • Minimize:
      • Reduces panel size
    • Search field:
      • Type in part strings
  • New items:
    • Edit Mode:
      • Add tags (for search)
      • Add screen grab
      • Add Description
    • Click edit again to save.
  • Open File:
    • Will open in a new session
  • Show Folder:
    • Will open the containing folder on disk

Renaming

  • Double-click on item name and rename.

Deleting

  • To delete an item, 'Show Folder' and delete item!
  • Also, if you delete an item, delete the manifest.meta file too!! (I'll get rebuilt on next launch.)

Scripts

  • Where is seen
    • replace with 'script' and
    • OK
    • This makes a blank script:
      • click 'Open'
      • write script, and save, then 'Import' to bring it back in.

DLight ShadowBox

  • Shift-B
  • Or look in Tools:
    • ShadowBox - DLight

IES Profile

  • Select a light:
    • select an IES Profile in ShadowBox
    • Import:
      • It'll drop into the light's variable box
  • When saving, DLight asks for:
    • Material:
      • Lights
      • Shaders
    • Geo:
      • Objects
