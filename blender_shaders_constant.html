Shaders - Constant

To make a constant (no light required) shader in Blender

From Right to Left

  • Working Backwards from output:
    • Material Output:
      • input - Surface (from Emission)
    • Emission:
      • output - Emission
      • input - Color (from Texture):
        • Texture (reading in image):
          • input - Vector (from UV Map):
            • UV Map:
              • output - UV
              • set uv set
      • input - Strength (from Light Path):
        • Light Path:
          • output - Is Camera Ray

Left to Right

  • From 'left to right':
    • UV Map (output UV, and set to uv set)
    • Texture (reading in image, output Color
    • Light Path (output 'Is Camera Ray')
    • Emission:
      • Color (input from Texture)
      • Strength (input from Light Path)
    • Material Output:
      • Surface (input from Emission)
