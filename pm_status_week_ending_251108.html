Color Patch

Released tool with this announcement:

Rob Holder 11:08 

Color_Patch
There is a new tool available for helping to remove unwanted patches from the character paint passes - Color_Patch. It's accessible from the Custom Scripts for avoa > Tools > Color_Patch menu (or via 'tab' and typing the node name).
Color_Patch is a one-step way of fixing color patches that have been called out for removal on character paint passes. The benefit is automatic integration into the color templates' matte creation nodes. Following the patch fixing edit, the standard matte extraction nodes should just work without any further noodling or fixing, ignoring the original (removed) patches.
The tool's page has an animated screen capture of the process:
https://paramount.atlassian.net/wiki/spaces/PFA/pages/690848189/Color_Patch
