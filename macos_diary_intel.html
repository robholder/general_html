To check status after reboot:

Checking TRIM Status: 

  1. Click on the Apple menu and hold down the Option key.  
  2. Click on System Information.  
  3. In the left sidebar, scroll down and click on NVMExpress. 
  4. Look for TRIM Support in the information panel – it will show either Yes (enabled) or No (disabled). 

In my case, the M3 MacBook Pro said it was NOT disabled.

To restore: 

sudo trimforce enable

260228

Changed Hostname via Terminal (Most Reliable)

  • HostName — used for networking and remote access.
  • LocalHostName — used for Bonjour (.local).
  • ComputerName — what appears in Finder and Sharing. 

Ran each command with IntelMacBookPro: 

sudo scutil --set HostName newname
sudo scutil --set LocalHostName newname
sudo scutil --set ComputerName newname

Then restarted Mac to apply changes everywhere.

This method is recommended when the UI doesn’t fully update all names, which can happen on older macOS versions.

🧭 What Each Name Actually Does

  • Computer Name — visible name in AirDrop, Finder, and Sharing.
  • Local Hostname — becomes newname.local for Bonjour services.
  • HostName — used by SSH, Terminal prompts, and network services. 

UPDATE: Subsequently, Gemini suggests this procedure:

  • MacOS - Renaming Machine
