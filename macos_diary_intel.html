I was using:

  • DejaVu Sans Mono for Powerline for MacVim.

I decided to try Maple Mono: Link: Maple Mono Font 

brew install --cask font-maple-mono

In the Terminal, I could set the font in Preferences and change the scheme.

In MacVim, I had to go to Edit > Font > Show Fonts and choose Maple Mono (regular, 12).
