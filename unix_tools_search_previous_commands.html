Reuse Previous Commands (History)

bck-i-search

<ctrl> r [start of command] <-- this will start a search of previous commands in your history - Enter to complete. (Either continue to populate the command, or r until it finds the best match).
<ctrl> c <-- to abort search without running a command. (You may have to do this twice depending on platform.)

Example: 

# <ctrl> r

# type 'v'

v
>> vim a_file.txt
# <ctrl> r
>> vim -o myfile[mc].*
# <ctrl> r
>> clever_tool -v stuff

# Enter to run command
