Add A Path to pluginPath()

Run this in the Script Editor in Nuke to temporarily add a directory to the pluginPath, enabling the loading of scripts by import. 

nuke.pluginAddPath(test_path)

A robust script to add the path and test its success: 

import nuke
import os

# --- 1. Define the Path ---
# Use a simple, absolute path for testing. Replace with your actual path.
test_path = "/Users/noddy/Documents/Projects/Projects_apps/Nuke/pm_framehold/" 
# Ensure the path uses the correct separators for the OS
test_path = os.path.normpath(test_path)
# On Windows, use double backslashes or a raw string: r"C:\path\to\gizmos"

# --- 2. Print Current State ---
print("--- BEFORE APPEND ---")
print(nuke.pluginPath())

# --- 3. Append the Path ---
nuke.pluginAddPath(test_path)

# --- 4. Print New State ---
print("\n--- AFTER APPEND ---")
print(nuke.pluginPath()) 

# --- 5. Final Check ---
if test_path in nuke.pluginPath():
    print(f"\n✅ SUCCESS: '{test_path}' found in pluginPath.")
else:
    print(f"\n❌ FAILURE: '{test_path}' was NOT added or was removed immediately.")

Importing A Module (as you write it)

Use importlib from Python 3 onwards. (reload alone worked in Python2.)

You can also use aliases with 'reload': 

import importlib
import framehold_set as fs
importlib.reload(fs)

# optionally run a function
fs.main()
