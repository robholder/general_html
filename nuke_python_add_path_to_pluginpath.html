Add A Path to pluginPath()

Run this in the Script Editor in Nuke to temporarily add a directory to the pluginPath, enabling the loading of scripts by import. 

import nuke
import os

# 1. Define the directory path you want to add.
# It's best practice to use absolute paths.
new_plugin_directory = "/path/to/your/custom/gizmos" 

# For a platform-independent approach, you can construct the path:
# new_plugin_directory = os.path.join(os.path.expanduser('~'), 'Nuke', 'MyGizmos')

# 2. Check if the directory is already in the plugin path (optional, but good practice)
if new_plugin_directory not in nuke.pluginPath():
    # 3. Add the directory to the plugin path
    nuke.pluginPath().append(new_plugin_directory)
    print(f"Successfully added {new_plugin_directory} to Nuke's plugin path.")
else:
    print(f"Directory {new_plugin_directory} is already in Nuke's plugin path.")#

Importing A Module (as you write it)

Use importlib from Python 3 onwards. (reload alone worked in Python2.)

You can also use aliases with 'reload': 

import importlib
import framehold_set as fs
importlib.reload(fs)

# optionally run a function
fs.main()
