Hacking With Swift - 03 - How to store whole numbers

Integer

A whole number (no decimal)

How to Store Whole Number (Integer)


let score = 10

For large numbers: 


let reallyBig = 100000000

# or use 'invisible to swift' underscores to separate figures:
let reallyBig = 100_000_000

# Even this is ok to Swift:
let reallyBig = 10____000__________000

You can do calculations during assignments: 

let lowerScore = score - 2
let higherScore = score + 10
# and this can be done with / and *

Math shortcuts or Special operations: 


var = counter = 10
counter += 5
print(counter) # -> 15

Methods (functions of the 'Class'): 


lset number = 120
print(number.isMultiple(of: 3)) # -> True

# Same result as:
print(120.isMultiple(of: 3)) # -> True
