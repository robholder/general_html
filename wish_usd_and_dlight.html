What's New In DLight?

Hierarchy

  • Lights:
    • Create Lights:
      • RMB > Create >CreateLight > dome etc.
    • You can create lights in the tools tab of DLight also:
      • Same tab as the one with the 'hammer' tool
    • Lights must have _llc suffix:
      • This is to have a filter handle for existing tools
      • There is no locator in Maya BTW:
        • Locations are transforms in the Outliner
        • Part of the shader
        • Multiple lights to one location now uses edit-groups

Lights

  • No 'World' palatte:
    • Replacement is shot_pallete
  • Enable/Disable blue 'local change' indication:
    • RMB in diagonal column labels:
      • 'Indicate Attribute Local Modifications'
  • Light edits:
    • Material Editor:
      • Split into Sections:
        • toggle on/off sections using center column
      • User Vars:
        • This is a separate tab

TMODS

  • Can no longer layer multiple TMODS
    • TMODs can be child of light:
    • Add additional TMOD for light for EACH target

TMOD edit groups

This page requires access permissions:
https://gogo.disneyanimation.com/pages/viewpage.action?spaceKey=GT&title=Many+Light+Locators+To+One+Material%3A+User+Workflow

Lights in an edit group act like 'clones' in Nuke.

  • Select lights:
    • Click sub-palette button
    • Rename palette (with '_EG' suffix)
    • RMB on one light:
      • Create Edit Group
      • Name
      • OK
    • Select the remaining lights of the group:
      • Assign the edit group name to them (Edit Group column)
    • Remove by RMB and remove

Ambient Light

  • A mode in the Dome light (sky):
    • Ambient Light Mode: Flat Shading

Distance Falloff

  • Distance Falloff setting on lights:
    • 1.0 - Physical
    • 0.5 - Linear
    • 0.0 - Off (no falloff)

Visibility Scale

  • TMOD:
    • visibilityScale
    • Adjust contribution of target light (geo to shadows)

Emission Numbers

The emission set context menu doesn't show the 'used' emission numbers yet.

Compare Lights (Shaders)

  • select lights (shaders)
  • RMB:
    • Compare lights (shaders)

You can RMB and 'add' lights (shaders).

Undo limitation

Undo is not implemented (although there is a basic one for common tasks/parameters.
