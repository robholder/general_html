Reverting to a Commit

Notes from:
https://www.freecodecamp.org/news/git-revert-file-reverting-a-file-to-a-previous-commit/

In summary: 

git checkout <SHA-hash> -- <file-path>

How To Find The Commit SHA/ID

For all commits: 

git log

For summary (as one line for each commit): 

git log --oneline

How To Find The Commit SHA/ID for a Single File

Example: 

git log --oneline <path_to_file>

# examples:
git log --oneline README.md
 
# Or:
git log --oneline src/App.js

How to Revert a File to a Previous Commit

git checkout <SHA/ID> -- <path_to_file>
 
# examples:
git checkout da9cc5f -- README.md
 
# Or:
git checkout 55a1dff -- src/App.js
