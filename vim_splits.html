Vim - Split Windows

Open Multiple Files

  • Open files: 
    # Horizontally:
vim -o <file> <file>
#or
vim -p <file> <file>
# Vertically:
vim -O <file> <file>

Close Other Windows

  • Close all other windows, leaving only the one in focus: 
    :only
  • Another option
  • %bd - all files; pipe to next command; edit last file used from buffer list: 
    :%bd|e#
  • Closing a window: 
    :bd #(Buffer delete)

New Split Windows

  • Horizontal split: 
    :split <file>
:sp <file>
:new
  • Vertical split: 
    :vsplit <file>
:vs <file>
:vnew

Bring Existing Buffers Into Splits

  • To see available buffers: 
    ls
  • To switch to one, use the number from the list: 
    :buffer#
:buffer2
#or
:b#
:b2
  • Open existing buffer in vertical split: 
    :vert sb# (where # is the number of the buffer to load)
:vert sb2

Equalize two windows

  • Ctrl with equals sign: 
    Ctrl+W =
Ctrl+W -

Resize Windows

  • If you have the mouse enabled, you can literally drag the border: 
    :set mouse=a

Adjustments

  • Increase/Decrease height (very small!) 
    Ctrl+W +
# or:
:res +#
Ctrl+W -
# or:
:res -#
  • Increase/Decrease width (very small!) 
    Ctrl+W <
# or:
:vertical res +#
Ctrl+W >
# or:
:vertical res -#
  • Set height (no count maximizes): 
    #Ctrl+W _
50Ctrl+W _
# or
:resize #
  • Set width (no count maximizes): 
    #Ctrl+W |
50Ctrl+W |
# or
:vertical res #

Switch Windows

  • Switch two windows/splits: 
    Ctrl-W x

Horizontal/Vertical

  • To Switch from Vertical and Horizontal: 
    # Vertical to Horizontal:
Ctrl+W J
# Horizontal to Vertical:
Ctrl+W H
