Hacking With Swift - 20 - How to reuse code with functions

Functions

Format: 

func showWelcome() {
    print("Welcome to my app!")
}

// Run with:
showWelcome()

Configuration Options

This syntax is not common in programming languages: 

// Notice the configuration options of (in: 1...20) for random output from 1 to/inc 20:
let roll = Int.random(in: 1...20)

//this doesn't work by the way:
let roll = Int.random()

Example

Notice 'number' as an 'Int:

  • 'number' is a parameter (placeholder)
  • 'Int' (or '5' in our actual example) is an argument (value) 
    func printTimesTable(number: Int) {
    for i in 1...12 {
        print("\(i) * \(number) is \(i * number)")
    }
}

printTimesTables(number: 5)

Adding another parameter: 

func printTimesTable(number: Int, end: Int) {
    for i in 1...end {
        print("\(i) * \(number) is \(i * number)")
    }
}

printTimesTables(number: 5, end: 20)

Order

The parameters and arguments must be supplied in the order they are stated in the function definition.

Scope

The contents of the function are destroyed once the function is run. The definition is still there, but the actual calculations are destroyed.
