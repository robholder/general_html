PyEnv - VirtualEnv - Anaconda

Using Pyenv-virtualenv to install Anaconda

Ater getting pyenv, and pyenv-virtualenv installed and working, I wanted to attempt to get Anaconda (and Jupyter Notebooks) up and running. 

# Install anaconda3
pyenv install anaconda3-2024.02-1

# Activated the version in a shell:
pyenv shell anaconda3-2024.02-1

# Launched notebook:
jupyter-notebook

# Notebook opened in chrome - then quit from Chrome page.
