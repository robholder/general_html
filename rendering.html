Rendering

Items

  • Cameras
    • Create from shelf or viewport
    • Lock camera to move via viewport view
    • Use Switcher node to switch between cameras in an animation
    • Different cameras available
      • Stereo
      • VR Camera
      • Gamepad Camera
  • Lights
    • Create from shelf
    • Look through like a camera, and lock to move
    • Types of lights:
      • Environment Light: surrounding sphere/hemisphere
      • Point Light: emits from point
      • Spot Light: radiates a cone
      • Distant Light: parallel rays
      • Area Light
        • Line
        • Tube
        • Grid
        • Disk
        • Sphere
      • Sky Light: Environment Light with sky map enabled
      • Geometry Light: uses geometry surface & shader for colors
      • Portal Light: Environment light via portal geometry
