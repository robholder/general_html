Vim - Folding

  • zi to toggle folding on/off
  • zo to open folded lines
  • zc to close folder lines
  • za to open/close folded lines
  • zR top open ALL lines
  • zM/zm to close ALL lines
  • zf create
  • zd/zE delete fold markers
  • zj, zk - next and last fold (Not default keys - set in vimrc):
    • Set up here:
    • My Folds Vimrc Settings

Fold methods

  • set foldmethod=indent
  • set foldmethod=manual
  • set foldmethod=marker
  • set foldmethod=syntax
  • set foldmethod=expr
  • shorcut is fdm

For Python

:set foldmethod=indent <-- Set to intent
:set fdm=intent <-- Abbreviation of above
:set foldnestmax=2 <-- For nested stuff
:set fnm=2 <-- Abbreviation of above
:set foldminlines=1 <-- minimum number of screen lines displayed on fold
:set fml=1 <-- Abbreviation of above

Persistant Folds

To remember the folds in a file, add to vimrc: 

" Save and restore folding status in a file:
augroup save_folds
    autocmd!
    autocmd BufWinLeave *.* mkview
    autocmd BufWinEnter *.* silent loadview
augroup END

Indent

This looks for indented lines (useful for Python, where syntax is largely weak with folding).

Manual

  • zf{movement} # to make folds:
    • zf10j # fold ten lines down
    • zfip # fold inner paragraph

Marker

  • use {{{ to mark start
  • and }}} as end of fold
  • Use regular escape characters to hide from syntax errors:
    • for VIMRC: 
      " Vimrc fold marker ---------------------------------------------------------{{{
" Inside stuff
" }}}

Syntax

  • This works with the syntax formulas in vim.

Expression

  • Uses expression like this (to fold lines that start with p): 
    :set foldexpr=getline(v:lnum)[0]==\\"p\\"

Foldcolumn (Show 'Folds' in Margin)

Hide/Remove fold column

  • set foldcolumn=0

or

  • set fdc=0

Set depth of fold column

  • set foldcolumn=1
  • set foldcolumn=2

or

  • set fdc=1
  • set fdc=2

Set Fold Level

  • set foldlevel=1

or

  • set fdl=1
