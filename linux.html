Setting 'Workspaces' (multiple desktops):

  • Right click on desktop > Display Settings
  • In the settings
    • Scroll up on the left panel of settings
    • Select - Multitasking
  • Scroll down to 'Workspace' in the main window (right)
  • Select 'Fixed number of workspaces'
  • Set your preferred number (I'm using 6)
  • Close the settings.

You should see 6 spaces appear (or however many you chose). These should be more robust then the dynamically created ones.
