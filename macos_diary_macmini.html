260220

Changed username

  • Changing Login Account details
  • I called my account 'firstnamesurname' which was too long.

Fix:

  • Make an alternative Admin account (AltAdmin perhaps).
  • Log into new alt admin account Go to /Users
    • change the account directory (firstnamesurname folder) to the new name (username)
  • Back in the Users & Groups panel, Ctrl-click on the original account
    • Change account and home directory to match the new ones:
    • Change name of account (from firstnamesurname to username)
    • Change name of original home dir:
      • /Users/firstnamesurname folder to /Users/username.
    • Reboot, and log into the newly changed account.

Changed Machine Name (as in @<machine_name>)

In the following, HostName is literal, <new_name> is any name of your choice 

sudo scutil --set HostName <new_name>

Installing PyEnv

See notes at:

  • PyEnv

Add Pyenv startup commands to ~/.zshrc by running the following in your terminal: 

echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.zshrc
echo '[[ -d $PYENV_ROOT/bin ]] && export PATH="$PYENV_ROOT/bin:$PATH"' >> ~/.zshrc
echo 'eval "$(pyenv init - zsh)"' >> ~/.zshrc

Then: 

echo 'export PYENV_ROOT="$HOME/.pyenv"' >> ~/.zshrc

echo 'eval "$(pyenv init - zsh)"' >> ~/.zshrc

Restart your shell: for the PATH changes to take effect.
