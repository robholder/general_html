Blank (Template) Blinkscript

This script can be entered into a new Blinkscript node (and saved as a default node!). 

kernel BlankKernel : ImageComputationKernel<ePixelWise>
//declares 'kernel'
//kernel 'name'
//kernal definition
//kernal definition type
//
//Another example:
//kernel ExampleKernel : ImageComputationKernel<eGranularity>
{
    Image<eRead, eAccessPoint, eEdgeClamped> src; // the input image
    Image<eWrite> dst; // the output image
    
    
    param:
      // This parameter is made available to the user.
      
    local:
      // This local variable is not exposed to the user.
      
    // In define(), parameters can be given labels and default values.
    void define() {
    
    }
    
    // The init() function is run before any calls to process().
    // Local variables can be initialized here.
    void init() {
    
    }
    
    void process() {
    
        // Write the result to the output image
        dst() = src();
    }
};
