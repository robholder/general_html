Git

The Git Secret:

  • Commits are the envelopes (Who, When, Why).
  • Trees are the manifests (A list of every file in the project).
  • Blobs are the content (The actual compressed text of your code).
  • Re-use: If a file doesn't change, the next Tree simply points to the old Blob. This makes Git incredibly small and incredibly fast.
