Tmux - Vi Bindings

To make Tmux feel like an extension of Vim, you’ll want to change the navigation keys from the clunky arrow keys to h, j, k, and l.

Add these to your ~/.tmux.conf file: 

# 1. Change the prefix from Ctrl-b to Ctrl-a (easier to reach)
set -g prefix C-a
unbind C-b
bind C-a send-prefix

# 2. Use Vim keys to move between panes
bind h select-pane -L
bind j select-pane -D
bind k select-pane -U
bind l select-pane -R

# 3. Use Vim keys to resize panes
bind -r H resize-pane -L 5
bind -r J resize-pane -D 5
bind -r K resize-pane -U 5
bind -r L resize-pane -R 5

# 4. Better splitting keys (more visual)
bind | split-window -h
bind - split-window -v
unbind '"'
unbind %

# 5. Enable mouse support (useful while learning)
set -g mouse on

# 6. Faster escape time (Vim users need this for 'Esc' to be instant)
set -s escape-time 0

Why this is a "Vimwiki" Win:

  • The Prefix: Ctrl-a is much closer to your home row than Ctrl-b.
  • The Splits: Instead of remembering % or ", you just use | (vertical) and - (horizontal).
  • Navigation: You can jump from your Vimwiki index to a terminal pane on the right just by hitting Ctrl-a then l.

How to apply these changes:

  • Save the file.
  • Inside an active Tmux session, press Ctrl-b (or Ctrl-a if already updated) then : and type:
    • source-file ~/.tmux.conf

Pro Tip : Since you are using WezTerm, you can now hold Shift and use your mouse to click and drag pane borders if the keyboard shortcuts feel too new!

Would you like to know how to enter "Copy Mode" so you can scroll back through your terminal history using j and k just like in Vim?

Copy Mode

To scroll through your terminal history using Vim keys, you use Copy Mode. This is Tmux's "Vim mode" for the terminal.

1. Enter Copy Mode

Press your Prefix (Ctrl-a) followed by [.

  • You’ll see a line counter (e.g., [0/100]) appear in the top-right corner.
  • Your cursor is now "decoupled" from the prompt, allowing you to move freely.

2. Enable Vim Keys for Scrolling

By default, Tmux uses Emacs keys. To make it feel like Vim, add this to your ~/.tmux.conf: 

# Use Vim keys in copy mode
setw -g mode-keys vi

# Setup 'v' to begin selection as in Vim
bind-key -T copy-mode-vi v send-shell -X begin-selection
bind-key -T copy-mode-vi y send-shell -X copy-selection-and-cancel

3. How to Navigate

Once in Copy Mode (Prefix + [), use these familiar keys:

  • k / j: Up and down one line.
  • Ctrl-u / Ctrl-d: Half-page up and down.
  • g / G: Jump to the top or bottom of the history.
  • /: Search forward (just like in Vim).
  • n / N: Next or previous search result.

4. How to Exit

Press Enter or q to drop back down to your active command prompt.

Why this is a "Vimwiki" Win:

If you’re taking notes and realize you missed a command output from 5 minutes ago, you can:

  • Prefix + [ to go up.
  • Find the text.
  • Press v to start selecting, y to yank (copy).
  • Jump back to your Vimwiki buffer and press p to paste it right in.

Would you like to know how to link your Tmux clipboard to your macOS system clipboard so your "yanks" work in other Mac apps?

Yanking to the MacOS Clipboard

To link your Tmux clipboard with the macOS system clipboard (so your yanks and selections are available in other apps), you need to "pipe" the copied text to the macOS pbcopy command.

On macOS Tahoe, you can achieve this by adding specific keybindings to your ~/.tmux.conf that use the copy-pipe feature.

1. Update your .tmux.conf

Add these lines to your configuration file: 

# Use Vim keys in copy mode (if not already set)
setw -g mode-keys vi

# Copy and yank directly to the macOS system clipboard
bind-key -T copy-mode-vi y send-keys -X copy-pipe-and-cancel "pbcopy"
bind-key -T copy-mode-vi MouseDragEnd1Pane send-keys -X copy-pipe-and-cancel "pbcopy"
bind-key -T copy-mode-vi Enter send-keys -X copy-pipe-and-cancel "pbcopy"
  • y: While in Copy Mode, pressing y will copy the selection and send it to your Mac clipboard.
  • MouseDragEnd: If you use the mouse to highlight text, it will automatically copy to the system clipboard as soon as you let go.

2. Enable Clipboard Support in WezTerm

Modern terminals often require an explicit setting to allow applications like Tmux to access the system clipboard. Ensure this is at the top of your ~/.tmux.conf: 

set -s set-clipboard on

3. How to use it

  • Keyboard: Enter Copy Mode (Prefix + [), select text with v, and press y.
  • mouse: simply click and drag to highlight text. when you release the mouse button, it is instantly ready to paste into any mac app with cmd + v.

Pro Tip for macOS Tahoe

macOS Tahoe introduces a new Spotlight Clipboard History. If you copy multiple items in Tmux, you can view your full history by pressing Command + Space then Command + 4.
