Tmux - Windows & Sessions

In Tmux-speak, a "Tab" is called a Window, and a "Session" is a collection of those windows.

Here is the quick refresher on the "Big Three" levels of Tmux:

1. Creating a New Window (The "Tab")

If you are already inside Tmux and want a new "tab" at the bottom of your screen:

  • Shortcut: Ctrl + b then c (c = create)
  • Command Line: tmux new-window

Pro Tip: To move between these "tabs," use Ctrl + b then n (next) or p (previous).

2. Creating a New Session

A session is an entirely separate workspace. You use these to keep different projects apart (e.g., one session for "Work," one for "Personal").

From outside Tmux (your normal terminal):

tmux new -s my_session_name

From inside an existing Tmux session:

  • Shortcut:
  • <Ctrl> b then : (to enter command mode), then type:
    • new -s session_name
  • To switch between sessions:
    • <Ctrl> b then s (shows a list you can scroll through).

3. Creating a New Pane (Splitting the Screen)

Often, people say "tab" but they actually want to see two things at once on the same screen. This is a Pane.

  • Vertical Split: <Ctrl> b then %
  • Horizontal Split: <Ctrl> b then "

NOTE: I've remapped to | & _ respectively.

Summary Table for your Cheat Sheet

Action Shortcut (After Ctrl+b) Description
New Window c Like a browser tab.
Rename Window , (comma) Keeps your tabs organized.
List Sessions s Switch between project workspaces.
Vertical Split % Side-by-side panes.
Horizontal Split " Top-and-bottom panes.
