MacOS - Mapping Jumplist in Xcode

Install Karabiner-Elements, and allow all the premissions it asks for from the System.

Xcode - ' o' and ' i' to 'jumplist navigation'

Xcode has a basic Vim emulation option. Editor > Vim Mode.

This mod makes <Ctrl> o and <Ctrl> i navigate back and forth between the edit jumplist.

  • Add this to the Karabiner-Elements Settings > 'Complex Modifications': 
{
    "description": "Map Ctrl+o and Ctrl+i to Xcode Jump History",
    "manipulators": [
        {
            "conditions": [
                {
                    "bundle_identifiers": [
                        "^com\\.apple\\.dt\\.Xcode$"
                    ],
                    "type": "frontmost_application_if"
                }
            ],
            "from": {
                "key_code": "o",
                "modifiers": { "mandatory": ["left_control"] }
            },
            "to": [
                {
                    "key_code": "left_arrow",
                    "modifiers": ["left_control", "left_command"]
                }
            ],
            "type": "basic"
        },
        {
            "conditions": [
                {
                    "bundle_identifiers": [
                        "^com\\.apple\\.dt\\.Xcode$"
                    ],
                    "type": "frontmost_application_if"
                }
            ],
            "from": {
                "key_code": "i",
                "modifiers": { "mandatory": ["left_control"] }
            },
            "to": [
                {
                    "key_code": "right_arrow",
                    "modifiers": ["left_control", "left_command"]
                }
            ],
            "type": "basic"
        }
    ]
}
