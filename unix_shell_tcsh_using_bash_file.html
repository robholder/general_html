Using Bash script in tcsh

I wanted to delete file 0001 thru 1001 (mistakingly rendered files). Tcsh is limited, so I made a bash file as follows, chmod +x <file and ran ./<file> 

#!/usr/bin/env bash
for i in {1..1000}; do
    printf -v num "%04d" "$i"
    rm "2230_0350_comp_v24.$num.exr"
done
