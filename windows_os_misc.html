Windows

  • Resize window:
    • <Meta>+RMB
    • <Alt+F8> arrow (enter, or escape to abort):
      • Arrow - adust
      • Shift + arrow - fast adjust
      • Alt + arrow - slow adjust
  • Move window:
    • <Meta>+LMB
  • Move window with keys:
    • <Alt+F7> arrow (enter, or escape to abort)
    • <Alt+F7+Shift> arrow # for faster movement
    • <Alt+F7+Ctrl> arrow # for slower movement
  • Set Always on Top:
    • Alt + Space then 'T' (Toggles)
