Windows

  • Resize window:
    • +RMB
    • arrow (enter, or escape to abort):
      • Arrow - adjust
      • Shift + arrow - fast adjust
      • Alt + arrow - slow adjust
  • Move window:
    • +LMB
  • Move window with keys:
    • arrow (enter, or escape to abort)
    • arrow # for faster movement
    • arrow # for slower movement
  • Set Always on Top:
    • Alt + Space then 'T' (Toggles)
