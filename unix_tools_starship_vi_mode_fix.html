Starship

Starship's 'eval "${starship init zsh)"' can cause issues when using vi mode on the commandline. It sounds like it's something to do with it being invoked more than once.

This is a suggested fix (on macoS) - a conditional BEFORE the standard eval line: 

if [[ "${widgets[zle-keymap-select]#user:}" == "starship_zle-keymap-select" || \
      "${widgets[zle-keymap-select]#user:}" == "starship_zle-keymap-select-wrapped" ]]; then
    zle -N zle-keymap-select "";
fi

eval "$(starship init zsh)"
