Vim in a-shell

  • Vim works just like a regular shell
  • The .vim/ dir can be made in ~/Documents
  • Add the vimrc file inside ~/Documents/.vim/
  • Dir/file structure:
    • .vim/vimrc
    • .vim/colors/

Vim - Plugins

  • Vim is version 8.1 in a_shell:
    • This has the advantage of not needing a package manager.

Installing vim-surround

  • Utilizing Vim Packages (Vim v8 and above):
    • This is a Plugin system within Vim
  • dulwich (Python module):
    • This is a Python package that performs 'git' clones.
  • vim-surround:
    • A test vim plugin to install

Step-by-step

  1. Set Vim's vimrc:
    • ~/Documents/.vim/vimrc
  2. Install Dulwich:
    • in a-shell: 
      pip install dulwich --global-option="pure"
  3. Create directory structure for Vim Packages and vim-surround:
    • The vim-surround plugin page suggests the following dir path: 
      mkdir -p ~/.vim/pack/tpope/start
cd ~/.vim/pack/tpope/start
git clone https://tpope.io/vim/surround.git
vim -u NONE -c "helptags surround/doc" -c q
    • However, the home dir in a-shell isn't conventional; the home drive is:
      • ~/Documents/
    • The ammended, a-shell version would be (and notice plugin name also): 
      mkdir -p ~/Documents/.vim/pack/tpope/start/surround
cd ~/Documents/.vim/pack/tpope/start/surround
    • a-shell doesn't have 'git' installed, so we have to use the python 'dulwich' module:
    • Launch ipython 
      from dulwich import porcelain
porcelain.clone("https://tpope.io/vim/surround.git", ".")
#> <Repo at '.'>
exit
      • This will echo the following if successful:
    • Back in a-shell, run the following: 
      vim -u NONE -c "helptags surround/doc" -c q

Installing vimwiki

With the ipython git substitute installed, and the basic dir structure in place:

  • Using the Vim Packages guidance on the 'vimwiki' Github repo:
    • Create directory: 
          mkdir -p ~/Documents/.vim/pack/plugins/start/vimwiki
    • ipython: 
          cd ~/Documents/.vim
    ipython
    from dulwich import porcelain
    porcelain.clone("https://github.com/vimwiki/vimwiki.git", "pack/plugins/start/vimwiki")
    #> <Repo at 'pack/plugins/start/vimwiki'>
    exit
    • To generate documentation i.e. ':h vimwiki' 
          vim -c 'helptags ~/.vim/pack/plugins/start/vimwiki/doc' -c quit
  • Vimwiki is launched with ww:
    • in vimrc: 
      let mapleader=" "
