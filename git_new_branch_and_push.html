Git - New Branch and Push

You have a repo, but you want to create a new branch and push it to GitHub.

Create a new branch in the current repo (main): 

git switch -c <new_branch_name>

Now you have and are on a new branch called .

To push this new branch to your existing repo on GitHub. 

# Either:
git push --set-upstream origin <new_branch_name>
# or, less verbose but the same:
git push -u origin <new_branch_name>

Example: 

# From main branch:
git switch -c macmini
git push --set-upstream origin macmini
