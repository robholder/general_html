Glob Library

Glob allows for file searching (in contrast to the os library that doesn't): 

import os

os.listdir('/etc/*.conf')

>>>FileNotFoundError

import glob

glob.glob('/etc/*.conf')

>>>['/etc/syslog.conf',
    '/etc/kern_loader.conf',
    '/etc/pf.conf']

Glob

Glob can search with a similar system to RegEx, but it is NOT THE SAME:

Glob/RegEx Differences

meta Glob RegEx
. literal dot any single character
* zero or more characters zero or more of preceding characters
? single occurance of any character zero or one of preceding characters

Glob in use: 

import glob

glob.glob('/etc/[abcd]*.conf')

>>>['/etc/afile.conf',
    '/etc/cfile.conf']

Glob allows for 'recursive' searches (sub-directories): 

import glob

## Use double-asterisks and 'recursive' flag to True:
glob.glob('/<dir>/**/*.txt', recursive=True)

>>> (all .txt files, including from subdirectories, in a list)

iGlob

If you expect a large number of results, consider returning a generator object (iterator): 

import glob

glob.iglob('/Users/rholder/Desktop/*.txt')
>>> <generator object _iglob at 0x105eec3c0>

for one_filename in glob.iglob('/Users/rholder/Desktop/*.txt'):
    print(one_filename)
>>> /Users/rholder/Desktop/afile.txt
    /Users/rholder/Desktop/another.txt
    /Users/rholder/Desktop/lastfile.txt

Dealing with awkward filenames

filename = 'my*file.txt'

glob.escape(filename)

>>>'my[*]file.txt'
