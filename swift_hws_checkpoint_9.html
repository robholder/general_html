Hacking With Swift - Checkpoint 9

Now that you understand a little about optionals, it’s time to pause for a few minutes and try a small coding challenge so you can see how much you’ve remembered.

Your challenge is this:

Write a function that accepts an optional array of integers, and returns one randomly. If the array is missing or empty, return a random number in the range 1 through 100.

If that sounds easy, it’s because I haven’t explained the catch yet: I want you to write your function in a single line of code. No, that doesn’t mean you should just write lots of code then remove all the line breaks – you should be able to write this whole thing in one line of code.

I’ll provide some hints in a moment, but first I recommend you go ahead and try it yourself.

Hints

Still here? Okay, here are some hints:

  • Your function should accept an [Int]? – an array of integers that might be there, or might be nil.
  • It needs to return a non-optional Int.
  • You can use optional chaining to call randomElement() on the optional array, which will in turn return another optional.
  • Because you need to return a non-optional integer, you should use nil coalescing to pick a random number from 1 through 100.

AI Assisted Answer


func pickRandom(_ numbers: [Int]?) -> Int { numbers?.randomElement() ?? Int.random(in: 1...100) }

print(pickRandom(numArray))

🧠 Why this works

✔ numbers is an optional array ([Int]?)

  • So numbers?.randomElement() uses optional chaining.

✔ randomElement() returns an Int?

  • So even if the array exists but is empty, the result is still nil.

✔ ?? provides a fallback

  • If the left side is nil, Swift evaluates: 
Int.random(in: 1...100)

✔ Entire function is one expression

  • Which means it can be written in one line without needing return.

🧪 Example usage


print(pickRandom([10, 20, 30]))   // returns 10, 20, or 30
print(pickRandom([]))             // returns a random number 1–100
print(pickRandom(nil))            // also returns a random number 1–100

let numNilArray = nil as [Int]?
print(pickRandom(numNilArray))

let numArray = [1, 2, 3, 4, 5]
print(pickRandom(numArray))
