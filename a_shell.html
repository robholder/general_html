a-shell

"a-shell" is an open source Linux style terminal environment for iOS.

  • Install from the App Store
  • Use commandline tools like a regular shell:
    • cd ~ will take you to ~/Documents
    • the default landing dir is '~', but that is unwritable.
  • .profile
  • Vim in a-shell
