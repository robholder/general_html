ffmpeg - Extracting Streams

ffmpeg can be installed with Homebrew on MacOS.

Web Link: Extracting Subtitles With ffmpeg

Web Link - How to extract subtitles with ffmpeg

# Getting info on an MKV or MP4 file/container
ffmpeg -i <my_video_file>.mkv

# To get info (like above) but feed the results into Vim:
vim <(ffmpeg -i <my_video_file>.mkv)


# Extracting the top/first subtitle stream from a video file:
ffmpeg -i <my_video_file>.mkv <my_out_file>.srt 

# Extracting a specific subtitle stream from a video file (in this case Stream '#24':
# One of the lines in the infor output was:
# Stream #0:24(kor): Subtitle: subrip
# So use the #0:24 --> 0:24
ffmpeg -i <my_video_file>.mkv -map 0:24 <my_video_file>_Korean.srt
