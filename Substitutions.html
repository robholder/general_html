Substitutions

`:s/old/new/` <-- search and replace on current line (first instance)<br>
`:s/old/new/g` <-- search and replace all on current line<br>
`:s/old/new/gc` <-- search and replace on current line (with confirmation)<br>
`:[range]s/old/new/g` <-- replace on line given in [range]<br>

`:259s/old/new/g` <-- change all on line 259<br>
`:1,5s/old/new/g` <-- change all between lines 1 to 5<br>

`:.,$/old/new/g` <-- change from current cursor position to end of file<br>
`:%s/old/new/g` <-- entire file<br>
`:%s/old/new/gc` <-- entire file with confirmation<br>

`:/<start pattern>/,/<end pattern>/s/old/new/g` <-- change between patterns<br>

`:/local/,166s/old/new/g` <-- change from 'local' search pattern through line 166.<br>

`:s/\/old\/dir\/new\/dir/` <-- change /old/dir to /new/dir<br>
`:s#/old/dir#/new/dir#` <-- same as before, but using arbitrary separaters to avoid backslashes.<br>
Quick clarification: these next two are alternatives based on system; one should work!<br>
`:s/ /<cntl>q<return>/g` <-- Replaces spaces with newlines on a line<br>
`:s/ /<cntl>v<return>/g` <-- Replaces spaces with newlines on a line<br>
The lines above should appear as ``:s/ /^M/g` if prior to ``<enter>` all is well<br>

`:g/^/+1 d3` <-- Deletes 3 lines repeating every 4th (use d<number> to taste)<br>

Summary:<br>
`:[range]s/{pattern}/{string}/[flags]`  format<br>
`:%s/[pattern]/[string]/g` global substitution<br>
