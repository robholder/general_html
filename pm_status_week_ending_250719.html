Conditional Expressions

Sent conditional syntax for a lighting expression (that Lisa is working on) so it is off, on for a duration, then off again: 

frame < 1003 || frame > 1010? 1 : 0

# with expression:
frame < 1003 || frame > 1010? 1 : random(frame/2)*(.9)+.2
  • frame -> will be frame number start (start)
  • frame (second) -> will be frame stop (plus 1) (end +1)
  • 1 -> default value (outside of expression curve) (Default value through the timeline)
  • ...then after the : add any expression you want, which will become active between the start/end frames.
