Title Case Mapping

NOTE: These mappings don't work most of the time. They came from Bing's Copilot, and they only work 'sometimes'.

Add this to your vimrc file: 

" Convert selected text to Title Case
vnoremap <leader>tc :s/\<\w\+/\u\0/g<CR>gv:s/\(\<\w\)\(\w\+\)/\1\L\2/g<CR>gv

Explanation:

Select text in Visual mode by pressing v and using arrow keys or V to select lines.

Invoke the mapping by pressing your leader key followed by tc (e.g., \tc if your leader key is \).

Here’S The Breakdown:

vnoremap <leader>tc: Creates a Visual mode mapping.

:s/\<\w\+/\u\0/g<CR>: Matches each word (\w\+) and converts the first character to uppercase (\u\0).

gv: Reselects the previously visually selected text.

:s/\(\<\w\)\(\w\+\)/\1\L\2/g<CR>gv: Matches the first letter (\(\<\w\)) and the rest of the word (\(\w\+\)) separately, then converts the rest of the word to lowercase (\L\2).

CamelCase Mapping

NOTE: These mappings don't work most of the time. They came from Bing's Copilot, and they only work 'sometimes'. 

" Convert selected text to CamelCase
vnoremap <leader>mc :s/\<./\u&/g<CR>gv:s/\s\+\(\w\)/\1/g<CR>gvnorm guiw<CR>

Explanation:

Select the text in Visual mode (press v and use arrow keys or V to select lines).

Invoke the mapping by pressing your leader key followed by mc (e.g., \mc if your leader key is \).

This mapping converts the selected text to CamelCase by capitalizing the first letter of each word, removing spaces, and ensuring the first character of the selection is lowercase.

Here's The Breakdown

vnoremap <leader>mc "This creates a Visual mode mapping that activates when you press the leader key followed by mc (you can replace <leader> with your chosen leader key, often \ by default).

" Convert the First Character of Each Word to Uppercase:

:s/\<./\u&/g<CR> "This substitution command (:s) matches the first character of each word (\<.) and converts it to uppercase (\u&). The g flag ensures this is applied globally across the selected text.

<CR> "Simulates pressing Enter to execute the command.

" Reselect the Visual Selection:

gv "This command reselects the previously visually selected text.

" Remove Spaces and Convert the Following Character to Uppercase:

:s/\s\+\(\w\)/\1/g<CR> "This substitution command matches one or more spaces (\s\+) followed by a word character (\(\w\)). It removes the spaces and keeps the matched character (\1). The matched character remains uppercase due to the previous step.

<CR> "Simulates pressing Enter to execute the command.

" Reselect the Visual Selection Again:

gv "Reselects the previously visually selected text.

" Convert the First Character of the Entire Selection to Lowercase:

norm guiw "This Normal mode command (norm) moves the cursor to the beginning of the selection (g), moves to the first character (0), and converts it to lowercase (guiw).
