Git - Get Name of Remote Repo

QUESTION

How do I get the current name of the remote repo?

ANSWER

To see the current name (or URL) of your remote repository on GitHub, you have a few direct options:

1. The Standard Way (Verbose List)

The most common command is git remote -v. This will list the "short name" of the remote (e.g., origin) alongside its full GitHub URL. 

git remote -v

Output will look like: 

origin git@github.com:USERNAME/REPOSITORY-NAME.git (fetch)
origin git@github.com:USERNAME/REPOSITORY-NAME.git (push)

2. The Detailed Way (Remote Report)

If you want to see the repository name along with tracking branch information, use git remote show followed by the remote's name: 

git remote show origin

3. The "Clean" Way (URL Only)

If you just want the URL string without the extra "fetch/push" text (useful for scripts or just a quick check), use git remote get-url: 

git remote get-url origin

4. For Your Vim Workflow (Pure Repo Name)

If you want to extract just the repository name (e.g., my-project) from the URL to use in a script or just to see it clearly, you can pipe the output to basename: 

basename -s .git $(git remote get-url origin)

Note: If you ever renamed your local folder, remember that the local folder name and the remote GitHub repository name can be different. The commands above will always show you the actual name hosted on GitHub.
