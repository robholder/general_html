Hacking With Swift - 06 - How to store truth with Booleans

Booleans

Boolean variables store either true or false. 

let filename = "paris.jpg"
print(filename.hasSuffix(".png")) # -> false

let number = 120
print(number.isMultiple(of: 3)) # -> true

Assigning values: 

let goodDog = true
let gameOver = false
let isMultiple = 120.isMultiple(of: 3)

Flipping Values

You can flip values with an exclamation mark: !, or with the 'toggle' method: 


var isAuthenticated = false

isAuthenticated = !isAuthenticated

print(isAuthenticated) # -> true

Or: 


var isAuthenticated = false

isAuthenticated.toggle()

print(isAuthenticated) # -> true
