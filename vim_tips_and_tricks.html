Settings

vim -u NONE -n <-- will launch vim with no configurations but keeps nocompatible goodies. ( -u alt vimrc, NONE is null, -n is no swap file )

vim -u path/to/temp_vimrc_file <-- to open with an alternative vimrc file

:set no<option> <-- Unset the option.
:set <option>! <-- Toggle the option.
:set <option>? <-- Return the option’s value.
:set <option>=<value> <-- Set a value (string or number).
:set <option>+=<value> <-- Add the value for a number option, append a string for a string option.
:set <option>-=<value> <-- Subtract the value for a number option, delete the string for a string option.
:set <option>& <-- Reset the option to its default value.

Drop the ':' when applying these in a vimrc (settings) file

Mapping to 'jk'

Set jk to replace escape key for command/normal mode in the vimrc config file:

inoremap jk <esc>

...or inline as:

:inoremap jk <esc>
...and you may need to enter as follows (depending on system)
:inoremap jk <ctrl>v<esc>
:inoremap jk <ctrl>q<esc>

Rather than entering `<esc>` as literal text, depending on systems, one of the following should be true:
<cntl>v<esc> <-- to enter an characters into command: appears as `^[ THIS IS NOT TYPED
<cntl>q<esc> <-- to enter an characters into command: appears as `^[ THIS IS NOT TYPED

Autocomplete with Tab and Shift-Tab (add to .vimrc)

" START Tab Autocomplete:
" Function to allow for tabbing to autocomplete a word based on the initial typed
" characters. Tab is just Tab if on empty line or start of line.
function! InsertTabWrapper()
    " Set variable 'col' equal to current column in current line (".") minus 1
    " (so that column 1 is zero, or false)
    let col = col(".") - 1
    " If col is zero, get character before the cursor, and check it's whitespace (\k - keyword)
    if !col || getline(".")[col - 1] !~ '\k'
        " If whitespace (non-character) just add Tab:
        return "\<tab>"
    else
        " If NOT whitespace, map control-n to the tab key:
        return "\<c-n>"
endfunction
" Map the tab key to the function above:
inoremap <tab> <c-r>=InsertTabWrapper()<cr>
" Map shift-tab to control-p (to allow for reversing up a list of autocompletions:
inoremap <s-tab> <c-p>
" END Tab Autocomplete:

Mapping 'highlight search' in your Vimrc config

Add to .vimrc for `F4 toggle option: (press f4 to toggle highlighting on/off, and show current value.)
:noremap <f4> :set hlsearch! hlsearch?<CR>
Same thing but mapped to the Backspace/Delete key:
:noremap <BS> :set hlsearch! hlsearch?<CR>
Using Spacebar and clear messages already displayed:
:nnoremap <silent> <space> :nohlsearch<bar>:echo<cr>

Substitutions

:s/old/new/ <-- search and replace on current line (first instance)
:s/old/new/g <-- search and replace all on current line
:s/old/new/gc <-- search and replace on current line (with confirmation)
:[range]s/old/new/g <-- replace on line given in [range]

:259s/old/new/g <-- change all on line 259
:1,5s/old/new/g <-- change all between lines 1 to 5

:.,$/old/new/g <-- change from current cursor position to end of file
:%s/old/new/g <-- entire file
:%s/old/new/gc <-- entire file with confirmation

:/<start pattern>/,/<end pattern>/s/old/new/g <-- change between patterns

:/local/,166s/old/new/g <-- change from 'local' search pattern through line 166.

:s/\/old\/dir\/new\/dir/ <-- change /old/dir to /new/dir
:s#/old/dir#/new/dir# <-- same as before, but using arbitrary separaters to avoid backslashes.
Quick clarification: these next two are alternatives based on system; one should work!
:s/ /<cntl>q<return>/g <-- Replaces spaces with newlines on a line
:s/ /<cntl>v<return>/g <-- Replaces spaces with newlines on a line
The lines above should appear as :s/ /^M/g if prior to <enter> all is well

:%!cat -s <-- Replaces multiple blank lines with a single one

Note: I don't understand this next one, even though it DOES work:
:g/^/+1 d3 <-- Deletes 3 lines repeating every 4th (use d to taste, and space is not necessary)
Explanation:
:g <-- invoke global
/^/ <-- match or search pattern
+1 <-- add a line to the matching line
d3 <--delete 3 times

:g/<string>/d <-- delete all lines with search string

:g!/<string>/d <-- delete all lines NOT containing search string
Or:
:v/<string>/d <-- delete all lines NOT containing search string

Summary:
:[range]s/{pattern}/{string}/[flags] format
:%s/[pattern]/[string]/g global substitution
