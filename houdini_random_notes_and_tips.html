Houdini Environment

SideFX Website - Houdini Environment

There are all kinds of things you can set via the Houdini environment in this file: 

/Users/<username>/Library/Preferences/houdini/<version>/houdini.env

Remember Fewer Recent Files

I appended this to houdini.env. 

HOUDINI_MAX_FILE_HISTORY = 4

...to stop Houdini keeping many filepaths in its file import dropdowns. (This variable can take numbers from 1 to 64.)

Legacy and Recent File Paths in Dialogs

Various file fields (like image import fields) keep a list of recent/not-so-recent file paths, and are shown when you use the dropdown menus at the end of the fields.

These are stored in a file: 

file.history

On MacOS: /Users/<username>/Library/Preferences/houdini/<houdini_version>/file.history 

/Users/myusername/Library/Preferences/houdini/20.5/file.history

Zero-Out Objects' Transforms

On each transforms panel for objects there is a mysterious dropdown Modify Pre-Transform:

  • Clean Transform
  • Clean Translates
  • Clean Rotates
  • Clean Scales
  • Extract Pre-transform
  • Reset Pre-transform

The 'Clean' options effectively set the transforms (all or individual types) to zero.

Reset 'deletes' the Pre-transform values (and can result in the movement of the object).

Extract takes the invisible Pre-transforms and plugs them back into the transform fields.

Reference Name of Parent Node

$OS is the expression or variable to get the current node name.

I wanted to add a Name node to some geometry, and wanted to reference the node (which would be the immediate parent) name to pass into the name field: 

# To get the (input) node name:
`opinput(".",0)`
 
# or to get full path:
`opinputpath(".",0)`
