Houdini Environment

SideFX Website - Houdini Environment

There are all kinds of things you can set via the Houdini environment in this file: 

/Users/<username>/Library/Preferences/houdini/<version>/houdini.env

Set Python Path for Scripts

In the houdini.env add the edited form (for your machine/account) of the following: 

# This is my path to my custom directory for my Python scripts referenced by shelf tools:
PYTHONPATH = /Users/<username>/Documents/Projects/Projects_apps/Houdini/scripts;&

I appended... 

HOUDINI_MAX_FILE_HISTORY = 4

...to stop Houdini keeping many filepaths in its file import dropdowns. (This variable can take numbers from 1 to 64.)

Shelf Tools Script

To merely import and run script (rather then store scripts on the actual buttons), I use this format within the button script: 

import importlib
import <script_name>

importlib.reload(<script_name>)
<script_name>.main()

Boiler Plate code for Button Scripts

# TO SET UP THE ENVIRONMENT TO PICK UP SCRIPTS:
#
# In your Houdini Env file - on a Mac, it's here:
# /Users/<username>/Library/Preferences/houdini/<version>/houdini.env
# add:
# PYTHONPATH = /Users/<username>/Documents/Projects/Projects_apps/Houdini/scripts;&


## Add to Houdini shelf button to call the script:
# import importlib
# import <script_name>
#
# importlib.reload(<script_name>)
# <script_name>.main()
#

import hou
import os

def confirm_action():

    # Get name of script file
    script_name = os.path.basename(__file__)
    # Display the confirmation dialog
    result = hou.ui.displayMessage(f"Are you sure you want to proceed with\n'{script_name}'?", buttons=("Yes", "No"), default_choice=1)

    # Check the user's response
    if result == 0:  # "Yes" button
        return True
    else:  # "No" button
        return False

def main():

    # Call a dialog to ask if to proceed
    if confirm_action():
        # Proceed with your script
        print("Proceeding with the script...")
    else:
        # Cancel the script
        print("Script cancelled by the user.")
        return
    
    # ADD YOUR PYTHON SCRIPT HERE


if __name__ == '__main__':
    main()

Legacy and Resent File Paths in Dialogs

Various file fields (like image import fields) keep a list of recent/not-so-recent file paths, and are shown when you use the dropdown menus at the end of the fields.

These are stored in a file: 

file.history

On MacOS: /Users/<username>/Library/Preferences/houdini/<houdini_version>/file.history 

/Users/myusername/Library/Preferences/houdini/20.5/file.history
