Blender - UVs

Re-Baking With New UVs

UVs

  • Make new UV set:
    • Data panel
    • UV Maps section
    • + to make new 'slot'
    • In the UV editor, switch to this new set and create the UVs layout

Setting up shader and render

  • In shader network:
    • create new Texture Image node:
      • click 'new' and set name and resolution (square)
    • create new UV Map node:
      • set to the new UV set
      • plug from 'UV' into the 'Vector' input of the Texture Image node.
  • Switch to Cycles renderer
  • In 'Bake' section of Cycles panel:
    • Set 'Bake Type' to 'Diffuse'
    • Contributions:
      • Check on 'Color' only.
  • Select both:
    • The model (3d view)
    • The texture node
  • Click on 'Bake' button

Saving image

While viewing the baked map in the Image Editor pane:

  • Image > Save/Save as/Save a copy
