get_iplayer

Only for UK and for License Holders

To read the result of a search of iplayers inventory, and load into Vim: 

get_iplayer ".*" | vim -

You can filter for more precise targets with regex: 

get_iplayer "wallace"

Getting Help

get_iplayer --help

Useful Substitutions

These can be stored in a file on disk, and run from within Vim.

:source my_subs.vim --> Reads the commands from the file 'my_subs.vim' (this should be a full path is in a different directory from the one which Vim launched from).
