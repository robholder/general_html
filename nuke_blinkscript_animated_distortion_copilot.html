Copilot - BlinkScript - Animated Effects (Wavy Distortion)

If you want a procedural distortion effect that waves dynamically, this approach uses sinusoidal motion to warp pixels.

BlinkScript Code: Wavy Animated Distortion

kernel WavyDistortion : ImageComputationKernel<ePixelWise>
{
    ImageParam src;
    param: float waveStrength;
    param: float time;

    void process()
    {
        float2 uv = getNormalizedCoord();

        // Apply wave-like movement based on time
        float waveOffsetX = sin(uv.y * 15.0 + time * 5.0) * waveStrength * 0.01;
        float waveOffsetY = cos(uv.x * 15.0 + time * 5.0) * waveStrength * 0.01;

        // Shift pixel coordinates
        float4 result = src(uv + float2(waveOffsetX, waveOffsetY));

        output() = result;
    }
};

How It Works

  • Sinusoidal functions (sin and cos) create a smooth animated distortion.
  • Time parameter (time) allows dynamic movement over frames.
  • Wave strength (waveStrength) lets you control the effect intensity.
