Hacking With Swift - Checkpoint 5

With closures under your belt, it’s time to try a little coding challenge using them.

You’ve already met sorted(), filter(), map(), so I’d like you to put them together in a chain – call one, then the other, then the other back to back without using temporary variables.

Your input is this: 

let luckyNumbers = [7, 4, 38, 21, 16, 15, 12, 33, 31, 49]

Your job is to:

  • Filter out any numbers that are even
  • Sort the array in ascending order
  • Map them to strings in the format “7 is a lucky number”
  • Print the resulting array, one item per line

So, your output should be as follows: 

7 is a lucky number
15 is a lucky number
21 is a lucky number
31 is a lucky number
33 is a lucky number
49 is a lucky number

I had to look this up with AI. This was the response: 

let luckyNumbers = [7, 4, 38, 21, 16, 15, 12, 33, 31, 49]

luckyNumbers
    .filter { $0 % 2 != 0 }                     // keep only odd numbers
    .sorted()                                   // ascending order
    .map { "\($0) is a lucky number" }          // convert to strings
    .forEach { print($0) }

After getting this response from AI, I looked at the hints on the problem page of the tutorial:

Here are some hints:

  • You need to use the filter(), sorted(), and map() functions.
  • The order you run the functions matters – if you convert the array to a string first, sorted() will do a string sort rather than an integer sort. That means 15 will come before 7, because Swift will compare the “1” in “15” against “7”.
  • To chain these functions, use luckyNumbers.first { }.second { }, obviously putting the real function calls in there.
  • You should use isMultiple(of:) to remove even numbers.

With this in mind, I adjusted my answer to: 

let luckyNumbers = [7, 4, 38, 21, 16, 15, 12, 33, 31, 49]

luckyNumbers
    .filter { !$0.isMultiple(of: 2) }           // Negate the result of divisible by 2 (!$0)
    .sorted()                                   // ascending order
    .map { "\($0) is a lucky number" }          // convert to strings
    .forEach { print($0) }

Since the course hasn't mentioned 'forEach' yet, I asked for a more verbose approach that stores the result in a variable to print (line by line): 

let luckyNumbers = [7, 4, 38, 21, 16, 15, 12, 33, 31, 49]

let results = luckyNumbers
    .filter { !$0.isMultiple(of: 2) }
    .sorted()
    .map { "\($0) is a lucky number" }

for line in results {
    print(line)
}
