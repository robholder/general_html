General Tips

Add Prefix or Suffix to Lines

  • Using line visual to add semicolon at the end of each line in the selection:
  • (Remember, in everyday vim, you can use A for line-append, I for line-prepend.)
  • Select text:
    • VG
    • or
    • ggVG
  • Enter normal mode with :normal,
    • A insert-after
    • or
    • I insert-before:
    • Then semicolon:
      • :normal A;

Full Version:

  • ggVG :norm A;

Add 'xxx' to the end of selected lines of a paragraph

" Visually select paragraph:
vap

" Prepend 'xxx'

:norm Ixxx

" Full command:
vap:norm Ixxx;

" To append xxx
vap:norm Ixxx

Reselect a Visual Selection:

gv

Copy - [range]copy {address}:

:t
# Duplicate current line:
:t.
# Copy from 'here' to end of file:
:t$
# Copy visually selected text to the start of the file:
:'<,'>t0

Move - like copy above: [range] move {address}:

:m <#>

# to move current line (or block selection of lines) to line 2 of the document/buffer:
:m 2

# to move current line (or block selection of lines) up a line: (notice -1 is the current location's line number - add one when going 'up)
:m -2

# to move current line (or block selection of lines) up 10 lines:
:m -11

# to move a current line (or block selection of lines) down a line:
:m +1

# to move a current line (or block selection of lines) down 10 lines:
:m +10

Switch between last buffer and back:

<Ctrl-^>

Use Counts to Do Simple Arithmatic

# Advance number:
<Ctrl-a>
# Decrease number:
<Ctrl-x>

Even if the cursor isn't on a number:

<Ctrl-a>

" and...

<Ctrl-x>

" ...will increment/decrement the next number on the line.

By preceding the increment commands with a number you can perform arithmatic

180<Ctrl-a> # will add 180
180<Ctrl-x> # will subtract 180

NOTE: Vim assumes numbers with leading zeros to be octal notation, so this can result in weird results:

To prevent this behavior: 

"In the vimrc:
set nrformats=
