Table of Window Commands

Here is a thorough reference table for managing and navigating tmux windows.

Command Action Description
prefix + c Create Window Opens a brand new window numbered sequentially
prefix + , Rename Window Clears the status bar prompt to type a custom name
prefix + & Kill Window Closes the current window and kills all active panes inside it
prefix + n Next Window Cycles forward to the next numerical window
prefix + p Previous Window Cycles backward to the preceding numerical window
prefix + 0 – 9 Go to Number Jumps instantly to that specific window index number
prefix + ' Go to Name Opens an index prompt to type a window's exact text name [INDEX]
prefix + w Interactive List Opens a visual tree menu to browse all windows and sessions [INDEX]
prefix + l Last Window Toggles back and forth between your two most recently used windows
prefix + . Move Index Prompts you to change the current window's index number

Useful Window Commands

Here is a quick reference list of handy, advanced commands to speed up your window creation and management workflow:

  • Create Named Window:
    • Run tmux new-window -n "my-name" to bypass the default numbering and open a window with a custom label instantly.
  • Split Window into Panes:
    • prefix + " for a horizontal split, (or prefix + _ in my setup)
    • prefix + % for a vertical split inside your current window (or prefix + | in my setup)
  • Break Pane into New Window:
    • prefix + ! to convert your currently active, zoomed, or focused pane into its own independent window.
  • Kill Current Pane:
    • prefix + x to safely close the individual focused pane rather than destroying the entire window sequence.
  • Toggle Pane Zoom:
    • prefix + z to maximize your current pane to fill the entire window, and press it again to minimize it back to the grid.

Tmux - Window Navigation

You can jump directly to any window using its number prefix or by typing its unique name [INDEX].

Method 1: Jump Directly by Number (Fastest)

Press your tmux prefix key combo (maybe Ctrl + j or Ctrl + b), release it, and then instantly press the number key of the window you want to switch to.

  • prefix then 0 → Switches to window 0
  • prefix then 1 → Switches to window 1
  • prefix then 2 → Switches to window 2

Method 2: Jump by Unique Name

If you have many windows and don't want to memorize numbers, you can search or jump by their unique text name [INDEX]:

  • Press prefix then ' (the single quote key) [INDEX].
  • A prompt will appear in the status bar: Index or name:.
  • Type the unique name of your window (or just the first few starting letters) and press Enter to jump straight to it.

Method 3: Visual Interactive Menu

If you ever forget the numbers or names completely, you can open an interactive visual list [INDEX]:

  • Press prefix then w [INDEX].
  • Use your arrow keys (or j and k if you use Vim keys) to scroll through an interactive tree of your active sessions and windows.
  • Press Enter on the highlighted window to open it.

Pro-Tip: Start Your Window Numbers at 1 Instead of 0

By default, tmux starts numbering windows at 0. Because the 0 key sits all the way on the right side of a standard QWERTY keyboard, it can feel awkward to reach.

To shift your windows so they map intuitively from left-to-right (1, 2, 3...), add these two lines to your ~/dotfiles/tmux/dot-config/tmux/tmux.conf file: 

# Start window and pane numbering at 1 instead of 0
set -g base-index 1
setw -g pane-base-index 1
