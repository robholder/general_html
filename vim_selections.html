Vim - Selections

  • v --> enter selection mode (followed by motion)
  • V --> enter line selection mode (followed by motion)

Returning to Start or End of Last Selection

  • '< --> return to the start of the last/previous selection
  • '> --> return to the end of the last/previous selection
  • '. --> return to the most recently modified line

Selecting To, Find, End

To select all up to a character on the same line (x in this example):

  • vtx --> select from selection to the first x character on the same line.

To select to the end of the line with the newline:

  • v$ --> Select and go to end of line (including the new line)

To select to the end of the line without the newline:

  • vg_:
    • v --> Enter Visual mode with v.
    • g_ --> Move the cursor to the last character of the line by using g_ instead of $. The g_ motion targets the last non-whitespace character of the line, and it excludes the newline.

So the sequence would be: 

vg_

When you yank (y) and paste (p), only the text of the line will be copied, without the newline character.

To select up to a search term or word (multiline):

  • v/<search><enter --> This selects (multiline if necessary) from the initial selection up to the search term.
