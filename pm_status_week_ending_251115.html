Terry Ziegelman

Helped set expression for painterly_pmnt effects on 'twos': 

Setting to Output On Twos
You've got a choice for setting painterly_pmnt for working on 2s. One of these should work for you!

Either set an expression on the Noise Seed field on the painterly_pmnt properties panel, or drop a FrameHold node after the painterly_pmnt node (presuming the painterly_pmnt is currently animated by the expression “frame”.)

Expression
To set the expression, apply the following in the Noise seed field:

floor(((frame)  / 2) ) * 2

…and if that seems out of sync:

floor(((frame - 1)  / 2) ) * 2 + 1

FrameHold

Set the properties of frame hold to:

first frame -> 1001
increment -> 2

...or if it's out of sync (one frame off for synch):

first frame -> 1000
increment -> 2
