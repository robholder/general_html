01-02-Panes And Panels

Example Panel:

  • Scene View:
    • Icons on left <-- selection tools
    • Icons on right <-- Display Options

Tabs

+ <-- choose pane type to open in a panel as a tab

x <-- close pane

You can drag between panels.

Between panes

▶ <-- Collapse or expand pane

▲ & ▼ <-- Expand pane upwards/downwards

... <-- in between the arrows, this icon swaps the panels either side of the icons

White square & triangle icon

□ Maximize/restore pane (or <ctrl-b>)

⛛ <-- to the side of the white icon: menu with many options for pane

Close a pane

Just close ALL the tabs it contains.

01-03-Playbar, Shelves, Desktops and Menus

Playbar

  • Along bottom of UI:
    • Left - Global Animation settings (square icon)
    • Audio (Chops)
    • Right - Keyframing

Shelves

  • Along top of UI:
    • Divider - can be resized
    • Arranged in Tabs
    • ▶ to see more (left and right)


+ in a square <-- Add shelf(s) to tabs (existing or 'new')

Menu

OS style menu along top.

  • File:
    • Projects
  • Edit:
  • Help:
    • Launch help
  • Desktops:
    • Pre-loaded UI layouts
    • Save new layouts
    • If you alter one to create a custom 'desktop', reload current to restore
  • Radial Menu:
    • 'c' key evokes the selected radial menu
    • options for different scenarios
  • Takes:
    • I've used these for different render options
  • ?:
    • launch help browser

02-01-Viewport Navigation

Initial icon is the Camera Viewport tool (left panel). This icon is disengaged if anything is done in the viewport (like instantiating an object).

Viewport Default

  • Invoke Viewport Navigation tool:
    • icon click on 'camera' icon (left panel)
    • <esc> key
    • <space> key (or <alt> key)

Mouse buttons

  • LMB <-- Orbit
  • MMB <-- Pan
  • RMB <-- Dolly (in & out)
  • <cntl> LMB <-- Roll
  • <shift> *MB <-- Finer control
