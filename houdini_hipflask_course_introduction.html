02-01-Viewport Navigation

Initial icon is the Camera Viewport tool (left panel). This icon is disengaged if anything is done in the viewport (like instantiating an object).

Viewport Default

  • Invoke Viewport Navigation tool:
    • icon click on 'camera' icon (left panel)
    • <esc> key
    • <space> key (or <alt> key)

Viewport Mouse buttons

  • LMB <-- Orbit
  • MMB <-- Pan
  • RMB <-- Dolly (in & out)
  • <cntl> LMB <-- Roll
  • <shift> *MB <-- Finer control (all mouse buttons)

Select items

  • LMB <-- Click on item/object (Tool will become the handle tool)

Creating Objects from poly shelf

  • <cntl> LMB <-- In viewport to drop item at origin (otherwise drop where mouse is)

Framing the Viewport

  • <space> F <-- frame selected
  • <space> A <-- frame all
  • <space> G <-- frame selected - homeview (default 3/4 view)
  • <space> H <-- frame all - homeview (default 3/4 view)

02-02-Basic Selections

Selection

Selection in viewport

  • Select tool:
    • Black arrow icon or S key
    • LMB <-- to select
    • <shift> LMB <-- add to selection
    • <cntl> LMB <-- remove from selection
    • A key <-- select all
    • N key <-- deselect all (none)
    • <cntl> i <-- invert selection

Selection Tool options

The top bar shows options when the selection tool is selected:

  • F2 - Box <-- marquee style
  • F3 - Lasso <-- freehand selection
  • F4 - Brush (radius option) <-- Good for points, faces or particles
  • F5 - Laser <-- selects any geo it touches

Secondary options:

  • Eye - <shift> v <-- visible/all
  • Partial - <shift> c <-- /Full enclosure to select
  • Material
  • Constraint

All options are available by RMB on selection icon on left panel.
Things like 'add to selection' can be locked.

Secure selection

  • Lock icon <-- Locks selection (so it can't be altered using other tools
  • ~ <-- tilde toggles selection lock on/off
  • s key <-- circumvents lock temporarily
  • Handle Tool - <ctrl> <-- to allow selection change while using

Selection in Network view

Selection:

  • <cntl> a <-- select all nodes
  • LMB in blank space <-- deselect
  • marquee LMB <-- drag to multi-select

View adjustment:

  • F <-- frame nodes
  • H <-- home view

02-03-Transforms - Part 1

Default units - One grid = 1 meter

Parameters

When you create an object, and a transform tool is used, you have parameters visible at the top of the viewport, but there main area for parameters is in the parameters pane.

To have color coded parameter fields in the parameters pane, set the colors in the preferences:

  • Edit > Preferences > Animation > checkbox for colors

Move Tool

  • Move icon or T key <-- move handles appear on selection:
    • Grab center cube to move in viewport space
    • Grab arrow tips to constrain movement to axis
    • Grab little square 'planes' to constraint movement to a plane (two axis)

02-04-Transforms - Part 2

Rotation Tool

  • Rotate icon or R key <-- rotation handles appear on selection
    • Grab perpendicular circle to constrain movement to axis
    • Grab intersections constraint movement to two axis
    • Grab in between circles to rotate freely in all axis

Scale Tool

  • Scale icon or E key <-- scale handles appear on selection:
    • Grab axis tips to scale along one axis
    • Grab center cube to scale uniformly
    • Gram little scare planes to constrain scale to two axis

Note: Parameter pane has option for 'Uniform Scale' (slider). Ideally only use this on objects, and scale individual axis at geometry level.

World, Object space etc

  • RMB (menu) on handles to choose 'space' for transforms:
    • World
    • Object
    • View
    • C-Plane (construction plane)
  • M key to cycle through 'space' options
  • Choice is common to all transform tools

02-06-The Value Ladder

The Value Ladder

The Value Ladder is a way of interactively adjusting values in increments, usually by powers of ten (0.01, 0.1, 1, 10, 100 etc.). It appears as a vertical stack of powers.
The value appears in the active power box of the ladder graphic.

  • Parameter Pane:
    • MMB <-- over parameters in the parameters pane and adjust by sliding left/right and jumping powers vertically.
    • <shift> MMB <-- allows you to move the cursor off the 'value ladder' to continue adjustments.
    • MMB <-- over parameters label to adjust all axis together.
  • Viewport (on handles of transform tools):
    • MMB <-- over handle axis
    • Note:
      • When adjusting Rotation in the viewport via the Value Ladder, the increments are logical for rotations (1, 15, 30, 45).
      • On parameter pane, it reverts to powers of 10.
  • Strings & Expressions:
    • Select the numerals in a string or expression
    • MMB over the selection to activate adjustments via the Value Ladder graphic.

02-07-Transform Modifier Keys and Middle Mouse Drag

Handle Size in Viewport

Adjust the size of the handles in the viewport using the Multiply and Divide keyboard keys.

Modifier Keys

  • <shift> MMB <-- shift allows for finer controls
  • <cntl> MMB <-- snap motion to increments (0.1 units, and 45º for rotation)

RMB on handles and choose Handle parameters to change increments for each of the transform handles.
The rotations options include a drop-down menu of logical value steps.
Changing the increments has more options for rotation (including to show increments as a graphic circular ruler).

MMB Drag Prefs

  • Edit > Preferences > Handles > Allow Indirect Handle Dragging with MMB:
    • Map drag to axis - constrains motion to first direction of MMB drag.
    • Map drag to c-plane - constrains to 2 axis defined by a construction plane (2nd icon down on right panel of viewport).
    • Map drag to view plane - constrain to perpendicular plane to viewport.

02-09-Transforming An Object's Pivot

Editing and Transforming the Pivot

(while in Move, Rotate or Scale tool)

The pivot is the source point of transformations.

  • To set the pivot to adjust mode:
  • While in MOVE, ROTATE or SCALE tools (NOT handle tool!!!):
    • Insert key
    • <shift> " key (double-quote character)
    • RMB on an object and choose pivot mode

When in 'pivot' mode, the tips of the transform tool's handle axis are wireframe (solid for regular transforms).
Switching to one of the dedicated move tools (t, r or e), exits the pivot edit mode.

The pivot is not restricted to the object's bounds. It can be anywhere in space.

The parameters pane has transforms cells for the pivot as well as the object proper.

Parameter & Viewport adjustments

When interacting with the pivot transformations in the viewport, Houdini treats each edit as a resolved step. This is the most intuitive way to make several adjustments one after another.

In the parameters pane, you have to use the Modify Pre-Transform pop-up to 'Clean transforms' between separate edits. (Otherwise, the numerical adjustments are accumulated into one transform.)

03-01-Handle Detachment

Handle Detachment

This is similar to the pivot edit, but this is a temporary detachment of the Handle tool (rather than the pivot itself, which is done in the transform tools 'proper').

  • To detach the handles of the HANDLE tool:
  • While in the HANDLE tool mode (NOT move, rotate or scale tool!!):
    • <shift> " keys (double-quote)
    • RMB on an object and choose Detach handles

TIP: Grid snapping is handy for this (Magnet on grid icon - left of Viewport panel).
TIP: X key will show a radial menu from which 'snap to grid' can be invoked.

  • Deselecting and re-selecting the object will restore the default position of the transform tool handles.
  • RMB and choose persistent to keep the handle visible and in the edited/detached position. (You can actually use the Handle tool AND the transform tools simultaneously in this way).

Forcing restore of position of Handle tool location

  • RMB > Handle parameters - choose snap to centroid or pivot (as well as Object and World space).
