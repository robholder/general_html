Config and Plugins

Stow Setup for Tmux

To layout tmux configs with plugins using Stow, create a ~/.config/tmux structure in your dotfiles directory, placing tmux.conf and plugin management scripts inside. Stow this to ~/.config/ so tmux automatically detects the configuration and TPM installs plugins there, keeping your home directory clean.

Recommended Directory Structure:

~/dotfiles/
└── tmux/
    └── dot-config/
        └── tmux/
            ├── tmux.conf
            └── tpm/ (or plugins installed here)

Steps to Implement:

  • Create the directory structure in your dotfiles repository:
    • mkdir -p ~/dotfiles/tmux/.config/tmux
  • Move your existing configuration to this new location:
    • mv ~/.tmux.conf ~/dotfiles/tmux/.config/tmux/tmux.conf
  • Create tmux.conf and include Plugin Manager (TPM) settings: 
# Set plugin path
set-environment -g TMUX_PLUGIN_MANAGER_PATH '~/.config/tmux/plugins'
# List plugins
set -g @plugin 'tmux-plugins/tpm'
set -g @plugin 'tmux-plugins/tmux-sensible'
# Initialize TPM
run '~/.config/tmux/plugins/tpm/tpm'

Stow the config:

Run stow tmux from inside your ~/dotfiles directory to symlink ~/.config/tmux/. 

# If you used the 'dot' substitution in your file structure names
# use the --dotfiles flat to force '.' in the symlink name.
stow --dotfiles tmux

Key Rules for --dotfiles

  • Replacement logic: Stow only replaces the prefix dot- at the beginning of a file or folder name.
  • Nested files: You do not need to prefix every file. For example, in dot-config/tmux/tmux.conf, only the top-level dot-config needs the prefix to become ~/.config. The files inside it will retain their original names.
  • Multiple dots: A file named dot-tmux.conf will be stowed as .tmux.conf.
  • Version requirement: This feature was introduced in Stow 2.3.0. Ensure your version is current by running stow --version.

Install Plugins:

  • Clone TPM into your configured directory:
    • git clone https://github.com/tmux-plugins/tpm ~/.config/tmux/plugins/tpm
  • Reload:
    • Restart or reload tmux (prefix + I to install plugins).

Using ~/.config/tmux is superior because it is the standard location for modern tmux versions (3.1+), and it keeps plugins, session data, and configurations in one organized directory.
