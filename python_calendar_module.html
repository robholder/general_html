Calendar Module

Below is an overview of Python’s built‑in calendar module—a reference guide that you can use to refresh or deepen your understanding of its key functionalities. This guide covers common functions and classes, explains what they’re used for, and provides usage examples.

Overview

The calendar module is designed to supply useful utilities for calendar-related tasks. It can:

  • Determine details about months and years (such as the number of days in a month).
  • Output formatted calendars as multi-line strings, either in plain text or HTML.
  • Help with determining weekdays, leap years, and more.

At its core, the module supports both functional programming (with functions you can call directly) and object-oriented approaches (using calendar classes).

Key Functions

calendar.monthrange(year, month)

  • Purpose: Returns a tuple (first_weekday, number_of_days) for the given month.
  • Example: 
import calendar

first_weekday, num_days = calendar.monthrange(2025, 5)
print(first_weekday, num_days)
# e.g. (3, 31) — meaning May 2025 starts on a Thursday (0=Monday) and has 31 days.

The first_weekday means the day that is the first of the month:

Days:

  • 0=Monday
  • 1=Tuesday
  • 2=Wednesday
  • 3=Thursday
  • 4=Friday
  • 5=Saturday
  • 6=Sunday

calendar.weekday(year, month, day)

  • Purpose: Returns the day of the week for a given date as an integer, where Monday is 0 and Sunday is 6.
  • Example: 
day_of_week = calendar.weekday(2025, 5, 2)
print(day_of_week)  # e.g., 4 (if the day is Friday)

calendar.isleap(year)

  • Purpose: Checks if a given year is a leap year.
  • Example: 
if calendar.isleap(2024):
    print("2024 is a leap year.")

calendar.leapdays(y1, y2)

  • Purpose: Counts the total number of leap years between two years, starting with y1 up to but not including y2.
  • Example: 
num_leap_years = calendar.leapdays(2000, 2025)
print(num_leap_years)

Calendar Formatting Functions

  • calendar.month(year, month, w=0, l=0)
    • Purpose: Returns a string with a calendar for a specific month.
    • Parameters:
      • w: Width (spacing) of date columns.
      • l: Number of lines for each week.
    • Example: 
cal_str = calendar.month(2025, 5)
print(cal_str)

calendar.calendar(year, w=2, l=1, c=6, m=3)

  • Purpose: Returns a multi-line string representing the calendar for an entire year.
  • Parameters:
    • w: Width of each date column.
    • l: Number of lines for each week.
    • c: Number of spaces between months.
    • m: Number of months per row.
  • Example: 
year_cal = calendar.calendar(2025)
print(year_cal)

calendar.monthcalendar(year, month)

  • Purpose: Returns a matrix (list of lists) where each sub-list represents a week, and each element is the day number (or 0 if that day is not in the current month).
  • Example: 
weeks = calendar.monthcalendar(2025, 5)
for week in weeks:
    print(week)

Calendar Classes

The module also provides classes to help generate calendars with greater control.

calendar.TextCalendar

  • Purpose: Generates plain text calendars.
  • Usage: 
tc = calendar.TextCalendar(firstweekday=0)  # 0 = Monday is the first day
cal_text = tc.formatmonth(2025, 5)
print(cal_text)

calendar.HTMLCalendar

  • Purpose: Outputs calendars as HTML tables.
  • Usage: 
hc = calendar.HTMLCalendar(firstweekday=0)
cal_html = hc.formatmonth(2025, 5)
print(cal_html)

These classes also allow you to customize formatting by overriding methods if needed.

Constants and Customization

Weekday Names & Constants

  • The module defines constants that can help with working with weekdays:
    • calendar.MONDAY, calendar.TUESDAY, … calendar.SUNDAY 
print(calendar.MONDAY)  # Usually equals 0

Setting the First Day of the Week

Many functions allow you to set which day is considered the start of the week (e.g., Monday vs. Sunday). This is often done by passing the firstweekday parameter to classes like TextCalendar or via the function calendar.setfirstweekday(). 

calendar.setfirstweekday(calendar.SUNDAY)  # Changes the first day of the week to Sunday

Practical Examples

Example: Determine a Month’s Date Range

Using monthrange to find the number of days in a month (as seen in previous scripts): 

import calendar
year, month = 2025, 5
first_weekday, num_days = calendar.monthrange(year, month)
print(f"May 2025 starts on weekday {first_weekday} and has {num_days} days.")

Example: Print a Year Calendar

import calendar
print(calendar.calendar(2025))

Example: Create a Calendar Matrix

weeks = calendar.monthcalendar(2025, 5)
for week in weeks:
    print(week)

This shows each week as a list of day numbers (with zeros filling in the gaps), which can be useful for scheduling or layout problems.

Summary

  • Basic Date Queries: Use functions like monthrange, weekday, isleap, and leapdays to get numeric calendar information.
  • Formatted Calendar Output: Functions like month, calendar, and classes like TextCalendar and HTMLCalendar allow you to generate human-readable representations of calendars.
  • Calendar Data Structures: monthcalendar returns calendar information in a list-of-lists format, useful for programmatically processing weeks and days.
  • Customization: The module gives you the flexibility to set the first day of the week and format calendars in various styles.

This breakdown of the calendar module should serve as a helpful reference for your day-to-day calendar operations in Python and help you integrate calendar functionality into your programs.
