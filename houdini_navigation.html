Navigation Tools

To Make the Viewport Behave (On MacOS at least)

  • Settings:
    • 3D Viewports:
      • Tumbling Method:
        • 90's Style Euler
  • LMB <-- Tumble (if tumble is tilting, Edit > Preferences > 3d Viewports > "90s Style Euler" and restart Houdini)
  • MMB <-- Track
  • RMB <-- Dolly

Return to navigation following alternative tool:

  • cam icon (left of panel) for navigation mode
  • <esc> key to revert to the cam icon navigation mode

From Foundations Course

Viewports

  • Viewports update based on context:
    • Pin - to keep the viewport at the current view
  • Camera tool:
    • Allows navigatin without 'Alt' key
  • Viewport:
    • LMB - Tumble
    • MMB - Track
    • RMB - Dolly in and out
  • Move tool:
    • Space too to invoke Viewport navigation
  • Spacebar F:
    • focus on selection
  • Spacebar H:
    • focus on Home Grid
  • Spacebar G:
    • focus on selection, but with framing to best fit
  • Spacebar A:
    • focus on 'all'
  • Spacebar Cntl:
    • rotate about camera vector

Views

  • Space 1:
    • perspective view
  • Space 2:
    • Top View
  • Space 3:
    • Right View
  • Space 4:
    • Front View

Radial Menus
