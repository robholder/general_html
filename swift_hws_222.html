Hacking With Swift - 222 - Challenge - My own tweaks

Delete Photo Row

QUESTION

I tried adding .onDelete to the List in ContentView to allow for photo item deletion. But it doesn't compile.

ANSWER

You stumbled into one of the trickiest syntactic speed bumps in SwiftUI!

Your logic is perfectly sound, but you ran into two roadblocks: a placement error that confuses the compiler, and a SwiftData restriction that prevents you from manipulating a @Query array directly.

Here is exactly why it didn't compile, followed by how to fix both pieces.

Roadblock 1: The Placement Error

In SwiftUI, you cannot attach .onDelete directly to a List.

Instead, .onDelete must always be attached to a ForEach block inside the List. The List view container handles the scrolling layout, but it relies on a ForEach loop to keep track of the dynamic rows that are eligible for swipe-to-delete behaviors.

Roadblock 2: The @Query Rule

You tried to do photos.remove(atOffsets: offsets).

Because photos is managed by the @Query property wrapper, it is a read-only reflection of whatever is inside the database. You cannot manually pluck items out of that array. Instead, you have to tell your modelContext assembly worker to delete the object from disk. Once the item is deleted from the database, @Query will instantly spot the change and update your list view automatically.

The Solution: Code Implementation

Here is how you adjust your view body and your deletion function to play nicely with SwiftData:

1. Update the View Layout (Adding ForEach)

Wrap your NavigationLink row template inside a ForEach container, and attach the .onDelete modifier to that ForEach: 

var body: some View {
    NavigationStack {
        List { // 1. List turns into a static container
            ForEach(photos) { item in // 2. ForEach loops through the data rows
                NavigationLink(destination: PhotoDetailView(namedPhoto: item)) {
                    HStack(spacing: 16) {
                        if let uiImage = UIImage(data: item.photoData) {
                            Image(uiImage: uiImage)
                                .resizable()
                                .scaledToFill()
                                .frame(width: 60, height: 60)
                                .clipShape(.rect(cornerRadius: 8))
                        }
                        
                        Text(item.name)
                            .font(.headline)
                    }
                }
            }
            .onDelete(perform: removePhoto) // 3. Attached directly to the ForEach!
        }
        .navigationTitle("Photo Collector")
    }
}

2. Update the Deletion Function

Rewrite your removePhoto function to fetch the object from the index, and hand it over to the modelContext to delete: 

func removePhoto(at offsets: IndexSet) {
    for index in offsets {
        // Find the specific item the user swiped on
        let photoToDelete = photos[index]
        
        // Tell the database context worker to destroy it
        modelContext.delete(photoToDelete)
    }
    
    // Tip: SwiftData automatically saves this deletion behind the scenes, 
    // causing the UI list row to smoothly disappear.
}

NamedPhoto.swift

import Foundation
import SwiftData

@Model
class NamedPhoto {
    var id: UUID
    var name: String
    
    // This tells SwiftData to save the raw image data as a separate file outside the DB
    @Attribute(.externalStorage) var photoData: Data
    
    init(name: String, photoData: Data) {
        self.id = UUID()
        self.name = name
        self.photoData = photoData
    }
}

PhotoDetailView.swift

import SwiftUI

struct PhotoDetailView: View {
    let namedPhoto: NamedPhoto
    
    var body: some View {
        VStack {
            if let uiImage = UIImage(data: namedPhoto.photoData) {
                Image(uiImage: uiImage)
                    .resizable()
                    .scaledToFit()
                    .clipShape(.rect(cornerRadius: 12))
                    .padding()
            } else {
                ContentUnavailableView("Loading Image Failed", systemImage: "xmark.circle")
            }
            
            Spacer()
        }
        .navigationTitle(namedPhoto.name)
        .navigationBarTitleDisplayMode(.inline)
    }
}

ContentView.swift

import PhotosUI
import SwiftData
import SwiftUI

struct ContentView: View {
    @Environment(\.modelContext) var modelContext
    
    // Automatically fetches items from SwiftData, pre-sorted alphabetically by name
    @Query(sort: \NamedPhoto.name) var photos: [NamedPhoto]
    
    // PhotosPicker States
    @State private var selectedPickerItem: PhotosPickerItem? = nil
    @State private var temporaryPhotoData: Data? = nil
    
    // Alert States for naming the photo
    @State private var showingNamePrompt = false
    @State private var newPhotoName = ""
    
    var body: some View {
        NavigationStack {
            List { // 1. List turns into a static container
                ForEach(photos) { item in // 2. ForEach loops through the data rows
                    NavigationLink(destination: PhotoDetailView(namedPhoto: item)) {
                        HStack(spacing: 16) {
                            if let uiImage = UIImage(data: item.photoData) {
                                Image(uiImage: uiImage)
                                    .resizable()
                                    .scaledToFill()
                                    .frame(width: 60, height: 60)
                                    .clipShape(.rect(cornerRadius: 8))
                            }
                            
                            Text(item.name)
                                .font(.headline)
                        }
                    }
                }
                .onDelete(perform: removePhoto) // 3. Attached directly to the ForEach!
            }
            .navigationTitle("Photo Collector")
            .overlay {
                if photos.isEmpty {
                    ContentUnavailableView("No Photos", systemImage: "photo.on.rectangle.angled", description: Text("Tap the button below to import your first photo!"))
                }
            }
            .toolbar {
                ToolbarItem(placement: .bottomBar) {
                    // 1. The Picker Button
                    PhotosPicker(selection: $selectedPickerItem, matching: .images) {
                        Label("Import Photo", systemImage: "photo.badge.plus")
                            .font(.headline)
                    }
                }
            }
            // 2. Listen for when the user successfully chooses a photo from the library
            .onChange(of: selectedPickerItem) { _, newItem in
                Task {
                    if let data = try? await newItem?.loadTransferable(type: Data.self) {
                        temporaryPhotoData = data
                        newPhotoName = "" // Reset text field
                        showingNamePrompt = true // Trigger the naming alert immediately
                    }
                }
            }
            // 3. Intercept the flow and ask the user to input a name
            .alert("Name Your Photo", isPresented: $showingNamePrompt) {
                TextField("Enter name", text: $newPhotoName)
                
                Button("Save") {
                    saveItem()
                }
                .disabled(newPhotoName.trimmingCharacters(in: .whitespacesAndNewlines).isEmpty)
                
                Button("Cancel", role: .cancel) {
                    temporaryPhotoData = nil
                }
            } message: {
                Text("Please give a descriptive name to the picture you just imported.")
            }
        }
    }
    
    // 4. Commit the paired Name and Data into SwiftData permanent memory
    func saveItem() {
        guard let data = temporaryPhotoData else { return }
        
        let newPhoto = NamedPhoto(name: newPhotoName, photoData: data)
        modelContext.insert(newPhoto)
        
        // Clean up state tracking values
        temporaryPhotoData = nil
        selectedPickerItem = nil
    }
    
    func removePhoto(at offsets: IndexSet) {
        for index in offsets {
            // Find the specific item the user swiped on
            let photoToDelete = photos[index]
            
            // Tell the database context worker to destroy it
            modelContext.delete(photoToDelete)
        }
        
        // Tip: SwiftData automatically saves this deletion behind the scenes,
        // causing the UI list row to smoothly disappear.
    }
}
