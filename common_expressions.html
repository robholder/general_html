detail()

detail(<string path to geo>, <string attribute>, <index; always zero>)

e.g., 

detail("../meta/", "iteration", 0)

point()

For Hscript: point(<string path to geo>, <point number>, <string attribute>, <index integer>) e.g., 

point("../grid1/", 5, "P", 0)

For VEX: point(<string path to geo>, <string attribute>, <point number>) e.g., 

point("../grid1/", "P", 0)

rand()

Returns a number between 0 and 1. (pseudo-random) 

rand(value) // where value is an integer or a float

fit()

Returns a value that is within a new range and comparable to a number from an old range. 

fit(num, oldmin, oldmax, newmin, newmax)

fit01()

Same as fit() but assumes the oldmin and oldmax are 0 and 1 respectively. 

fit01(numb, newmin, newmax)

fit10()

Same as fit() but assumes the oldmin and oldmax are 1 and 0 respectively. 

fit10(numb, newmin, newmax)

ch()

Returns value of a parameter to which you are pointing.

ch(<string parmeter_path>) 

ch("sizey")/2

You can use RMB to select 'Copy Paramter' to get the name/string, and RMB to paste.

ch(), chi(), chf(), chs()

These are like channel reference above, but are for VEX and determine the channel type within the call. 

string attrib_name = chs("<attribute>");
float radius = chf("<radius>");
int maxPoints = chi("maxPoints");

chramp()

Returns ramp parameters as a value. 

float ramp = chramp('Ramp')

another example: 

@gradient = (float)@ptnum/(float)(@numpt - 1);
float ramp = chramp('Ramp', @gradient);

@P.y = ramp;

centroid()

Hscript: Returns axis coordinate for the center of the bounding box.

centroid(<string geometry>, <integer index>) //where index is as follows: 0=X, 1=Y, 2=Z 

centroid("../box1/", 0) centroid("../box1/", 1) centroid("../box1/", 2)

getpointbox_center()

VEX: Returns axis coordinate for the center of the bounding box.

getpointbox_center(<string geometry, <integer index>) //where index is as follows: 0=X, 1=Y, 2=Z 

getpointbox_center("../box1/", 0) getpointbox_center("../box1/", 1) getpointbox_center("../box1/", 2)

bbox()

Hscript: Returns bounding-box information.

bbox(<string geometry>, <integer index>)

  • Index:
    • 0 = Xmin
    • 1 = Ymin
    • 2 = Zmin
    • 3 = Xmax
    • 4 = Ymax
    • 5 = Zmax
    • 6 = Xsize
    • 7 = Ysize
    • 8 = Zsize 
      bbox("../box1", 1)

relbbox()

VEX: Returns bounding-box information.

relbbox(<string geometry>, <string prim_group>, <vector [x, y, z]>)

xyzdist()

Hscript: Returns distance between coordinates and geometry.

xyzdist(x, y, z, <surface_node>, <prim_num>, <return_type>)

  • xyz:
    • integers or floats
  • surface_node:
    • node to reference
  • prim_num:
    • Primitive's number from the geometry that is used to find the distance from the xyz coords
    • -1 is the closest primitive
  • return_type:
    • 0 = minimum distance
    • 1 = closest U parametric value
    • 2 = closest V parametric value
    • 3 = closest prim num

VEX: Returns distance between coordinates and geometry.

xyzdist(<surface_node>, <origin>)
