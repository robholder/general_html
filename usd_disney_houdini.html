What is USD

UDS Vocabulary

(Taken from "What is USD: A Primer By Rob Stauffer".)

  • Universal Scene Description:
    • A file format that describes a scene graph
  • Reference or composite files
  • Each department has its own USD file:
    • Each department can non-destructively edit passed information
  • Primitive:
    • Container for data - atomic level
  • Namespace
    • the path to the Primitive in the scene graph
  • Opinion - sometimes a layering priority, sometimes a stated data priority:
    • Higher layers
    • Explicit data (missing from higher layers) has higher priority/opinion
  • Stage:
    • The palette of all UDS files.
  • Layer:
    • Anything that lives on disk
    • All are sublayers of Root (not at Root layer)
  • Hydra:
    • API that deals with UDS
    • Render Delegates are the Renderers that communicate with Hydra
    • Render Delegates can be switched at any time
    • Hydra viewport:
      • You can select geometry directly from the 'picture'
  • Variants:
    • Variations stored in a single asset
  • References:
    • Pointer to file on disk
  • Payloads:
    • References with on/off facility
    • All references are payloadable
  • Collections:
    • Similar to groups
    • Organization of assets
    • Useful for lights (especially using light linker and light mixer)

A USD Primer

  • USD is a format that defines a Scene Graph:
    • Geo
    • anim
    • lights
    • cameras
    • shaders
    • references assets and layers them
  • Describes whole scene
  • Allows artist to colaborate on a scene
  • Each shot in a single file
  • Each dept. has a component:
    • Layout:
      • Asset 1
      • Asset 2
    • Animation:
      • Character 1
      • Character 1
    • FX:
      • stuff
    • Lighting:
      • stuff
  • Non-destructive as the data flows through departments.
  • each file is a layer
  • Stage:
    • The final composition of the scene is the 'Stage'
  • Composition Arcs:
    • Different wasy to combine layers

Placing Lights

You can adjust and place lights using spec, diff or shadow!

  • Click to place taret
  • Cntl-drag to change distance
  • Shift-s for spec
  • Shift-d for diffuse
  • Shift-f for shadow:
    • Click to place pivot
    • Shift-click to place target of shadow placement
  • Cnrl-Shift-drag to change brightness
  • Shift-t to toggle look at handle
